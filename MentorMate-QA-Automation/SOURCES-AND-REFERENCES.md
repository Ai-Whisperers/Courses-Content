## üèóÔ∏è Testing Pyramid & Methodologies

### **Mike Cohn - "Succeeding with Agile"** (2009)
- **Author:** Mike Cohn, Agile Alliance co-founder
- **Book:** *Succeeding with Agile: Software Development Using Scrum*
- **Concept:** Test Automation Pyramid
- **Used For:** Module 1 - Testing Pyramid explanation
- **Key Idea:** Many unit tests, fewer integration tests, even fewer UI tests
- **Reference:** Cohn, M. (2009). *Succeeding with Agile: Software Development Using Scrum*. Addison-Wesley.

### **Martin Fowler - "TestPyramid"**
- **Author:** Martin Fowler, ThoughtWorks Chief Scientist
- **Article:** "The Practical Test Pyramid" 
- **Website:** https://martinfowler.com/articles/practical-test-pyramid.html
- **Used For:** Testing strategy and test distribution concepts
- **Relevance:** Expands on Cohn's pyramid with modern interpretations
