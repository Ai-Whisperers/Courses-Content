## ‚öñÔ∏è AI Ethics & Best Practices

### Do's ‚úÖ
- Use AI to **accelerate** learning, not replace it
- **Verify** all AI-generated code and tests
- **Understand** what AI creates before using it
- **Cite** AI assistance in your work
- Use AI for **ideation and iteration**

### Don'ts ‚ùå
- Don't blindly copy AI-generated code
- Don't skip understanding fundamentals
- Don't use AI for assessments without permission
- Don't share proprietary code with AI tools
- Don't rely solely on AI for critical decisions

## üí° Pro Tips

1. **Context is King:** The better your prompts, the better AI results
2. **Iterate:** Refine prompts based on outputs
3. **Compare:** Generate multiple solutions, choose the best
4. **Learn:** Ask AI to explain its reasoning
5. **Validate:** Always test AI-generated code
6. **Document:** Keep track of effective prompts

### AI Tool Not Working?
- Check `resources/troubleshooting-guide.md`
- Verify API key configuration
- Ensure free tier limits not exceeded

### Poor AI Results?
- Improve prompt specificity
- Add more context
- Use templates from `templates/`
- Try a different AI tool

- Review course AI policy
- Ask instructor for guidance
- When in doubt, cite your AI usage
