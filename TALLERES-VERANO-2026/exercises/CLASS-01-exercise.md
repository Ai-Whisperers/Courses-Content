# Ejercicio Clase 1: Tu Primer Prompt Efectivo

**Duración**: 15 minutos  
**Dificultad**: Principiante  
**Herramienta**: OpenCode o ChatGPT/Claude

---

## Objetivo

Aprender a escribir prompts efectivos usando la estructura CRAFT:
- **C**ontexto - Situación o background
- **R**ol - Quién debe ser la IA
- **A**cción - Qué debe hacer
- **F**ormato - Cómo presentar el resultado
- **T**ono - Estilo de comunicación

---

## Ejercicio 1: Prompt Básico vs Prompt CRAFT

### Parte A: Prompt Básico (MAL ejemplo)

Copia y pega esto en OpenCode:

```
Dame información sobre el cambio climático
```

**Observa**: La respuesta es genérica y probablemente no útil para tu propósito específico.

---

### Parte B: Prompt CRAFT (BUEN ejemplo)

Ahora prueba este prompt estructurado:

```
CONTEXTO: Soy estudiante de Ingeniería Ambiental en Paraguay y debo 
preparar una presentación de 10 minutos para la clase de Ecología.

ROL: Actúa como un profesor universitario experto en cambio climático 
con experiencia en la región del Cono Sur.

ACCIÓN: Explica los 3 principales efectos del cambio climático en 
Paraguay, con datos específicos y ejemplos locales (Chaco, río Paraguay).

FORMATO: 
- 3 puntos principales con subtítulos
- Cada punto con 1 estadística y 1 ejemplo local
- Máximo 300 palabras total

TONO: Académico pero accesible para estudiantes de pregrado.
```

**Compara** las dos respuestas. ¿Cuál es más útil para tu propósito?

## Ejercicio 2: Crea tu Propio Prompt CRAFT

### Instrucciones

1. Piensa en una tarea real que necesites hacer para alguna clase
2. Completa la plantilla CRAFT abajo
3. Prueba tu prompt en OpenCode
4. Ajusta según los resultados

### Tu Plantilla

```
CONTEXTO: [Describe tu situación - qué clase, qué materia, qué necesitas]

ROL: [¿Qué tipo de experto necesitas? Profesor, profesional, tutor...]

ACCIÓN: [¿Qué específicamente quieres que haga?]

FORMATO: [¿Cómo quieres el resultado? Lista, párrafos, tabla...]

TONO: [¿Formal, casual, técnico, simple?]
```

---

## Ejercicio 3: Refinamiento Iterativo

### El Problema

Tu primer prompt raramente es perfecto. Aprende a mejorar.

### Tarea

1. Usa tu prompt del Ejercicio 2
2. Si el resultado no es exactamente lo que querías, agrega:
   - "Enfócate más en..."
   - "Hazlo más corto/largo"
   - "Agrega ejemplos de..."
   - "Usa un lenguaje más simple"

3. Repite hasta obtener el resultado deseado

- [ ] Guardé mi mejor prompt para referencia futura

## Solución / Ejemplo de Buen Resultado

Un buen prompt CRAFT para un estudiante de Marketing podría ser:

```
CONTEXTO: Soy estudiante de Marketing en FPUNA y necesito crear 
una campaña para una panadería local en San Lorenzo.

ROL: Eres un consultor de marketing digital especializado en 
pequeñas empresas latinoamericanas.

ACCIÓN: Diseña una estrategia de Instagram de 1 semana para 
aumentar las ventas de la panadería, incluyendo:
- 3 tipos de posts con ejemplos de texto
- Mejores horarios para publicar
- Hashtags locales relevantes

FORMATO: Tabla con día, tipo de post, texto sugerido, y hashtags.

TONO: Creativo y cercano, apropiado para redes sociales.
```

3. ¿Qué harías diferente la próxima vez?
