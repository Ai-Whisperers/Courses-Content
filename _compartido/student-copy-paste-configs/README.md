# Configuraciones Copy-Paste Claude Code

Este directorio contiene configuraciones listas para copiar y usar para estudiantes nuevos en Claude Code. Elegidas de los mejores repositorios de GitHub con complejidad minimal y defaults sensatos.

## Cómo Usar

1. Elegir configuración appropriate por nivel
2. Copiar archivos CLAUDE.md a proyecto
3. Copiar skills adicionales según necesidad
4. Customizar paths y preferences
5. Probar con tarea simple

## Configs Disponibles

### Config Básico Inicial (claude-starter-kit)
- Archivo principal: `basic-claude-starter.md`
- Incluye: Rules coding básicas, tools esenciales, examples
- Complejidad: ⭐ (Muy fácil)

### Config Desarrollo Medio (AizenvoltPrime)
- Archivo principal: `intermediate-setup.md`  
- Incluye: MCP servers integrado, comandos personalizados, hooks calidad
- Complejidad: ⭐⭐ (Moderado)

### Skills Esenciales Estudiantil
- Directorio: `skills-basics/`
- Incluye: Testing helper, code review basic, git shortcuts
- Focus: Productividad educación vs desarrollo avanzado

### Examples Educacionales
- Directorio: `examples-education/`
- CLAUDE.md templates para diferentes tipos proyecto
- Comentarios explicando cada sección

---

**Nota**: Todos configs optimizados para estudiantes con balance poder/simplicidad. No incluyen configuraciones enterprise complejas intencionalmente.
