## Framework CREA

```
C - CONTEXTO     → ¿Cuál es la situación?
R - ROL          → ¿Qué experto debe ser la IA?
E - ESPECIFICACIONES → ¿Qué requisitos exactos?
A - ACCIÓN       → ¿Qué debe hacer?
```

**Ejemplo Rápido:**
```
[C] Soy estudiante de ingeniería en Paraguay.
[R] Actúa como profesor de programación.
[E] Lenguaje simple, código comentado, máximo 100 líneas.
[A] Escribe un programa que calcule el factorial de un número.
```

## Técnicas de Prompting

| Técnica | Cuándo Usar | Cómo |
|---------|-------------|------|
| **Zero-shot** | Preguntas simples | Preguntar directo |
| **Few-shot** | Clasificación/formato | Dar 2-3 ejemplos primero |
| **Chain-of-thought** | Problemas complejos | "Piensa paso a paso" |
| **Role-playing** | Audiencia específica | "Actúa como..." |

## Limitaciones a Recordar

---

## ⚠️ NO Compartir con IA

- Contraseñas
- Datos bancarios
- Información médica personal
- Datos confidenciales de trabajo
- Información personal de terceros

---

## Ética - Reglas Rápidas

1. **Verificar** información crítica
2. **Declarar** uso de IA cuando corresponda
3. **Entender** antes de usar contenido generado
4. **Proteger** datos sensibles
5. **Usar como herramienta**, no como reemplazo de pensamiento

## Verificación de Respuestas

Antes de usar información de IA:

- [ ] ¿Es crítica? → Verificar externamente
- [ ] ¿Es reciente? → Confirmar actualidad
- [ ] ¿Tiene citas? → Verificar que existen
- [ ] ¿Tiene números? → Validar cálculos
- [ ] ¿Es código? → Probar y revisar seguridad
