# Quiz - Módulo 03: Frontend con IA

**Duración:** 5 minutos | **Puntos:** 100

---

### Pregunta 1
¿Cuál es la ventaja de usar Tailwind CSS con IA?
- A) Es más lento
- B) Las clases son descriptivas y fáciles de generar/modificar con IA
- C) No funciona con React
- D) Requiere más código

<details><summary>Respuesta</summary>B - Las clases utilitarias de Tailwind son predecibles y la IA las maneja muy bien.</details>

---

### Pregunta 2
¿Por qué usar TypeScript con componentes React generados por IA?
- A) Es obligatorio
- B) Ayuda a detectar errores y documenta las props esperadas
- C) Hace el código más largo
- D) La IA no puede generar TypeScript

<details><summary>Respuesta</summary>B - TypeScript añade seguridad de tipos y sirve como documentación viva.</details>

---

### Pregunta 3
¿Qué librería se recomienda para formularios en React?
- A) jQuery Forms
- B) react-hook-form con zod
- C) Formularios nativos HTML solamente
- D) Angular Forms

<details><summary>Respuesta</summary>B - react-hook-form es performante y zod proporciona validación tipada.</details>

---

### Pregunta 4
¿Qué es mobile-first en diseño responsive?
- A) Solo diseñar para móviles
- B) Diseñar primero para móvil y luego expandir para pantallas grandes
- C) Ignorar dispositivos móviles
- D) Usar JavaScript para detectar móviles

<details><summary>Respuesta</summary>B - Mobile-first significa estilos base para móvil, luego media queries para expandir.</details>

---

### Pregunta 5
¿Para qué sirve SWR o React Query?
- A) Animaciones
- B) Data fetching, caching, y sincronización con el servidor
- C) Estilos CSS
- D) Routing

<details><summary>Respuesta</summary>B - Estas librerías manejan el estado del servidor de forma eficiente.</details>
