## For Students

### How to Use These Materials

1. **Before Each Sunday Session:**
   - Review previous week's material
   - Complete assigned readings
   - Prepare questions

2. **During Sunday Session:**
   - Participate in hands-on labs
   - Complete in-class exercises
   - Take notes

3. **After Sunday Session:**
   - Complete homework exercises
   - Take weekly quiz
   - Study for next week

4. **Weeks 3-4:**
   - Work on final project
   - Attend office hours if needed
   - Prepare presentation

### Submission Guidelines

**Exercises:**
- Submit via GitHub repository
- Follow naming conventions
- Include README with instructions
- Due: Before next Sunday class

**Quizzes:**
- Complete within 48 hours of class
- Open book/notes
- One retake allowed if needed
- Submit via [LMS/Google Forms]

**Final Project:**
- Submit by end of Week 4
- All components required
- Presentation (live or recorded)
- See FINAL-PROJECT.md for details

## Assessment Policies

### Late Submission Policy

**Quizzes:**
- Up to 24 hours late: No penalty
- 24-48 hours late: 10% penalty
- After 48 hours: Can still take for learning (no credit)

**Exercises:**
- Up to 2 days late: No penalty
- 2-7 days late: 10% penalty per day
- After 7 days: 50% max score

**Final Project:**
- Grace period: 3 days (no penalty)
- After grace period: 5% per day
- Maximum: 2 weeks late

### Retake Policy

**Quizzes:**
- One retake per quiz
- Must wait 24 hours
- Higher score counts
- Different questions

**Exercises:**
- Can resubmit within 1 week
- Must show significant improvement
- Max 80% of original grade

**Final Project:**
- One revision opportunity
- Must submit within 2 weeks of feedback
- Full credit possible

### Academic Integrity

**Allowed:**
- Using course materials
- Consulting official documentation
- Asking instructor questions
- Discussing concepts with peers
- Using AI tools (for AI-track students)

**Not Allowed:**
- Copying code from classmates
- Submitting others' work as your own
- Sharing quiz answers
- Using AI for quizzes
- Plagiarizing documentation

**Consequences:**
- First offense: Zero + warning
- Second offense: Fail course
- Severe cases: Report to organization

## FAQ

**Q: What if I miss a Sunday session?**
A: Watch recording (if available), complete exercises, attend office hours for catch-up. Participation points will be affected.

**Q: Can I submit exercises early?**
A: Yes! Submit as soon as ready. Faster feedback cycle.

**Q: What if I fail a quiz?**
A: Take advantage of the one retake. Study the material and try again after 24 hours.

**Q: Can I work with a partner?**
A: Exercises and final project must be individual work. You can discuss concepts, but code must be yours.

**Q: What's the minimum passing grade?**
A: 70% overall. Must complete final project to pass.

**Q: Can I use AI tools?**
A: For AI-track students, yes with documentation. For traditional track, discouraged for learning purposes, but allowed for final project if documented.

**Q: How do I know my current grade?**
A: Check the grade tracking spreadsheet (shared link) or ask instructor.
