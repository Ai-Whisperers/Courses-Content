## Project Options

### Option A: Full-Stack Feature
Build a complete feature for a web application including:
- Backend API endpoints
- Frontend components
- Database integration
- Tests and documentation

### Option B: Developer Tool
Create a developer utility that demonstrates:
- CLI or VS Code extension
- Useful automation
- Clear documentation
- Published package

### Option C: Refactoring Project
Take legacy code and modernize it using AI:
- Identify improvement areas
- Apply design patterns
- Add comprehensive tests
- Document changes
