# Module 4 Exercise: Visual Content Creation Lab

**Duration**: 2 hours
**Difficulty**: Intermediate

---

## Objective

Create a cohesive set of branded visual content using AI tools: social media graphics, a blog header, and presentation visuals.

---

## Setup

### Prerequisites
- [ ] Completed Module 4 content
- [ ] Canva account (free tier works)
- [ ] Brand context and colors defined

## Part 2: Social Media Graphics (~40 minutes)

### Task

Create 4 social media graphics using Canva or AI image tools.

### Graphic 1: Quote Graphic (Instagram)

Create a quote graphic with:
- An inspiring/insightful quote relevant to your industry
- Your brand colors and style
- Format: 1080x1080

**Prompt approach:**
```
In Canva: Use Magic Design with description:
"Create a [mood] quote graphic for [industry].
Quote: '[Your quote]'
Style: [Your style guide snippet]
Format: Instagram square"
```

### Graphic 2: Statistic/Data Graphic (LinkedIn)

Create a data visualization graphic:
- Feature one compelling statistic
- Make the number visually prominent
- Include context/source
- Format: 1200x1200 or 1200x628

### Graphic 3: Tip Graphic (Twitter/X)

Create a "quick tip" graphic:
- One actionable tip from your expertise
- Easy to read at small size
- High contrast
- Format: 1200x675

### Graphic 4: Behind-the-Scenes/Process (Instagram Story)

Create a story graphic showing:
- A process, workflow, or "how we do it"
- Your brand personality
- Format: 1080x1920

### For each graphic, document:
- Tool used
- Prompt or approach
- Iterations needed
- Brand consistency check

### Deliverables
- [ ] 4 graphics (exported as images)
- [ ] Documentation of prompts/approach for each

## Part 3: Blog Header Image (~25 minutes)

### Task

Create a blog header image for an article from your content batch (Module 2).

### Requirements:
- Aspect ratio: 16:9 or 2:1
- Matches your blog topic
- Consistent with brand style
- Suitable for both desktop and mobile preview
- No text (or minimal text that's part of the design)

### Approach Options:

**Option A: Canva Magic Design**
```
"Create a blog header about [topic]. Style: [brand style].
Colors: [brand colors]. Mood: [mood]. Professional and clean,
suitable for B2B blog."
```

**Option B: DALL-E/Midjourney**
```
Blog header image for article about [topic].
[Style description from your guide].
Abstract, professional, corporate blog aesthetic.
[Brand colors] color palette.
Wide format, 16:9 aspect ratio.
Clean composition with space for text overlay.
No text in the image.
```

### Document:
- Prompt used
- Iterations needed
- Why you chose the final version

### Deliverables
- [ ] Blog header image
- [ ] Prompt documentation

## Part 4: Presentation Visuals (~25 minutes)

### Task

Create visual elements for a short presentation.

### Create 3 slides worth of visuals:

**Slide 1: Title Slide Background**
- Topic: [Your main presentation topic]
- Style: Aligns with brand
- Space for title and subtitle

**Slide 2: Data Visualization**
- Visualize this data point: [Choose a relevant statistic]
- Make the key number prominent
- Include brief context

**Slide 3: Concept Icons**
- Create or find 3-4 icons representing key concepts
- Consistent style across all icons
- On-brand colors

### Document your approach and tools used.

### Deliverables
- [ ] 3 presentation visual elements
- [ ] Approach documentation

## Tips for Success

1. **Start with style guide** - Don't skip this; it makes everything easier
2. **Save your prompts** - Build a library for future use
3. **Iterate** - First output rarely perfect; refine
4. **Check mobile** - View all graphics at small sizes
5. **Be consistent** - When in doubt, refer back to style guide
