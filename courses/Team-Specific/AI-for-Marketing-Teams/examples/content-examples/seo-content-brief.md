# Example: SEO Content Brief

## Project Details

- **Content Type**: Pillar Page (Long-form guide)
- **Target Keyword**: "AI content marketing strategy"
- **Search Volume**: 1,300/month
- **Keyword Difficulty**: 42 (Medium)
- **Business Goal**: Establish thought leadership, drive organic leads

---

## The Prompt Used

```
ROLE: SEO content strategist for B2B marketing.

CONTEXT:
Company: GrowthLab Marketing (B2B marketing agency)
Target keyword: "AI content marketing strategy"
Search intent: Informational/Commercial (people researching how to implement)
Audience: Marketing managers and directors at B2B companies
Our expertise: We've helped 50+ companies implement AI content systems

TASK: Create a comprehensive content brief for a pillar page targeting this keyword.

INCLUDE:
1. Content objectives (what success looks like)
2. Search intent analysis
3. Competitor analysis summary (what top results cover)
4. Recommended outline with H2s and H3s
5. Key questions to answer
6. Internal linking opportunities
7. Content specifications (length, format, media)
8. SEO checklist

FORMAT: Detailed brief that a writer could use to create the content.
```

## AI Generated Brief

### Content Objectives

- Generate 500+ organic visits/month
- Capture 50+ email subscribers via content upgrade
- Establish authority for related long-tail keywords

**Success Metrics:**
- Rankings: Top 10 within 90 days, Top 5 within 180 days
- Traffic: 200 visits/month at 90 days, 500 at 180 days
- Engagement: 4+ minute average time on page
- Conversions: 10% content upgrade download rate

---

### Search Intent Analysis

**Intent Type:** Mixed (Informational + Commercial Investigation)

**What searchers want:**
- Understand what AI content marketing strategy means
- Learn how to create/implement one
- See examples and frameworks
- Evaluate if AI is right for their situation

**What they DON'T want:**
- Basic "what is AI" explanations
- Pure theory without practical application
- Sales pitch disguised as content

**Intent signals from SERPs:**
- Top results are comprehensive guides (2,000-5,000 words)
- Multiple "how to" and "step-by-step" results
- People Also Ask questions focus on implementation

---

### Competitor Analysis

**Top 3 Results Analysis:**

| Competitor | Word Count | Strengths | Gaps |
|------------|------------|-----------|------|
| HubSpot | 3,200 | Comprehensive, strong brand | Generic, no specific tools |
| CMI | 2,800 | Expert quotes, research-backed | Dated (2023 data), surface-level |
| Semrush | 4,100 | Detailed, good visuals | Too tool-focused, overwhelming |

**Content Gap Opportunities:**
1. Real implementation timeline (not just theory)
2. Specific prompt examples and templates
3. ROI calculation framework
4. Common mistakes and how to avoid them
5. Team structure recommendations
6. Tool comparison (not just one tool)

### Content Specifications

**Length:** 4,000-5,000 words
- Longer than competitors to be comprehensive
- But well-organized so scannable

**Format Requirements:**
- H2 every 300-400 words
- Bulleted lists for steps and features
- Numbered lists for processes
- Tables for comparisons
- Pull quotes for key insights

**Visual Elements:**
- Hero image (custom branded)
- Process diagram for strategy framework
- Comparison table for tools
- ROI calculator screenshot
- Implementation timeline infographic
- 2-3 relevant screenshots

**Content Upgrades:**
- Downloadable AI Content Strategy Template (PDF)
- ROI Calculator (Spreadsheet)
- Brand Voice Document Template

**Reading Level:** 8th-10th grade
- Professional but accessible
- Avoid jargon without explanation
- Short paragraphs (2-4 sentences)

### SEO Checklist

**On-Page Optimization:**
- [ ] Primary keyword in title (front-loaded)
- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least 2 H2s
- [ ] Secondary keywords naturally distributed
- [ ] Meta description includes keyword (155 chars)
- [ ] URL slug: /ai-content-marketing-strategy/

**Technical:**
- [ ] Schema markup (Article, HowTo, FAQ)
- [ ] Internal links (5-10 relevant)
- [ ] External links to authoritative sources (3-5)
- [ ] Image alt text with keywords
- [ ] Mobile-friendly formatting
- [ ] Page speed optimized (images compressed)

**Content Quality:**
- [ ] Answers search intent completely
- [ ] Provides unique value beyond competitors
- [ ] Includes original examples/data
- [ ] Expert perspective/quotes included
- [ ] Actionable takeaways clear
- [ ] Updated date visible

**Engagement:**
- [ ] Table of contents (jump links)
