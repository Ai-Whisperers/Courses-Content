# AI for Marketing Teams - Content Examples

This folder contains example marketing content created with AI assistance, demonstrating best practices taught in the course.

## Contents

### content-examples/
Real examples of AI-assisted marketing content:

- **blog-post-example.md** - Complete blog post with AI workflow documentation
- **social-media-batch.md** - Week's worth of social media content
- **email-campaign-example.md** - 3-email nurture sequence
- **ad-copy-variations.md** - PAS and AIDA ad copy examples
- **seo-content-brief.md** - Complete SEO content brief

## How to Use These Examples

1. **Study the structure** - Note how prompts are documented
2. **Compare before/after** - See how editing improves AI output
3. **Adapt for your brand** - Use as templates for your content
4. **Learn from annotations** - Read the explanations of why changes were made

## Example Format

Each example includes:
- Original AI prompt used
- Raw AI output
- Edited final version
- Annotations explaining changes
- Key learnings
