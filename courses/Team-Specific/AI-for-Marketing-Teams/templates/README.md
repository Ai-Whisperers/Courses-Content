# AI Marketing Templates

This folder contains templates to help you set up AI tools for marketing projects.

## Contents

### project-setup/
Templates for configuring AI tools to understand your brand and marketing context.

- **CLAUDE.md** - Context document for Claude.ai sessions
- **.claude/settings.json** - Claude Code configuration
- **.github/copilot-instructions.md** - GitHub Copilot instructions (if used)
- **.gemini/settings.json** - Google Gemini configuration

## How to Use

### For Chat-Based AI (Claude.ai, ChatGPT)

1. Copy `project-setup/CLAUDE.md`
2. Fill in your brand information
3. Paste at the start of each AI session
4. Update as your brand evolves

### For Code-Based AI (Claude Code, Copilot)

1. Copy the relevant config files to your project
2. Customize for your tech stack and preferences
3. Commit to your repository

## Template Usage Tips

### Keep Context Documents Updated
- Review monthly
- Add learnings from successful prompts
- Remove outdated information

### Create Variations for Different Purposes
- One for content creation
- One for analytics/reporting
- One for campaign planning

### Share with Team
- Ensure consistency across team members
- Establish single source of truth
- Update collaboratively

## Quick Start

1. Start with `CLAUDE.md` template
2. Fill in minimum required fields:
   - Company name and description
   - Brand voice (3 adjectives + examples)
   - Target audience
   - Example of your best content
3. Use in your first AI session
4. Iterate based on output quality
