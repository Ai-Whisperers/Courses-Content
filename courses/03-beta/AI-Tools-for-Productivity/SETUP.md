# Setup - AI Tools for Productivity

Get your environment ready for the course.

---

## Prerequisites

| Requirement | Details |
|-------------|---------|
| Claude.ai account | [claude.ai](https://claude.ai) - Free tier works |
| ChatGPT account | [chat.openai.com](https://chat.openai.com) - Free tier works |
| Office suite | Google Workspace or Microsoft 365 |
| Browser | Chrome, Firefox, or Edge |

---

## Quick Setup (10 minutes)

### 1. Create AI Accounts

**Claude (Anthropic):**
1. Go to [claude.ai](https://claude.ai)
2. Sign up with email
3. Verify your account

**ChatGPT (OpenAI):**
1. Go to [chat.openai.com](https://chat.openai.com)
2. Sign up with email
3. Verify your account

### 2. Prepare Your Workspace

Create folders for organizing AI-assisted work:

```
ai-workflows/
├── documents/       # AI-generated documents
├── meetings/        # Meeting prep and notes
├── research/        # Research outputs
├── templates/       # Reusable prompts
└── tracking/        # Productivity metrics
```

### 3. Create Prompt Templates Folder

Save prompts that work well for reuse:

```markdown
# My Prompt Templates

## Email Templates
- Professional response
- Meeting request
- Follow-up

## Document Templates
- Report outline
- Requirements doc
- Meeting summary

## Analysis Templates
- Data insights
- Competitive analysis
- Decision matrix
```

## AI-Assisted Setup

Copy this prompt into Claude:

```
Help me set up for the AI Tools for Productivity course:

1. Create a folder structure for AI-assisted workflows:
   - documents/ (reports, emails, content)
   - meetings/ (agendas, notes, action items)
   - research/ (analysis, summaries)
   - templates/ (reusable prompts)
   - tracking/ (time saved, wins)

2. Create a prompt template library with categories:
   - Email responses
   - Meeting summaries
   - Document generation
   - Data analysis

3. Create a "time saved" tracking template

4. Give me 5 quick-win prompts I can use immediately:
   - Email drafting
   - Meeting agenda
   - Report outline
   - Data summary
   - Action items extraction

5. Show me how to evaluate AI output quality
```

## Quick Win Templates

### Email Response

```
Write a professional email response:

Context: [DESCRIBE SITUATION]
Tone: [professional/friendly/formal]
Key points to address:
1. [POINT 1]
2. [POINT 2]
3. [POINT 3]

Keep it concise (under 150 words).
```

### Meeting Summary

```
Summarize this meeting:

[PASTE MEETING NOTES OR TRANSCRIPT]

Include:
1. Key decisions made
2. Action items (with owners and deadlines)
3. Open questions
4. Next steps

Format as bullet points.
```

### Report Outline

```
Create an outline for a report on: [TOPIC]

Audience: [WHO WILL READ IT]
Purpose: [WHAT SHOULD IT ACHIEVE]
Length: [SHORT/MEDIUM/DETAILED]

Include sections for:
- Executive summary
- Key findings
- Recommendations
- Next steps
```
