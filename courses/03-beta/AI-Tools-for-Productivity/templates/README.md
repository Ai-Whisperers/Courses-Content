# AI Tools for Productivity - Templates

> Reusable templates for common productivity tasks

## Document Templates

### Meeting Agenda

```markdown
# Meeting: [Title]

**Date:** [Date]
**Time:** [Time] ([Duration])
**Attendees:** [Names]

## Objectives
1. [Objective 1]
2. [Objective 2]

## Agenda

| Time | Topic | Owner | Notes |
|------|-------|-------|-------|
| 5 min | Welcome & context | [Name] | |
| 15 min | [Topic 1] | [Name] | |
| 15 min | [Topic 2] | [Name] | |
| 10 min | Discussion & next steps | All | |

## Pre-reads
- [Link to document]

## Action Items (to be filled during meeting)
- [ ] [Action] - @owner - Due: [date]
```

### Weekly Status Report

```markdown
# Weekly Status: [Project/Team Name]

**Week of:** [Date Range]
**Prepared by:** [Name]

## Summary
[One paragraph overview of the week]

## Progress
### Completed This Week
- [x] [Task 1]
- [x] [Task 2]

### In Progress
- [ ] [Task 3] - [% complete]
- [ ] [Task 4] - [% complete]

## Metrics
| Metric | This Week | Last Week | Change |
|--------|-----------|-----------|--------|
| [Metric] | [Value] | [Value] | [+/-] |

## Blockers
1. [Blocker] - Impact: [High/Med/Low] - Needed: [What's needed]

## Risks
| Risk | Status | Action |
|------|--------|--------|
| [Risk] | [Status] | [Action] |

## Next Week
- [Priority 1]
- [Priority 2]

## Needs/Asks
- [Request with context]
```

## Email Templates

### Project Update

```
Subject: [Project Name] Update - Week [#]

Hi [Team/Name],

Quick update on [Project Name]:

**Progress:**
• [Achievement 1]
• [Achievement 2]

**Next Steps:**
• [Next step 1]
• [Next step 2]

**Need From You:**
• [Request, if any]

Let me know if you have questions.

[Your name]
```

---

### Meeting Request

```
Subject: Meeting Request: [Topic] - [Duration]

Hi [Name],

I'd like to schedule a [duration] meeting to discuss [topic].

**Purpose:** [What you want to accomplish]

**Proposed Times:**
• [Option 1]
• [Option 2]
• [Option 3]

**Agenda:**
1. [Topic 1]
2. [Topic 2]
3. Next steps

Please let me know what works for you, or suggest alternatives.

Thanks,
[Your name]
```

---

### Follow-Up

```
Subject: Follow-up: [Topic from Meeting/Conversation]

Hi [Name],

Thanks for [meeting/chatting/connecting] [timeframe].

**Key Takeaways:**
• [Point 1]
• [Point 2]

**Next Steps:**
• [Your action] - by [date]
• [Their action] - by [date]

Please let me know if I missed anything or if you have questions.

[Your name]
```

## AI Prompt Templates

### Document Drafting

```
Create a [document type] for [audience].

Context:
- Purpose: [why this document]
- Key message: [main takeaway]
- Tone: [formal/casual/persuasive]

Include:
- [Section 1]
- [Section 2]
- [Section 3]

Length: [word count or page count]
Format: [specific formatting requirements]
```

---

### Data Analysis Request

```
Analyze this [data type] and provide:

1. Summary statistics
2. Top [number] insights
3. Comparison to [benchmark/previous period]
4. Recommendations

Data:
[Paste data or describe location]

Focus specifically on: [area of interest]
Present results as: [table/bullet points/narrative]
```

---

### Content Repurposing

```
Transform this [original format] into [new format]:

Original:
[Paste content]

Target format: [blog post/social posts/email/slides]
Target audience: [who will consume this]
Key points to emphasize: [priorities]
Length: [requirements]
Tone: [requirements]
```

---

## Usage Instructions

1. Copy the template you need
2. Fill in the bracketed `[placeholders]`
3. Remove sections that don't apply
4. Add any additional context needed
5. Save your customized version for reuse
