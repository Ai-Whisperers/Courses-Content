# IA para Electrónica y Automatización Industrial
## Curso de Verano - FPUNA Paraguay

---

## Descripción del Curso

Este curso intensivo de 16 horas introduce el uso de herramientas de Inteligencia Artificial como asistentes en el desarrollo de sistemas electrónicos y automatización industrial. Los participantes aprenderán a utilizar LLMs (Claude, ChatGPT, GitHub Copilot) para acelerar el diseño de circuitos, programación de microcontroladores, desarrollo de PLCs, sistemas SCADA y mantenimiento predictivo.

## Objetivos del Curso

### Objetivo General
Capacitar a profesionales y estudiantes de ingeniería electrónica en el uso efectivo de herramientas de IA para aumentar la productividad en el diseño, desarrollo y mantenimiento de sistemas electrónicos y de automatización.

### Objetivos Específicos

Al finalizar el curso, el participante será capaz de:

1. **Configurar** entornos de desarrollo con asistentes de IA para proyectos embebidos
2. **Generar** código funcional para microcontroladores (ESP32, Arduino) usando prompts efectivos
3. **Analizar** señales industriales con técnicas de procesamiento asistido por IA
4. **Diseñar** circuitos electrónicos con cálculos y verificaciones automatizadas
5. **Programar** PLCs en Ladder y Structured Text con asistencia de IA
6. **Desarrollar** interfaces SCADA/HMI siguiendo estándares industriales
7. **Implementar** sistemas de mantenimiento predictivo con Machine Learning
8. **Integrar** todos los conocimientos en un proyecto IoT industrial completo

### Enfoque Práctico

- GitHub Copilot (autocompletado en IDE)

## Contenido por Módulo

### Módulo 01: Introducción a IA para Electrónica
- Panorama de herramientas de IA para ingeniería
- Configuración de entorno de desarrollo
- Estructura de prompts efectivos
- Limitaciones y consideraciones de seguridad

### Módulo 02: Microcontroladores con IA
- Generación de código ESP32/Arduino
- Integración de sensores y actuadores
- Comunicación WiFi y MQTT
- Deep sleep y optimización energética

### Módulo 03: Procesamiento de Señales
- Análisis de señales con Python
- Filtrado digital (Butterworth, etc.)
- Transformada de Fourier (FFT)
- Detección de anomalías (Isolation Forest)

### Módulo 04: Diseño de Circuitos
- Cálculo de componentes con IA
- Análisis térmico
- Generación de BOM
- Revisión de diseños PCB

### Módulo 05: Programación de PLCs
- Generación de lógica Ladder
- Structured Text (IEC 61131-3)
- Diagnóstico de fallos
- Documentación automática

### Módulo 06: SCADA y HMI
- Diseño de interfaces ISA-101
- Configuración de alarmas
- Scripts de control
- Integración con análisis ML

### Módulo 07: Mantenimiento Predictivo
- Fundamentos de mantenimiento basado en condición
- Clasificación de fallas con ML
- Estimación de vida útil (RUL)
- ROI de mantenimiento predictivo

### Módulo 08: Proyecto Final
- Integración de todos los módulos
- Sistema IoT industrial completo
- Presentación de proyectos
