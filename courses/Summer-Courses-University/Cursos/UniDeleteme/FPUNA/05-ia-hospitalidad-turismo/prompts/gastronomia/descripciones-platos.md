# Prompts para Descripciones de Platos
## Biblioteca de Prompts Gastronómicos

---

## Descripciones Básicas

### Plato Individual
```
Escribe una descripción de menú para [NOMBRE DEL PLATO].

Incluye:
- Ingredientes principales
- Técnica de preparación
- Palabras sensoriales (textura, sabor)
- Máximo 2-3 líneas

Tono: [elegante/casual/tradicional]
Restaurante: [tipo de restaurante]
```

### Plato con Historia
```
Crea una descripción gastronómica para [PLATO PARAGUAYO]
que incluya:

- Su origen o historia cultural
- Ingredientes tradicionales
- Por qué es especial
- Descripción sensorial
- Máximo 4 líneas

Tono: nostálgico pero moderno
```

### Plato Premium
```
Escribe una descripción fine dining para [PLATO]:

- Destacar técnica de preparación
- Mencionar origen de ingredientes
- Usar lenguaje sofisticado
- Sugerir maridaje
- Máximo 3 líneas elegantes
```

## Por Tipo de Plato

### Entradas
```
Describe esta entrada para el menú:
[NOMBRE Y COMPONENTES]

La descripción debe:
- Despertar el apetito
- Ser breve (2 líneas)
- Mencionar textura o presentación
- Incluir un detalle que la haga especial
```

### Platos Principales - Carnes
```
Escribe la descripción de menú para:
[CORTE DE CARNE Y PREPARACIÓN]

Incluir:
- Origen de la carne (si es especial)
- Punto de cocción ideal
- Acompañamientos
- Palabras como: jugoso, tierno, dorado, ahumado
```

### Platos Principales - Pescados
```
Describe este plato de pescado:
[NOMBRE DEL PESCADO Y PREPARACIÓN]

Enfatizar:
- Frescura
- Ligereza
- Técnica de cocción
- Acompañamiento
```

### Postres
```
Crea una descripción irresistible para este postre:
[NOMBRE Y COMPONENTES]

Usar palabras como:
- Indulgente, cremoso, sedoso, crujiente
- Contrastar texturas
- Mencionar temperatura ideal
```

## Platos Paraguayos Específicos

### Sopa Paraguaya
```
Escribe 3 versiones de descripción para Sopa Paraguaya:

1. Tradicional (para turistas)
2. Moderna (para público joven local)
3. Premium (para restaurante fine dining)

Cada una máximo 3 líneas.
```

### Chipa
```
Describe la chipa para un menú:

Versión 1: Chipa tradicional
Versión 2: Chipa rellena
Versión 3: Chipa so'o

Incluir textura, aroma, momento ideal para comerla.
```

### Asado
```
Crea descripción para un asado paraguayo:

Mencionar:
- Cortes incluidos
- Método de cocción (parrilla, tatakua, estaca)
- Tiempo de preparación (sin ser exacto)
- Acompañamientos
- Ambiente que evoca
```

### Mbejú
```
Describe el mbejú para diferentes contextos:

1. Como snack/entrada
2. Como acompañamiento
3. Para desayuno

Destacar: crocante, queso derretido, tradición.
```

## Adaptaciones Dietéticas

### Versión Vegetariana
```
Adapta la descripción de [PLATO] para su versión vegetariana:

Plato original: [descripción original]

La versión vegetariana usa: [ingredientes alternativos]

Mantener el atractivo y no sonar "compensatorio".
```

### Sin Gluten
```
Escribe la descripción para la opción sin gluten de [PLATO]:

Ingredientes: [listar]

La descripción debe:
- Sonar igual de apetitosa que la versión regular
- Indicar claramente que es sin gluten
- No hacer que parezca "especial" o "diferente"
```

### Vegano
```
Crea descripción atractiva para [PLATO] versión vegana:

Ingredientes plant-based usados: [listar]

Enfocarse en:
- Sabores y texturas
- Ingredientes frescos
- Beneficios sin sonar "saludable" de forma aburrida
```

## Menús Especiales

### Menú Ejecutivo
```
Escribe descripciones breves (1 línea) para menú ejecutivo:

Entrada: [plato]
Principal: [plato]
Postre: [plato]

Las descripciones deben ser concisas pero apetitosas.
Precio del combo: Gs. XXX.XXX
```

### Menú Degustación
```
Crea descripciones para menú degustación de [N] tiempos:

Cada descripción:
- Nombre creativo del tiempo
- Descripción de 2 líneas
- Maridaje sugerido

Tema del menú: [tradicional paraguayo/contemporáneo/fusión]
```

### Menú Niños
```
Escribe descripciones divertidas para menú infantil:

Platos: [listar]

Las descripciones deben:
- Usar lenguaje simple
- Ser atractivas para niños
- Mencionar ingredientes "escondidos" saludables de forma divertida
```

## Consejos de Uso

### Palabras Poderosas para Gastronomía
```
TEXTURA: crujiente, cremoso, suave, tierno, esponjoso
SABOR: intenso, delicado, equilibrado, fresco, profundo
TÉCNICA: ahumado, gratinado, a la parrilla, marinado
ORIGEN: artesanal, de la huerta, casero, regional
TEMPERATURA: humeante, tibio, refrescante, helado
```

### Evitar
```
❌ "Delicioso" (muy genérico)
❌ "Rico" (muy básico)
❌ "El mejor" (subjetivo)
❌ Descripciones largas (+4 líneas)
❌ Listar todos los ingredientes
```
