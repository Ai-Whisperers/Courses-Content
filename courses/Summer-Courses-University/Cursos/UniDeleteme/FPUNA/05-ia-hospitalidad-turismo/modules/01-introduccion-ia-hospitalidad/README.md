### 1.1 ¿Qué es la Inteligencia Artificial? (10 min)

#### Definición Simple

La Inteligencia Artificial (IA) es tecnología que permite a las computadoras realizar tareas que normalmente requieren inteligencia humana:

- **Entender lenguaje:** Comprender preguntas y textos
- **Generar contenido:** Escribir textos, crear imágenes
- **Aprender patrones:** Mejorar con el uso
- **Asistir decisiones:** Sugerir opciones basadas en datos

#### IA en la Vida Cotidiana

| Ejemplo | Dónde lo usamos |
|---------|-----------------|
| Autocorrector | WhatsApp, emails |
| Recomendaciones | Netflix, Spotify, Instagram |
| Traducción | Google Translate |
| Asistentes de voz | Siri, Alexa, Google |
| Mapas | Waze, Google Maps |

#### IA vs. Magia

```
❌ Lo que la IA NO es:
   • No piensa como un humano
   • No tiene sentimientos
   • No siempre tiene razón
   • No reemplaza la creatividad humana

✅ Lo que la IA SÍ es:
   • Una herramienta muy potente
   • Un asistente que ahorra tiempo
   • Un generador de ideas inicial
   • Un ayudante para tareas repetitivas
```

### 1.2 IA en la Industria de Hospitalidad (15 min)

#### Áreas de Impacto

| Área | Aplicación de IA | Beneficio |
|------|------------------|-----------|
| **Marketing** | Crear contenido para redes | Ahorro de tiempo, consistencia |
| **Atención al cliente** | Respuestas a reseñas, chatbots | Respuesta 24/7, profesionalismo |
| **Gastronomía** | Descripciones de menú, recetas | Menús más atractivos |
| **Ventas** | Propuestas, cotizaciones | Mayor conversión |
| **Operaciones** | Planificación, itinerarios | Eficiencia |

#### Estadísticas Relevantes (2024-2025)

- 67% de hoteles están explorando IA para servicio al cliente
- 54% de viajeros usaron chatbots en su última reserva
- Los restaurantes con buenos contenidos en redes venden 23% más
- Las respuestas rápidas a reseñas mejoran rating en 0.5 estrellas promedio

#### Ejemplos por Énfasis

**Gastronomía:**
```
• Descripciones de platos que despiertan apetito
• Menús para dietas especiales (vegano, sin gluten)
• Historias detrás de cada plato
• Respuestas a comentarios sobre comida
• Contenido para promociones diarias
```

**Hotelería:**
```
• Respuestas personalizadas a reseñas en Booking/TripAdvisor
• Emails de bienvenida y seguimiento
• Descripciones de habitaciones y servicios
• Propuestas para eventos corporativos
• Chatbots para preguntas frecuentes
```

**Turismo:**
```
• Itinerarios personalizados por tipo de viajero
• Descripciones de destinos y experiencias
• Paquetes turísticos creativos
• Contenido para vender Paraguay como destino
• Guías de viaje automatizadas
```

### 1.5 Primera Interacción con IA (10 min)

#### Paso a Paso: Tu Primer Prompt

**1. Acceder a ChatGPT**
```
1. Ir a chat.openai.com
2. Click en "Sign up" (crear cuenta) o "Log in" (si ya tiene)
3. Usar cuenta de Google o email
4. Aceptar términos
```

**2. Escribir tu primer prompt**

En la caja de texto, escribir:
```
Hola, soy estudiante de gestión de hospitalidad en Paraguay.
¿Cómo podrías ayudarme en mi carrera?
```

**3. Explorar la respuesta**

- Leer la respuesta completa
- Notar cómo estructura la información
- Identificar ideas útiles

#### Ejemplos de Primeros Prompts

**Para Gastronomía:**
```
Soy estudiante de gastronomía. Describe en 3 líneas
atractivas un plato de sopa paraguaya para un menú
de restaurante turístico.
```

**Para Hotelería:**
```
Soy estudiante de hotelería. Escribe una respuesta
profesional para un huésped que agradeció su estadía
en una reseña de Google.
```

**Para Turismo:**
```
Soy estudiante de turismo. Sugiere 3 actividades
únicas para hacer en Encarnación que un turista
argentino no encontraría fácilmente.
```

### 1.6 Consideraciones Éticas (5 min)

#### Principios Básicos

| Principio | Significado | Ejemplo |
|-----------|-------------|---------|
| **Autenticidad** | El contenido debe reflejar tu negocio real | No inventar reseñas falsas |
| **Transparencia** | Ser honesto sobre el uso de herramientas | No engañar a clientes |
| **Verificación** | Revisar antes de publicar | Corregir errores de IA |
| **Personalización** | Adaptar a tu estilo | No publicar sin editar |

#### Lo que NO Debes Hacer

```
❌ Usar IA para crear reseñas falsas
❌ Publicar contenido sin revisarlo
❌ Copiar textualmente sin adaptar
❌ Mentir sobre servicios o productos
❌ Ignorar errores o información incorrecta
```

#### Lo que SÍ Debes Hacer

```
✅ Usar IA como punto de partida
✅ Editar y personalizar todo
✅ Verificar información factual
✅ Mantener la voz auténtica del negocio
✅ Revisar antes de publicar
```
