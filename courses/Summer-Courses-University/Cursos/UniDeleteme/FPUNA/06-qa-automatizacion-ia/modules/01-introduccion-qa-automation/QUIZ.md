### Pregunta 1

¿Cuál es la principal ventaja de Playwright sobre Selenium?

- A) Playwright es más antiguo y tiene más documentación
- B) Playwright tiene auto-wait inteligente y es más rápido
- C) Playwright solo funciona con Chrome
- D) Selenium es más moderno

**Respuesta correcta:** B

---

### Pregunta 2

En la pirámide de testing, ¿cuál tipo de test debería haber en mayor cantidad?

- A) Tests E2E
- B) Tests de integración
- C) Tests unitarios
- D) Tests manuales

**Respuesta correcta:** C

---

### Pregunta 3

¿Qué comando se usa para inicializar un proyecto de Playwright?

- A) `npm install playwright`
- B) `npm init playwright@latest`
- C) `npx create-playwright`
- D) `playwright init`

**Respuesta correcta:** B

---

### Pregunta 4

¿Cuál es una ventaja principal de usar TypeScript para testing?

- A) Es más rápido que JavaScript
- B) No necesita compilación
- C) Detecta errores de tipo antes de ejecutar el código
- D) Solo funciona con Playwright

**Respuesta correcta:** C

---

### Pregunta 5

¿Qué archivo contiene la configuración principal de Playwright?

- A) `package.json`
- B) `tsconfig.json`
- C) `playwright.config.ts`
- D) `test.config.js`

**Respuesta correcta:** C

---

### Pregunta 6

¿Qué comando ejecuta los tests en modo interactivo con UI?

- A) `npx playwright test --headed`
- B) `npx playwright test --ui`
- C) `npx playwright test --interactive`
- D) `npx playwright run --ui`

**Respuesta correcta:** B

---

### Pregunta 7

En TypeScript, ¿cómo se declara una variable de tipo string?

- A) `var name = string;`
- B) `const name: string = 'valor';`
- C) `let name = 'valor' as String;`
- D) `string name = 'valor';`

**Respuesta correcta:** B

---

### Pregunta 8

¿Cuál es el ROI típico de implementar testing automatizado según la industria?

- A) 50-100% en 1 año
- B) 100-200% en 2 años
- C) 300-500% en 2 años
- D) No hay ROI medible

**Respuesta correcta:** C

---

### Pregunta 9

¿Qué navegadores soporta Playwright nativamente?

- A) Solo Chrome
- B) Chrome, Firefox, Safari, Edge
- C) Solo Chrome y Firefox
- D) Chrome, Firefox, Edge (no Safari)

**Respuesta correcta:** B

---

### Pregunta 10

Para ver el reporte HTML después de ejecutar tests, se usa:

- A) `npx playwright report`
- B) `npx playwright show-report`
- C) `npx playwright open-report`
- D) `npm run report`

**Respuesta correcta:** B
