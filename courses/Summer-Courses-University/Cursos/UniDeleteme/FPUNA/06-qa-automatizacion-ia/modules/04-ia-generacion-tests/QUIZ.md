# Quiz: Módulo 4
## IA para Generación de Tests

---

**Instrucciones:** Seleccione la respuesta correcta para cada pregunta.

---

### Pregunta 1

¿Cuál es la mejor forma de obtener sugerencias de GitHub Copilot para tests?

- A) Escribir el código completo y esperar correcciones
- B) Escribir comentarios descriptivos antes del código
- C) Copiar tests de internet
- D) Solo usar nombres de funciones

**Respuesta correcta:** B

---

### Pregunta 2

¿Cuáles son los componentes de un buen prompt para generar código de QA?

- A) Solo el HTML de la página
- B) Contexto, tarea específica, input y output esperado
- C) Solo el nombre de la funcionalidad
- D) Una lista de errores a encontrar

**Respuesta correcta:** B

---

### Pregunta 3

¿Por qué es importante revisar el código generado por IA?

- A) La IA siempre genera código perfecto
- B) La IA puede generar selectores incorrectos, tipos faltantes o lógica errónea
- C) Solo para agregar comentarios
- D) Para cambiar el formato del código

**Respuesta correcta:** B

---

### Pregunta 4

¿Qué biblioteca se recomienda para generar datos de prueba realistas?

- A) lodash
- B) axios
- C) faker.js (@faker-js/faker)
- D) moment

**Respuesta correcta:** C

---

### Pregunta 5

En una factory de datos, ¿qué permite el parámetro `overrides`?

- A) Eliminar campos del objeto
- B) Sobrescribir valores específicos manteniendo los demás por defecto
- C) Validar los datos generados
- D) Conectar con una base de datos

**Respuesta correcta:** B

---

### Pregunta 6

¿Cuál es el atajo de teclado para aceptar una sugerencia de GitHub Copilot?

- A) Enter
- B) Ctrl+A
- C) Tab
- D) Space

**Respuesta correcta:** C

---

### Pregunta 7

Al generar un Page Object con IA, ¿qué debes incluir en el prompt?

- A) Solo el nombre de la página
- B) El HTML de la página, requisitos y formato esperado
- C) Solo los selectores CSS
- D) El código de tests existente

**Respuesta correcta:** B

---

### Pregunta 8

¿Cuál es un error común en código generado por IA para Playwright?

- A) Usar demasiados comentarios
- B) Olvidar `await` en operaciones asíncronas
- C) Usar nombres muy largos
- D) Tener demasiados tests

**Respuesta correcta:** B

---

### Pregunta 9

¿Cuál es el flujo de trabajo recomendado al usar IA para QA?

- A) Generar → Usar directamente
- B) Generar → Revisar → Probar → Corregir → Refactorizar
- C) Escribir manual → Pedir a IA que mejore
- D) Solo usar IA para documentación

**Respuesta correcta:** B

---

### Pregunta 10

¿Por qué es útil crear presets en factories (como createAdminUser)?

- A) Para hacer el código más largo
- B) Para tener datos consistentes en casos de prueba específicos
- C) Porque la IA lo recomienda siempre
- D) Para evitar usar faker.js

**Respuesta correcta:** B
