### Pregunta 1

¿Cuál es la mejor manera de obtener sugerencias precisas de GitHub Copilot?

- A) Escribir código primero y esperar correcciones
- B) Escribir comentarios descriptivos antes del código
- C) Copiar código de internet
- D) Usar solo nombres de variables genéricos

**Respuesta correcta:** B

---

### Pregunta 2

¿Por qué es importante usar `np.random.seed()` en código de investigación?

- A) Hace el código más rápido
- B) Permite reproducir los mismos resultados aleatorios
- C) Es requerido por Python
- D) Genera mejores números aleatorios

**Respuesta correcta:** B

---

### Pregunta 3

¿Cuál es el formato recomendado para docstrings en código científico de Python?

- A) HTML
- B) Markdown
- C) NumPy/Google style
- D) No se necesitan docstrings

**Respuesta correcta:** C

---

### Pregunta 4

Al recibir código generado por IA, el investigador debe:

- A) Ejecutarlo inmediatamente sin revisar
- B) Confiar en que es correcto porque la IA lo generó
- C) Revisar, probar y verificar que produce resultados correctos
- D) Eliminarlo y escribir todo manualmente

**Respuesta correcta:** C

---

### Pregunta 5

¿Cuál es la estructura recomendada para un Jupyter Notebook científico?

- A) Solo código sin explicaciones
- B) Header → Setup → Data → Analysis → Results
- C) Resultados primero, luego código
- D) No hay estructura recomendada

**Respuesta correcta:** B

---

### Pregunta 6

¿Qué información debe incluir un archivo requirements.txt?

- A) Solo los nombres de las librerías
- B) Nombres de librerías con versiones específicas
- C) El código fuente de las librerías
- D) Solo librerías de pago

**Respuesta correcta:** B

---

### Pregunta 7

¿Cuándo es apropiado usar Claude/ChatGPT en lugar de GitHub Copilot?

- A) Siempre, son equivalentes
- B) Para explicaciones detalladas, debugging y código largo
- C) Solo para código simple
- D) Nunca, Copilot es siempre mejor

**Respuesta correcta:** B

---

### Pregunta 8

¿Cuál es una ventaja de modularizar código en funciones pequeñas?

- A) El código se ejecuta más lento
- B) Es más difícil de leer
- C) Facilita testing, debugging y reutilización
- D) Usa más memoria

**Respuesta correcta:** C

---

### Pregunta 9

Para ejecutar código en un cluster HPC, generalmente se necesita:

- A) Solo el script de Python
- B) Un script de submission (ej. SLURM) y el código
- C) Conexión a internet constante
- D) Instalar Python en el cluster manualmente

**Respuesta correcta:** B

### Pregunta 10

¿Qué debe hacer si el código generado por IA tiene un bug?

- A) Descartar todo y empezar de nuevo
- B) Pedir a la IA que explique el error y sugiera corrección
- C) Ignorar el error
- D) Publicar el código con el bug

**Respuesta correcta:** B
