# Biblioteca de Prompts: Revisión Bibliográfica
## Prompts para Búsqueda y Análisis de Literatura Científica

---

## 1. Formulación de Pregunta de Investigación

### 1.1 Formato PICO (Estudios Clínicos/Experimentales)

```
Tengo una idea de investigación sobre [TEMA].

Ayúdame a formular una pregunta de investigación usando el formato PICO:
- P (Población): ¿Quiénes son los sujetos de estudio?
- I (Intervención): ¿Qué se está evaluando?
- C (Comparación): ¿Con qué se compara?
- O (Outcome): ¿Qué resultado se mide?

Proporciona:
1. Pregunta PICO bien formulada
2. Versión en inglés para búsquedas
3. Sub-preguntas derivadas
```

### 1.2 Formato PEO (Estudios Observacionales)

```
Mi investigación es observacional sobre [TEMA].

Formula la pregunta usando formato PEO:
- P (Población): ¿A quién se estudia?
- E (Exposición): ¿Qué factor o condición?
- O (Outcome): ¿Qué resultado?

Incluye variaciones para búsqueda sistemática.
```

## 2. Generación de Términos de Búsqueda

### 2.1 Términos y Sinónimos

```
Para mi pregunta de investigación:
"[INSERTAR PREGUNTA]"

Genera:
1. 5-7 términos de búsqueda principales en inglés
2. 3 sinónimos para cada término
3. Términos MeSH relacionados (si aplica medicina/salud)
4. Términos específicos del campo
5. Combinaciones con operadores booleanos (AND, OR, NOT)
```

### 2.2 Estrategia de Búsqueda Completa

```
Diseña una estrategia de búsqueda sistemática para:
PREGUNTA: [X]
BASES DE DATOS: Semantic Scholar, PubMed, Scopus

Incluye:
1. Cadena de búsqueda para cada base de datos
2. Filtros recomendados (años, tipo de estudio)
3. Criterios de inclusión/exclusión sugeridos
4. Estimación de resultados esperados
```

## 3. Análisis de Papers

### 3.1 Análisis de Abstract

```
Analiza el siguiente abstract:

[PEGAR ABSTRACT]

Extrae:
1. Objetivo/hipótesis del estudio
2. Metodología utilizada
3. Tamaño de muestra (si se menciona)
4. Hallazgo principal
5. Conclusión de los autores
6. Limitaciones (mencionadas o inferidas)
7. Relevancia para mi investigación sobre [TEMA]
```

### 3.2 Análisis de Paper Completo

```
Analiza el siguiente paper científico:

TÍTULO: [X]
AUTORES: [Y]
REVISTA: [Z]
[PEGAR SECCIONES CLAVE]

Proporciona:
1. Resumen ejecutivo (100 palabras)
2. Fortalezas metodológicas
3. Debilidades/limitaciones
4. Cómo se relaciona con [MI TEMA]
5. Citas clave que puedo usar
6. Preguntas que deja sin responder
```

### 3.3 Comparación de Papers

```
Compara estos estudios sobre [TEMA]:

ESTUDIO 1:
- Título: [X]
- Método: [Y]
- Hallazgo: [Z]

ESTUDIO 2:
- Título: [X]
- Método: [Y]
- Hallazgo: [Z]

ESTUDIO 3:
- Título: [X]
- Método: [Y]
- Hallazgo: [Z]

Genera tabla comparativa con:
- Metodología
- Muestra
- Hallazgos
- Consistencia/contradicción
- Calidad metodológica
```

## 4. Síntesis de Literatura

### 4.1 Síntesis Narrativa

```
Basándote en los siguientes estudios sobre [TEMA]:

[LISTA DE ESTUDIOS CON HALLAZGOS PRINCIPALES]

Escribe una síntesis narrativa de 500 palabras que:
1. Identifique temas/patrones comunes
2. Señale contradicciones entre estudios
3. Muestre evolución del conocimiento
4. Identifique gaps en la literatura
5. Use conectores apropiados
6. Cite correctamente (Autor, año)
```

### 4.2 Identificación de Gaps

```
Basándome en mi revisión sobre [TEMA], he encontrado:

ESTUDIOS EXISTENTES:
[Listar principales estudios y hallazgos]

Identifica:
1. Preguntas sin responder en la literatura
2. Poblaciones no estudiadas
3. Metodologías no aplicadas
4. Contextos geográficos/culturales no explorados
5. Variables no consideradas
6. Relaciones no examinadas

Sugiere posibles preguntas de investigación para cada gap.
```

### 4.3 Mapa Conceptual

```
Crea un mapa conceptual en formato texto/markdown sobre [TEMA]:

Basándote en estos conceptos clave:
[LISTAR CONCEPTOS]

Estructura:
1. Concepto central
2. Ramas principales (3-5)
3. Sub-ramas (2-3 por rama)
4. Conexiones entre conceptos
5. Papers clave para cada rama
```

## 5. Gestión de Referencias

### 5.1 Verificar Referencia

```
Verifica la siguiente referencia bibliográfica:

[PEGAR REFERENCIA]

Comprueba:
1. ¿La información parece correcta?
2. ¿Existe este paper realmente? (ADVERTENCIA: No puedo verificar en tiempo real)
3. ¿El formato es correcto (APA/IEEE/etc)?
4. ¿Hay errores evidentes?

IMPORTANTE: Debo verificar independientemente en Google Scholar.
```

### 5.2 Formatear Referencias

```
Convierte estas referencias al formato [APA 7/IEEE/Vancouver]:

[PEGAR LISTA DE REFERENCIAS]

Verifica que incluyan:
- Todos los autores (o et al. si >7)
- Año
- Título completo
- Revista/Libro
- Volumen, número, páginas
- DOI si disponible
```

## 6. Crítica Metodológica

### 6.1 Evaluar Calidad de Estudio

```
Evalúa la calidad metodológica de este estudio:

TIPO DE ESTUDIO: [experimental/observacional/revisión]
METODOLOGÍA: [describir]
MUESTRA: [N y características]
ANÁLISIS: [tipo de análisis]

Evalúa según criterios:
□ Validez interna
□ Validez externa
□ Sesgo de selección
□ Sesgo de medición
□ Confusores controlados
□ Análisis apropiado
□ Conclusiones justificadas

Proporciona puntuación y justificación.
```

## 7. Uso Responsable

### Recordatorios Importantes

⚠️ **SIEMPRE verificar:**
- Las referencias generadas por IA pueden ser fabricadas
- Los resúmenes pueden omitir información crucial
- Los datos específicos deben verificarse en la fuente

✅ **Mejores prácticas:**
- Usar IA como punto de partida, no como fuente
- Verificar toda información en fuentes primarias
- Documentar qué prompts utilizó
- Mantener registro de proceso de revisión
