# Biblioteca de Prompts: Escritura Científica
## Prompts para Redacción de Papers y Manuscritos

---

## 1. Estructura General

### 1.1 Outline de Paper

```
Ayúdame a crear un outline para mi paper:

TÍTULO TENTATIVO: [X]
TIPO DE ESTUDIO: [experimental/observacional/revisión/teórico]
REVISTA OBJETIVO: [nombre, si conocida]
HALLAZGOS PRINCIPALES:
1. [Hallazgo 1]
2. [Hallazgo 2]
3. [Hallazgo 3]

Genera outline con:
1. Estructura IMRaD con subsecciones
2. Puntos clave para cada sección
3. Extensión sugerida (palabras)
4. Figuras/tablas necesarias
```

---

## 2. Sección: Introduction

### 2.1 Párrafo de Apertura

```
Escribe un párrafo de apertura para la introducción de mi paper sobre [TEMA].

CONTEXTO: [Describir área de investigación]
PROBLEMA: [Qué problema existe]
IMPORTANCIA: [Por qué importa]

El párrafo debe:
1. Captar atención del lector
2. Establecer contexto amplio
3. Transicionar hacia el problema específico
4. Ser conciso (100-150 palabras)
```

### 2.2 Revisión de Antecedentes

```
Sintetiza los antecedentes para mi introducción:

ESTUDIOS PREVIOS:
1. [Autor (año): hallazgo]
2. [Autor (año): hallazgo]
3. [Autor (año): hallazgo]

GAP IDENTIFICADO: [X]
MI CONTRIBUCIÓN: [Y]

Escribe 2-3 párrafos que:
1. Resuman el estado del conocimiento
2. Identifiquen el gap
3. Justifiquen la necesidad del estudio
```

### 2.3 Objetivos e Hipótesis

```
Formula objetivos e hipótesis para mi estudio:

PREGUNTA DE INVESTIGACIÓN: [X]
VARIABLES: [listar]
DISEÑO: [tipo de estudio]

Proporciona:
1. Objetivo general (1)
2. Objetivos específicos (2-4)
3. Hipótesis (si aplica)
4. Formato apropiado para cada uno
```

---

## 3. Sección: Methods

### 3.1 Descripción de Participantes

```
Redacta la sección de participantes/muestra:

N TOTAL: [X]
GRUPOS: [describir]
CRITERIOS INCLUSIÓN: [listar]
CRITERIOS EXCLUSIÓN: [listar]
RECLUTAMIENTO: [método]
APROBACIÓN ÉTICA: [comité, número]

Formato estándar para revista científica.
```

### 3.2 Descripción de Procedimiento

```
Redacta la sección de procedimiento:

PASOS DEL ESTUDIO:
1. [Paso 1 con detalle]
2. [Paso 2 con detalle]
3. [Paso 3 con detalle]

INSTRUMENTOS USADOS: [listar]
DURACIÓN: [tiempo]
CONDICIONES: [ambiente, controles]

Usar pasado, voz pasiva cuando apropiado.
```

### 3.3 Análisis Estadístico

```
Redacta la sección de análisis estadístico:

VARIABLE DEPENDIENTE: [X]
VARIABLES INDEPENDIENTES: [listar]
PRUEBAS USADAS:
1. [Prueba]: para [propósito]
2. [Prueba]: para [propósito]

SOFTWARE: [nombre, versión]
ALFA: [0.05 u otro]

Incluir justificación de cada prueba.
```

---

## 4. Sección: Results

### 4.1 Estadísticas Descriptivas

```
Redacta la presentación de estadísticas descriptivas:

DATOS:
- Grupo 1 (n=X): M=Y, SD=Z
- Grupo 2 (n=X): M=Y, SD=Z

Formato: APA 7
Incluir: Referencia a tabla correspondiente
```

### 4.2 Resultados de Pruebas Estadísticas

```
Redacta el reporte de resultados estadísticos:

PRUEBA: [t-test/ANOVA/correlación/etc.]
ESTADÍSTICO: [valor]
GRADOS DE LIBERTAD: [si aplica]
P-VALOR: [valor]
TAMAÑO DEL EFECTO: [valor]
IC 95%: [intervalo]

Formato: APA 7
Incluir interpretación breve.
```

### 4.3 Descripción de Figuras

```
Escribe la leyenda y descripción para esta figura:

TIPO DE FIGURA: [barras/dispersión/líneas/etc.]
MUESTRA: [qué datos]
EJES: X=[variable], Y=[variable]
ELEMENTOS: [barras de error, líneas, etc.]

La leyenda debe ser auto-explicativa.
```

---

## 5. Sección: Discussion

### 5.1 Interpretación de Resultados

```
Ayúdame a interpretar estos resultados:

HALLAZGO: [describir resultado principal]
ESTUDIOS PREVIOS: [qué decía la literatura]
TEORÍA: [marco teórico relevante]

Escribe 2 párrafos que:
1. Interpreten el hallazgo en contexto
2. Comparen con literatura previa
3. Expliquen mecanismos posibles
```

### 5.2 Limitaciones

```
Redacta la sección de limitaciones:

LIMITACIONES IDENTIFICADAS:
1. [Limitación 1]
2. [Limitación 2]
3. [Limitación 3]

Para cada una:
- Descripción clara
- Impacto potencial en resultados
- Cómo se intentó mitigar (si aplica)
```

### 5.3 Futuras Direcciones

```
Sugiere direcciones de investigación futura basándote en:

HALLAZGOS: [principales resultados]
LIMITACIONES: [del estudio actual]
GAPS RESTANTES: [en la literatura]

Proporciona 3-5 sugerencias específicas y justificadas.
```

## 6. Abstract

### 6.1 Abstract Estructurado

```
Genera un abstract estructurado (máx. 250 palabras):

CONTEXTO: [1-2 oraciones]
OBJETIVO: [1 oración]
MÉTODOS: [2-3 oraciones]
RESULTADOS: [2-3 oraciones con datos]
CONCLUSIÓN: [1-2 oraciones]

Incluir: Palabras clave (5-7)
```

### 6.2 Abstract No Estructurado

```
Genera un abstract de párrafo único (máx. 200 palabras):

Contenido a incluir:
- Objetivo: [X]
- Método: [Y]
- Resultados: [Z]
- Conclusión: [W]

Fluido, conciso, auto-contenido.
```

---

## 7. Mejora de Redacción

### 7.1 Mejora de Claridad

```
Mejora la claridad de este párrafo:

[PEGAR TEXTO]

Mantén el significado pero:
1. Simplifica oraciones largas
2. Elimina redundancias
3. Usa términos precisos
4. Mejora transiciones
5. Indica qué cambios hiciste
```

### 7.2 Reducción de Palabras

```
Reduce este texto a [X] palabras manteniendo información esencial:

[PEGAR TEXTO ACTUAL: Y palabras]

Priorizar: [qué información es más importante]
```

### 7.3 Mejora de Estilo Académico

```
Convierte este texto a estilo académico formal:

[PEGAR TEXTO INFORMAL]

Cambios a realizar:
1. Eliminar contracciones
2. Usar voz pasiva donde apropiado
3. Evitar primera persona (si requerido)
4. Usar terminología técnica correcta
```

## 8. Respuesta a Revisores

### 8.1 Respuesta Estándar

```
Ayúdame a responder a este comentario de revisor:

COMENTARIO:
"[Pegar comentario textual]"

MI ESTUDIO:
[Contexto relevante]

CAMBIO POSIBLE:
[Lo que podría hacer]

Genera respuesta que:
1. Agradezca el comentario
2. Responda punto por punto
3. Indique cambios realizados
4. Sea profesional y diplomático
```

### 8.2 Respuesta a Crítica

```
El revisor critica:
"[Pegar crítica]"

Mi posición es que [explicar]:

Ayúdame a:
1. Responder respetuosamente
2. Defender mi posición con evidencia
3. Ofrecer compromiso si es posible
4. Mantener tono profesional
```

---

## 9. Cover Letter

```
Genera una cover letter para envío a revista:

REVISTA: [nombre]
TÍTULO DEL PAPER: [X]
TIPO DE ARTÍCULO: [original/revisión/carta]
HALLAZGO PRINCIPAL: [Y]
RELEVANCIA PARA LA REVISTA: [Z]

Incluir:
1. Presentación del manuscrito
2. Por qué es relevante para la revista
3. Declaración de originalidad
4. Conflictos de interés
5. Cierre profesional
```
