## 2. Herramientas de Búsqueda Bibliográfica

### Semantic Scholar

| Aspecto | Detalle |
|---------|---------|
| **URL** | semanticscholar.org |
| **Fortaleza** | Búsqueda semántica con IA, TLDR de papers |
| **Base de datos** | 200+ millones de papers |
| **Costo** | Gratuito |
| **API** | Disponible |

**Mejor para:** Búsqueda inicial, descubrimiento de literatura, TLDRs

### Connected Papers

| Aspecto | Detalle |
|---------|---------|
| **URL** | connectedpapers.com |
| **Fortaleza** | Visualización de redes de citas |
| **Uso** | Identificar papers relacionados |
| **Costo** | Gratuito (límites) / Premium |

**Mejor para:** Explorar conexiones entre papers, encontrar literatura seminal

### Elicit

| Aspecto | Detalle |
|---------|---------|
| **URL** | elicit.org |
| **Fortaleza** | Extracción estructurada de información |
| **Uso** | Tablas de síntesis automáticas |
| **Costo** | Gratuito (límites) / Premium |

**Mejor para:** Crear matrices de síntesis, extraer datos de múltiples papers

### Consensus

| Aspecto | Detalle |
|---------|---------|
| **URL** | consensus.app |
| **Fortaleza** | Síntesis de consenso científico |
| **Uso** | Respuestas basadas en evidencia |
| **Costo** | Gratuito (límites) / Premium |

**Mejor para:** Verificar afirmaciones, buscar consenso en literatura

### Research Rabbit

| Aspecto | Detalle |
|---------|---------|
| **URL** | researchrabbit.ai |
| **Fortaleza** | Descubrimiento personalizado |
| **Uso** | Alertas y recomendaciones |
| **Costo** | Gratuito |

**Mejor para:** Mantenerse actualizado, descubrir nuevos papers

## 7. Flujo de Trabajo Recomendado

### Fase 1: Revisión Bibliográfica

```
Semantic Scholar (búsqueda inicial)
         ↓
Connected Papers (expandir red)
         ↓
Elicit (extraer información)
         ↓
Claude (analizar y sintetizar)
         ↓
Zotero (organizar referencias)
```

### Fase 2: Análisis de Datos

```
Claude/ChatGPT (diseñar análisis)
         ↓
GitHub Copilot (escribir código)
         ↓
Claude (debugging si necesario)
         ↓
Python/R (ejecutar)
         ↓
Claude (interpretar resultados)
```

### Fase 3: Escritura

```
Claude (estructura y borrador)
         ↓
Writefull (estilo académico)
         ↓
Grammarly (gramática final)
         ↓
Claude (respuesta a revisores)
```

## 8. Costos y Planes Académicos

### Herramientas Gratuitas (Esenciales)

| Herramienta | Plan Gratuito |
|-------------|---------------|
| Semantic Scholar | Completo |
| Connected Papers | 5 graphs/mes |
| Zotero | 300 MB storage |
| Claude | Límites diarios |
| ChatGPT | GPT-3.5, límites |

### Herramientas con Descuento Académico

| Herramienta | Descuento |
|-------------|-----------|
| GitHub Copilot | Gratis con GitHub Education |
| Overleaf | Premium gratis con institución |
| Grammarly | Descuento educativo |

### Inversión Recomendada

**Presupuesto limitado:**
- Todo gratuito es viable
- Priorizar: Semantic Scholar, Zotero, Claude gratis

**Presupuesto medio (~$20/mes):**
- Claude Pro O ChatGPT Plus
- Elicit premium

**Presupuesto completo (~$50/mes):**
- Claude Pro + ChatGPT Plus
- Elicit + Consensus premium
- Writefull premium
