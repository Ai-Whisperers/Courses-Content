## Materiales y Recursos

### Equipamiento por Estudiante

- Laptop con:
  - Arduino IDE
  - TIA Portal (licencia educativa)
  - Python 3.x + bibliotecas
  - VS Code + GitHub Copilot

- Kit de desarrollo:
  - Arduino Uno o ESP32
  - Sensores básicos (temperatura, movimiento)
  - Protoboard y componentes

### Recursos del Laboratorio

- PLCs Siemens S7-1200 (compartidos)
- Paneles HMI
- Osciloscopios digitales
- Módulos de entrenamiento
