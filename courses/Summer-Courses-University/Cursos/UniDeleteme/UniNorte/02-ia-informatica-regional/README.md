## Descripción

Curso de desarrollo de software con IA enfocado en las oportunidades del mercado paraguayo y regional. Incluye contexto de ambas sedes (Asunción y frontera Brasil) para desarrollar soluciones adaptadas a diferentes realidades.

**Transformación esperada:** Los participantes desarrollarán software profesional un 50% más rápido utilizando herramientas de IA de vanguardia.

---

## Objetivos de Aprendizaje

Al finalizar el curso, los participantes podrán:

1. **Desarrollar aplicaciones** web/móviles con IA
2. **Utilizar GitHub Copilot** eficientemente
3. **Crear APIs** y backends con asistencia IA
4. **Implementar testing** automatizado con IA
5. **Documentar proyectos** con herramientas de IA
6. **Desplegar aplicaciones** en la nube

---

## Contenido del Curso

### Día 1: Desarrollo Frontend/Backend con IA (6 horas)

#### Módulo 1: Introducción a IA para Desarrollo (1.5 horas)
- Herramientas de IA para programadores
- GitHub Copilot, Claude, ChatGPT
- Cursor IDE y alternativas
- Mejores prácticas y limitaciones
- **Práctica:** Configurar entorno de desarrollo

#### Módulo 2: Frontend con IA (2 horas)
- React/Next.js con Copilot
- Componentes UI con IA
- Estilos CSS con asistencia
- Responsive design automatizado
- **Práctica:** Crear interfaz de usuario

#### Módulo 3: Backend con IA (2.5 horas)
- Node.js/Express con IA
- APIs RESTful asistidas
- Base de datos y ORM
- Autenticación con IA
- **Práctica:** Desarrollar API completa

---

### Día 2: Testing, Documentación y Deploy (6 horas)

#### Módulo 4: Testing con IA (2 horas)
- Unit tests generados por IA
- Integration tests automatizados
- E2E testing con Playwright + IA
- Cobertura y calidad de código
- **Práctica:** Suite de tests completa

#### Módulo 5: Documentación y DevOps (2 horas)
- README y documentación con IA
- CI/CD pipelines asistidos
- Docker y containerización
- Deploy en Vercel/Railway
- **Práctica:** Pipeline completo

#### Módulo 6: Proyecto Integrador (2 horas)
- Aplicación full-stack completa
- Testing, documentación, deploy
- **Evaluación:** Demo del proyecto
