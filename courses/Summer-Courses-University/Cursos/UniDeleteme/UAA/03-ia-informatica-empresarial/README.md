## Descripción

Curso que integra las competencias de informática con la gestión empresarial, enseñando a utilizar IA para desarrollar soluciones tecnológicas que resuelvan problemas de negocios. Combina desarrollo de software con visión empresarial ACBSP.

**Transformación esperada:** Los participantes podrán desarrollar soluciones empresariales con IA un 50% más rápido.

---

## Objetivos de Aprendizaje

Al finalizar el curso, los participantes podrán:

1. **Desarrollar** aplicaciones empresariales con asistencia de IA
2. **Automatizar** procesos de negocio con scripts
3. **Crear** dashboards y reportes automatizados
4. **Integrar** APIs de IA en sistemas existentes
5. **Diseñar** soluciones tecnológicas para PYMES
6. **Documentar** proyectos técnicos con IA

---

## Contenido del Curso

### Día 1: Desarrollo con IA para Negocios (6 horas)

#### Módulo 1: IA para Desarrollo Empresarial (2 horas)
- GitHub Copilot para desarrollo rápido
- APIs de IA: OpenAI, Claude, Google
- Arquitectura de soluciones empresariales
- **Práctica:** Configurar entorno de desarrollo con IA

#### Módulo 2: Automatización de Procesos (2 horas)
- Scripts Python para tareas repetitivas
- Automatización de Excel/Sheets con IA
- Integración con ERPs y sistemas
- **Práctica:** Automatizar proceso contable

#### Módulo 3: Dashboards y Reportes (2 horas)
- Power BI / Google Data Studio con IA
- Conexión a bases de datos
- Visualizaciones inteligentes
- **Práctica:** Crear dashboard de ventas

---

### Día 2: Soluciones Empresariales (6 horas)

#### Módulo 4: APIs y Integración (2 horas)
- Consumir APIs de IA desde aplicaciones
- Chatbots empresariales
- Automatización de emails y notificaciones
- **Práctica:** Crear chatbot de atención

#### Módulo 5: Soluciones para PYMES (2 horas)
- Diagnóstico de necesidades tecnológicas
- Soluciones low-cost con IA
- Implementación ágil
- **Práctica:** Propuesta para PYME paraguaya

#### Módulo 6: Proyecto Integrador (2 horas)
- Desarrollar solución empresarial completa
- Documentación técnica con IA
- **Evaluación:** Demostración del proyecto
