# Ejercicio Práctico - Módulo 02
## Configuración y Comparación de Herramientas

---

## Objetivo

Configurar al menos 2 herramientas de IA generativa y comparar sus respuestas ante la misma pregunta.

**Duración:** 15 minutos
**Tipo:** Práctico de configuración

---

## Parte 1: Configuración de Cuentas (8 min)

### Paso 1.1: Primera Herramienta - ChatGPT

1. Ve a [chat.openai.com](https://chat.openai.com)
2. Crea una cuenta (email, Google o Microsoft)
3. Verifica tu email
4. Envía un mensaje de prueba:

```
Hola, estoy probando la herramienta.
¿Puedes confirmar que funciona respondiendo con un emoji?
```

## Parte 2: Comparación de Respuestas (5 min)

### Ejercicio: La Misma Pregunta en Diferentes Herramientas

Envía **exactamente el mismo prompt** a ChatGPT y Claude:

```
Explica en 3 puntos breves qué es la computación en la nube.
Usa un lenguaje simple que un estudiante de secundaria pueda entender.
Formato: lista numerada.
```

### Registro de Respuestas

**Respuesta de ChatGPT:**
```
1. _______________________________________________
2. _______________________________________________
3. _______________________________________________
```

**Respuesta de Claude:**
```
1. _______________________________________________
2. _______________________________________________
3. _______________________________________________
```

### Análisis Comparativo

| Aspecto | ChatGPT | Claude |
|---------|---------|--------|
| ¿Siguió el formato pedido? | Sí / No | Sí / No |
| ¿Lenguaje simple? | 1-5 | 1-5 |
| ¿Información correcta? | Sí / No | Sí / No |
| ¿Cuál preferiste? | | |
