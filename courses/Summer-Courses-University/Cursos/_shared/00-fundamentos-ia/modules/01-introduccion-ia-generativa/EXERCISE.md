# Ejercicio Práctico - Módulo 01
## Primera Interacción con IA Generativa

---

## Objetivo

Experimentar por primera vez con una herramienta de IA generativa, observando sus capacidades y limitaciones básicas.

**Duración:** 15 minutos
**Tipo:** Exploratorio guiado
**Herramientas:** ChatGPT, Claude o Gemini (cualquiera disponible)

---

## Parte 1: Primera Conversación (5 min)

### Paso 1.1: Acceder a la Herramienta

Abre una de estas opciones (la que tengas disponible):
- [ChatGPT](https://chat.openai.com) - Cuenta gratuita
- [Claude](https://claude.ai) - Cuenta gratuita
- [Gemini](https://gemini.google.com) - Con cuenta Google

### Paso 1.2: Saludo y Exploración

Escribe exactamente:

```
Hola, soy estudiante de [TU CARRERA] en Paraguay.
¿Qué puedes hacer para ayudarme en mis estudios?
```

**Anota:** ¿Qué tipo de ayuda ofrece? ¿Es relevante para tu carrera?

```
Tu observación:
_________________________________________________
_________________________________________________
```

### Paso 1.3: Pregunta Específica

Ahora pregunta algo de tu área:

```
Dame 3 ejemplos de cómo la IA está cambiando [TU INDUSTRIA]
```

Ejemplos de industrias:
- Desarrollo de software
- Ingeniería electrónica
- Medicina
- Administración de empresas
- Agricultura

**Anota:** ¿Los ejemplos son actuales y relevantes?

```
Tu observación:
_________________________________________________
_________________________________________________
```

---

## Parte 2: Explorando Capacidades (5 min)

### Ejercicio 2.1: Generación de Texto

Prueba este prompt:

```
Escribe un párrafo corto explicando la importancia de
[UN CONCEPTO DE TU CARRERA] para alguien que no sabe nada del tema.
```

Evalúa la respuesta:
- [ ] ¿Es correcta técnicamente?
- [ ] ¿Es fácil de entender?
- [ ] ¿Usarías esta explicación?

### Ejercicio 2.2: Traducción y Adaptación

```
Traduce este texto al inglés y luego simplifícalo
para un niño de 12 años:

"La inteligencia artificial generativa utiliza redes neuronales
profundas para crear contenido original basado en patrones
aprendidos de grandes conjuntos de datos de entrenamiento."
```

Compara:
- Traducción técnica vs. versión simplificada
- ¿Mantiene el significado esencial?

### Ejercicio 2.3: Formato y Estructura

```
Crea una tabla comparando 3 lenguajes de programación
(o 3 herramientas de tu área) con estas columnas:
- Nombre
- Ventajas
- Desventajas
- Cuándo usar
```

Observa:
- [ ] ¿Generó una tabla correctamente?
- [ ] ¿La información parece precisa?

---

## Parte 3: Descubriendo Limitaciones (5 min)

### Ejercicio 3.1: Información Actualizada

Pregunta algo muy reciente:

```
¿Cuál fue la noticia más importante de Paraguay de esta semana?
```

**Resultado esperado:** La IA probablemente no pueda responder con información actual.

```
¿Qué respondió?
_________________________________________________
```

### Ejercicio 3.2: Forzando un Error (Alucinación)

Pregunta algo inventado:

```
¿Puedes explicarme la teoría de Mendélez-Rodriguez
sobre computación cuántica?
```

**Nota:** Esta teoría NO existe. Observa si la IA:
- [ ] Admite que no conoce el término
- [ ] Inventa una explicación (alucinación)
- [ ] Pide aclaración

```
¿Qué hizo la IA?
_________________________________________________
```

### Ejercicio 3.3: Límites de Conocimiento

```
Resuelve esta integral: ∫(x³ + 2x² - 5x + 3)dx

Ahora muéstrame el gráfico de la función.
```

Observa:
- [ ] ¿Resolvió la integral correctamente?
- [ ] ¿Pudo mostrar un gráfico real?
