# Ejercicio 1: Crear Contenido para Redes Sociales con IA

## Objetivo
Aprender a usar IA para generar contenido de redes sociales adaptado al público paraguayo.

## Duración
40-50 minutos

## Prerrequisitos
- Cuenta en al menos una red social
- Conocimiento básico de marketing digital
- Acceso a Claude

## Advertencia Importante

**El contenido generado por IA debe ser:**
- Verificado por hechos (la IA puede inventar datos)
- Adaptado a tu voz de marca
- Revisado antes de publicar
- Nunca publicado automáticamente sin revisión

---

## Parte 1: El Problema del Contenido Genérico (10 minutos)

### El escenario

Eres community manager de una cafetería en Asunción. Necesitas crear 5 posts para la semana.

### Prompt INCORRECTO

Prueba este prompt vago:
```
Hazme posts para una cafetería
```

### Observa el resultado
- [ ] ¿Los posts suenan paraguayos?
- [ ] ¿Mencionan precios en guaraníes?
- [ ] ¿Usan expresiones locales?
- [ ] ¿Son apropiados para Instagram/Facebook/WhatsApp?
- [ ] ¿Puedes publicarlos tal cual?

---

## Parte 2: Prompt con Contexto Local (15 minutos)

### Prompt CORRECTO

```
Soy community manager de "Café del Centro", una cafetería
en el microcentro de Asunción, Paraguay.

**Sobre el negocio:**
- Público: oficinistas y estudiantes (25-45 años)
- Precio promedio: ₲25,000-40,000
- Diferenciador: café de especialidad paraguayo
- Horario: 7:00-19:00, lunes a sábado

**Para esta semana necesito 5 posts de Instagram:**

1. **Lunes:** Motivacional para empezar la semana
2. **Martes:** Promoción del "combo oficina" (café + chipá)
3. **Miércoles:** Educativo sobre café paraguayo
4. **Jueves:** Testimonio de cliente (inventar uno típico)
5. **Viernes:** Anticipación del fin de semana

**Requisitos:**
- Tono: cercano, cálido, profesional
- Incluir: emojis moderados (3-4 por post)
- Hashtags: 5-7 relevantes para Paraguay
- Precio en guaraníes cuando corresponda
- Call to action en cada uno
- Largo: 100-150 palabras máximo

**NO usar:**
- Expresiones de otros países (padre/chido/tío)
- Precios en dólares o pesos
- Referencias culturales no paraguayas
```

### Compara los resultados

| Aspecto | Prompt vago | Prompt con contexto |
|---------|-------------|---------------------|
| Relevancia local | /5 | /5 |
| Tono apropiado | /5 | /5 |
| Usable sin editar | /5 | /5 |
| Hashtags buenos | /5 | /5 |

## Parte 3: Adaptar para Diferentes Plataformas (10 minutos)

### El mismo mensaje, diferentes formatos

```
Tengo este contenido base para promocionar el "combo oficina":

"Café de especialidad paraguayo + chipá recién horneado
Por solo ₲28,000. Disponible de 7:00 a 10:00 am."

Adáptalo para:

1. **Instagram Feed:**
   - Caption atractivo con emojis
   - Hashtags para descubrimiento
   - Call to action para guardar/compartir

2. **Instagram Stories:**
   - Texto corto (lectura en 3 segundos)
   - Sugerencia de sticker/encuesta
   - Swipe up o link en bio

3. **WhatsApp Status:**
   - Mensaje directo y personal
   - Como si hablara un amigo
   - Máximo 2 líneas

4. **Facebook:**
   - Más texto explicativo
   - Pregunta para generar comentarios
   - Horario de publicación sugerido

5. **Google My Business:**
   - Post de oferta con detalles
   - Botón de acción sugerido
```

## Parte 4: Crear Variaciones A/B (10 minutos)

### Pide variaciones para testear

```
Para el post del "combo oficina", necesito 3 variaciones
diferentes para hacer test A/B:

**Variación A:** Enfocado en el precio/ahorro
**Variación B:** Enfocado en la calidad del café
**Variación C:** Enfocado en la conveniencia (cerca de la oficina)

Para cada variación:
1. Caption completo
2. Sugerencia de imagen/visual
3. Mejor hora para publicar
4. Métrica clave a medir

¿Cuál de las tres variaciones crees que funcionará mejor
para oficinistas en el microcentro de Asunción? ¿Por qué?
```

### Analiza las variaciones

| Variación | Gancho principal | ¿Por qué funcionaría? |

| A (precio) | | |

## Parte 5: Calendario de Contenido (10 minutos)

### Pide un calendario semanal

```
Crea un calendario de contenido para "Café del Centro"
para una semana típica.

**Formato:**
| Día | Hora | Plataforma | Tipo de contenido | Descripción |
|-----|------|------------|-------------------|-------------|

**Incluir:**
- 2 posts diarios (mañana y tarde)
- Mix de plataformas (Instagram, Facebook, WhatsApp)
- Variedad de formatos (feed, stories, reels)
- Contenido evergreen vs temporal

**Fechas especiales de Paraguay a considerar:**
- Si estamos cerca de San Juan, incluir algo relacionado
- Fechas patrias si aplica
- Ningún contenido polémico religioso/político

**Restricciones:**
- No publicar antes de las 7:00 am
- No publicar después de las 21:00
- Domingo solo 1 post (si acaso)
```

```

- Público: ________________ (edad, perfil)

- [ ] Promocional

**Especificaciones:**

**Evitar:**
- ________________
- ________________

**Referencia visual:**
[Descripción de imagen/video deseado]

Genera el contenido completo listo para publicar.
```

## Errores Comunes a Evitar

### La IA puede cometer estos errores:

- [ ] **Expresiones de otros países:** "Está padrísimo" (México), "Mola" (España)
- [ ] **Monedas incorrectas:** Precios en dólares o pesos
- [ ] **Hashtags incorrectos:** #BuenosAires cuando es #Asuncion
- [ ] **Horarios incorrectos:** Sugerir publicar a las 5pm cuando Paraguay no tiene horario de verano igual
- [ ] **Referencias culturales:** Mencionar festividades de otros países
- [ ] **Datos inventados:** "El 80% de los paraguayos prefiere..." (verificar siempre)

### Siempre revisar:

1. ¿Suena natural para un paraguayo?
2. ¿Los precios son realistas para el mercado local?
3. ¿Los hashtags tienen tracción en Paraguay?
4. ¿La referencia cultural es correcta?
5. ¿El call to action es apropiado para la plataforma?

## Entregable

### Pack de contenido para una semana

```markdown
# Contenido Semanal - [Nombre del negocio]
## Semana del [fecha] al [fecha]

### Lunes
**Instagram (12:00):**
[Caption completo]
Visual: [Descripción]
Hashtags: [lista]

**WhatsApp Status (8:00):**
[Mensaje]

### Martes
[Similar estructura]

### [Resto de la semana]

---
## Notas
- Verificar precios antes de publicar
- Adaptar si hay evento especial
- Revisar métricas el lunes siguiente
```
