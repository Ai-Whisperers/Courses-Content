# SETUP-AI.md - Configuración de Proyectos de Marketing con IA

## Prompts para que la IA Configure tu Estrategia de Marketing

---

## 1. Setup Inicial de Marca

```bash
claude "Configura el proyecto de marketing para mi negocio:

Nombre: [nombre del negocio]
Industria: [sector]
Ubicación: [ciudad, Paraguay]
Tiempo en el mercado: [X años/meses]

Crea:
1. Estructura de carpetas para assets y contenido
2. Template de brand guidelines básico
3. Checklist de presencia digital
4. Calendario de contenido vacío (mensual)
5. Plantilla de métricas a seguir
6. Documento de buyer persona"
```

---

## 2. Definición de Buyer Persona

```bash
claude "Crea el buyer persona principal:

Negocio: [descripción]
Producto/servicio más vendido: [producto]
Precio típico: ₲[X]
Clientes actuales: [descripción general]

Genera perfil completo:
1. Nombre ficticio representativo
2. Datos demográficos (edad, género, ubicación, ocupación)
3. Nivel socioeconómico e ingresos
4. Comportamiento de compra
5. Pain points y frustraciones
6. Motivaciones y deseos
7. Canales donde consume información
8. Objeciones típicas antes de comprar
9. Palabras/frases que usa
10. Un día típico en su vida"
```

---

## 3. Auditoría de Presencia Digital

```bash
claude "Audita mi presencia digital actual:

Redes sociales:
- Instagram: @[usuario] - [X] seguidores
- Facebook: [página] - [X] seguidores
- TikTok: @[usuario] - [X] seguidores

Website: [url o 'no tengo']
Google My Business: [sí/no]
Email marketing: [herramienta o 'no tengo']

Evalúa:
1. Consistencia de marca entre plataformas
2. Frecuencia de publicación actual
3. Engagement rate estimado
4. Oportunidades no aprovechadas
5. Prioridades de mejora (top 5)
6. Quick wins inmediatos"
```

---

## 4. Voz y Tono de Marca

```bash
claude "Define la voz de marca:

Negocio: [descripción]
Audiencia: [descripción del buyer persona]
Personalidad deseada: [profesional/amigable/divertido/etc.]

Genera:
1. Descripción de la marca como persona
2. Adjetivos que definen la voz (5)
3. Tono en diferentes situaciones:
   - Post educativo
   - Promoción/venta
   - Respuesta a cliente
   - Situación de crisis
4. Palabras y frases a usar
5. Palabras y frases a evitar
6. Ejemplos de copy en la voz definida"
```

---

## 5. Estrategia de Contenido

```bash
claude "Diseña estrategia de contenido:

Negocio: [tipo]
Objetivo principal: [awareness/engagement/conversión]
Plataforma principal: [Instagram/Facebook/TikTok]
Recursos disponibles: [tiempo, equipo, presupuesto]

Genera:
1. Pilares de contenido (3-4 temas principales)
2. Proporción de contenido:
   - Educativo: X%
   - Entretenimiento: X%
   - Venta: X%
   - User-generated: X%
3. Formatos recomendados por pilar
4. Frecuencia de publicación realista
5. Ejemplos de contenido por pilar
6. Calendario semanal tipo"
```

---

## 6. Setup de Email Marketing

```bash
claude "Configura estrategia de email marketing:

Negocio: [tipo]
Base actual: [X suscriptores o 'empezando de cero']
Herramienta: [Mailchimp/Brevo/otra]

Genera:
1. Lead magnet recomendado para captar emails
2. Secuencia de bienvenida (5 emails - estructura)
3. Tipos de emails para calendario mensual:
   - Newsletter semanal
   - Promociones
   - Contenido exclusivo
4. Segmentación recomendada
5. Métricas a seguir (open rate, CTR, etc.)
6. Buenas prácticas de asunto"
```

## 7. Configuración de Meta Ads

```bash
claude "Planifica estructura de Meta Ads:

Negocio: [tipo]
Objetivo: [awareness/tráfico/leads/ventas]
Presupuesto mensual: ₲[X]
Experiencia previa: [ninguna/básica/intermedia]

1. Estructura de campaña recomendada
2. Audiencias a crear:
   - Lookalike de clientes
   - Intereses específicos
   - Remarketing
3. Tipos de anuncios a probar

5. Métricas clave a monitorear
6. Proceso de optimización semanal"
```

## 8. SEO Local Setup

```bash
claude "Configura SEO local:

Negocio: [nombre]
Dirección: [dirección completa]
Categoría: [tipo de negocio]
Servicios: [lista de servicios]
Área de servicio: [ciudades/barrios]

Genera:
1. Keywords principales (10)
2. Keywords long-tail (15)
3. Optimización de Google My Business:
   - Descripción
   - Categorías secundarias
   - Atributos a agregar
4. Estrategia de reseñas
5. Schema markup para website (si aplica)
6. Contenido local a crear"
```

---

## 9. Setup de Métricas y KPIs

```bash
claude "Configura dashboard de métricas:

Negocio: [tipo]
Canales activos: [lista]
Objetivos: [lista de objetivos]

Define:
1. KPIs por canal:
   - Redes sociales (reach, engagement, growth)
   - Website (tráfico, bounce, conversión)
   - Email (open, click, unsubscribe)
   - Ads (CPC, CTR, ROAS)
2. Frecuencia de medición
3. Herramientas para medir cada métrica
4. Template de reporte mensual
5. Alertas (qué número es preocupante)
6. Benchmarks de la industria"
```

---


## 11. Respuestas y Atención al Cliente

```bash
claude "Configura sistema de respuestas:

Negocio: [tipo]
Canales de atención: [WhatsApp/DM Instagram/etc.]
Horarios: [horarios de atención]
Preguntas más frecuentes: [lista]

1. Saludo inicial estándar
2. Respuestas para cada FAQ
3. Respuesta cuando no tenemos disponibilidad
4. Cómo manejar quejas
5. Respuesta a reviews positivos
6. Respuesta a reviews negativos

```

## Checklist de Setup Completo

### Fundamentos
- [ ] Buyer persona definido y documentado
- [ ] Voz de marca establecida
- [ ] Propuesta de valor clara
- [ ] Objetivos SMART definidos

### Presencia Digital
- [ ] Perfiles de redes optimizados (bio, foto, highlights)
- [ ] Google My Business configurado
- [ ] Website funcional (si aplica)
- [ ] Links de contacto funcionando

### Contenido
- [ ] Pilares de contenido definidos
- [ ] Calendario del mes planificado
- [ ] Templates visuales creados
- [ ] Banco de hashtags organizado

### Email Marketing
- [ ] Lead magnet creado
- [ ] Secuencia de bienvenida configurada
- [ ] Newsletter template listo
- [ ] Lista segmentada

### Publicidad
- [ ] Píxel de Meta instalado
- [ ] Audiencias creadas
- [ ] Creativos de test listos
- [ ] Presupuesto definido

### Medición
- [ ] KPIs definidos
- [ ] Herramientas de medición conectadas
- [ ] Reporte mensual template
- [ ] Frecuencia de revisión establecida
