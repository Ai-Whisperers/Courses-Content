# Capstone Rubric - UAV Design
## Track 03 - Aeronautical Engineering

## 1. Aerodynamic Design (20 points)

### Excelente (18-20)
- Airfoil selection justified
- Wing design optimized
- CFD simulation complete
- Stability analysis thorough

### Bueno (14-17)
- Good airfoil choice
- Wing design adequate
- CFD performed
- Basic stability

### Suficiente (10-13)
- Basic airfoil
- Simple wing
- Limited CFD

### Insuficiente (<10)
- Poor aerodynamics
- No analysis

## 2. Structural Design (20 points)

### Excelente (18-20)
- Material selection optimized
- FEA complete (multiple loads)
- Safety factors appropriate
- Weight minimized

### Bueno (14-17)
- Good materials
- FEA performed
- Adequate safety factors

### Suficiente (10-13)
- Basic materials
- Simple FEA

### Insuficiente (<10)
- Poor structural design
- No analysis

## 3. Propulsion System (15 points)

### Excelente (13-15)
- Motor perfectly sized
- Battery optimized
- Propeller designed
- Endurance calculations accurate

### Bueno (10-12)
- Motor appropriate
- Battery adequate
- Calculations correct

### Suficiente (7-9)
- Basic sizing
- Simple calculations

### Insuficiente (<7)
- Undersized/oversized
- Poor calculations

## 4. CAD Quality (15 points)

### Excelente (13-15)
- Complete 3D model
- Assembly drawings
- Part drawings professional
- Manufacturing-ready

### Bueno (10-12)
- Good 3D model
- Basic drawings

### Suficiente (7-9)
- Simple model
- Limited drawings

### Insuficiente (<7)
- Incomplete model
- No drawings

## 5. Performance Analysis (15 points)

### Excelente (13-15)
- All performance calculated
- Mission profile validated
- Trade studies performed
- Meets requirements

### Bueno (10-12)
- Key performance calculated
- Requirements mostly met

### Suficiente (7-9)
- Basic performance
- Some requirements met

### Insuficiente (<7)
- Poor analysis
- Requirements not met

## 6. Documentation (10 points)

### Excelente (9-10)
- Professional report (30-40 pages)
- Clear, well-organized
- All sections complete
- Technical writing excellent

### Bueno (7-8)
- Good report (20-30 pages)
- Organized
- Most sections complete

### Suficiente (5-6)
- Basic report (15-20 pages)
- Acceptable organization

### Insuficiente (<5)
- Poor/incomplete report

## 7. Presentation (5 points)

### Excelente (5)
- Professional presentation
- Clear communication
- Good visuals
- Q&A handled well

### Bueno (3-4)
- Good presentation
- Clear communication

### Suficiente (1-2)
- Basic presentation

### Insuficiente (0)
- Poor/no presentation

---

## Bonus Points (+15 max)

- **+10**: Physical prototype built and flown
- **+5**: Wind tunnel testing
- **+3**: Advanced CFD (turbulence, transient)
- **+3**: Manufacturing cost analysis detailed
- **+2**: Regulatory approval process documented

---

**Total**: 100 points = 50% final grade
