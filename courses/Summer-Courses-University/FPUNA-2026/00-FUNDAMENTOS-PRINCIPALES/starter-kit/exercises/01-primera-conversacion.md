# Ejercicio 1: Tu Primera Conversación con IA

## Objetivo
Aprender a formular preguntas claras y evaluar respuestas de IA.

## Duración
20-30 minutos

## Materiales Necesarios
- Acceso a Claude u otro modelo de lenguaje
- Este documento para seguir los pasos

---

## Parte 1: Pregunta Básica (5 minutos)

### Instrucciones
1. Abre Claude (u otro modelo disponible)
2. Escribe la siguiente pregunta EXACTAMENTE como está:

```
¿Qué es la inteligencia artificial?
```

3. Lee la respuesta completa
4. Anota mentalmente: ¿Fue clara? ¿Fue útil?

### Reflexión
- ¿La respuesta fue demasiado técnica o demasiado simple?
- ¿Te quedaste con dudas?

---

## Parte 2: Pregunta Mejorada (10 minutos)

### Instrucciones
Ahora haz la misma pregunta pero con CONTEXTO. Usa este formato:

```
Soy estudiante universitario de [tu carrera] en Paraguay.
Explícame qué es la inteligencia artificial de manera simple,
como si se lo explicaras a alguien de mi edad que nunca
escuchó el término. Usa ejemplos cotidianos.
```

### Compara las respuestas
| Aspecto | Pregunta 1 | Pregunta 2 |
|---------|------------|------------|
| Claridad | | |
| Relevancia para ti | | |
| Ejemplos útiles | | |
| Largo apropiado | | |

### Lección
¿Notas la diferencia cuando das contexto?

---

## Parte 3: Profundiza (10 minutos)

### Instrucciones
Basándote en la respuesta anterior, haz una pregunta de seguimiento:

```
Interesante. ¿Puedes darme 3 ejemplos de cómo yo, como
estudiante de [tu carrera], podría usar IA en mi día a día
para estudiar mejor o hacer trabajos más eficientemente?
```

### Evalúa la respuesta
- [ ] Los ejemplos son relevantes para mi carrera
- [ ] Son prácticos (puedo hacerlos hoy)
- [ ] Entiendo cómo aplicarlos

---

## Parte 4: Descubre un Límite (5 minutos)

### Instrucciones
Ahora pregunta algo que la IA NO puede hacer bien:

```
¿Cuál es el menú del día en el comedor de la
Universidad Nacional de Asunción hoy?
```

### Observa
- ¿Qué responde?
- ¿Admite que no sabe?
- ¿Inventa algo?

### Lección Clave
La IA NO tiene acceso a información en tiempo real ni a sistemas cerrados.
Esto es fundamental entenderlo.

Completaste el ejercicio si:

- [ ] Reflexionaste sobre cómo mejorar tus preguntas
