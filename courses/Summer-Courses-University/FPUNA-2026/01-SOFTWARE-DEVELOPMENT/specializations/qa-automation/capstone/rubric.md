# Rúbrica de Evaluación - Capstone QA
## FPUNA 2026

**Total**: 100 puntos

---

## 1. Cobertura de Tests (30 puntos)

| Criterio | Excelente (30) | Bueno (25) | Suficiente (20) | Insuficiente (0-19) |
|----------|----------------|------------|-----------------|---------------------|
| **Cantidad** | 100+ tests | 80-99 tests | 60-79 tests | <60 tests |
| **Diversidad** | UI + API + Visual + Performance | UI + API + Visual | UI + API | Solo UI |
| **Escenarios** | Edge cases, errores, happy path | Happy path + algunos errores | Solo happy path | Incompleto |

---

## 2. Calidad del Código (25 puntos)

| Criterio | Excelente (25) | Bueno (20) | Suficiente (15) | Insuficiente (0-14) |
|----------|----------------|------------|-----------------|---------------------|
| **Selectores** | Por rol, semánticos | Mezcla rol/CSS | CSS selectores | IDs hardcoded |
| **DRY** | Sin repetición | Poca repetición | Repetición moderada | Mucha repetición |
| **Comentarios** | Bien documentado | Comentarios clave | Pocos comentarios | Sin comentarios |
| **TypeScript** | Tipado completo | 80% tipado | 50% tipado | Sin tipos |

---
