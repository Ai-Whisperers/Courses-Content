## 1. Respuesta a Consulta de WhatsApp
**Qué:** Contestar preguntas frecuentes de clientes.

```bash
claude "Responde esta consulta de WhatsApp:

Mi establecimiento: Hotel 3 estrellas en Encarnación
Habitaciones: Simple ₲180.000, Doble ₲280.000, Suite ₲450.000
Check-in: 14:00, Check-out: 11:00
Incluye: Desayuno, WiFi, estacionamiento

Consulta del cliente:
'Hola, tienen disponible para el 15 y 16 de agosto para 2 personas?
Cuánto sale y qué incluye?'

Responde:
- Cálido pero profesional
- Breve (máximo 4 oraciones)
- Pedir que confirme para verificar disponibilidad
- Incluir siguiente paso claro"
```

**IMPORTANTE:** Nunca confirmes disponibilidad sin verificar en tu sistema.

**Aprenderás:** Atención al cliente, respuestas efectivas.

## 2. Descripción de Habitación
**Qué:** Escribir textos atractivos para el sitio web o OTAs.

```bash
claude "Escribe descripción de habitación para Booking.com:

Tipo: Habitación Doble Superior
Tamaño: 28 m²
Camas: 1 queen size o 2 individuales
Vista: Jardín interior
Amenities: AC, TV smart, minibar, caja fuerte, secador de pelo
Baño: Privado con ducha, amenities de cortesía

Genera:
1. Título atractivo (máximo 50 caracteres)
2. Descripción principal (máximo 200 palabras)
3. Lista de amenities en formato de bullets
4. Nota sobre qué tipo de huésped es ideal"
```

**Aprenderás:** Copywriting hotelero, venta de experiencia.

## 3. Respuesta a Review Positivo
**Qué:** Agradecer opiniones de clientes satisfechos.

```bash
claude "Escribe respuesta a este review de TripAdvisor:

Review (5 estrellas):
'Excelente hotel! La atención fue increíble, especialmente María
en recepción que nos ayudó con todo. La habitación limpia y el
desayuno muy completo. Volveremos seguro. Recomendado 100%!'

Respuesta debe:
- Agradecer específicamente (mencionar a María)
- Resaltar lo que valoraron
- Invitar a volver
- Máximo 60 palabras
- Tono cálido y personal"
```

**Aprenderás:** Gestión de reputación, refuerzo positivo.

## 4. Respuesta a Review Negativo
**Qué:** Manejar quejas públicas profesionalmente.

```bash
claude "Escribe respuesta a este review de Google:

Review (2 estrellas):
'El hotel está bien ubicado pero la habitación tenía olor a humedad
y el aire acondicionado hacía mucho ruido. Pedimos cambio de
habitación y tardaron 2 horas. No valió lo que pagamos.'

Respuesta debe:
- Agradecer el feedback
- Disculparse por la experiencia
- Explicar sin excusas
- Ofrecer solución/compensación
- Invitar a dar segunda oportunidad
- Máximo 80 palabras
- Profesional pero humano"
```

**Aprenderás:** Manejo de crisis, recuperación de servicio.

## 12. Propuesta para Cliente Corporativo
**Qué:** Cotización para empresa con viajeros frecuentes.

```bash

Empresa prospecto: Consultora con 50 empleados, viajan 15-20 noches/mes
Competencia: Otros hoteles de la zona ofrecen 15-20% descuento

Genera:
1. Carta de presentación personalizada
2. Beneficios de tarifa corporativa
3. Propuesta de descuento escalonada:
   - 10+ noches/mes: X% descuento
   - 20+ noches/mes: Y% descuento
   - 30+ noches/mes: Z% descuento
4. Servicios adicionales sin costo
5. Condiciones de pago
6. Vigencia y siguiente paso"
```

**Aprenderás:** Venta B2B, negociación.

## 15. Forecast de Demanda Mensual
**Qué:** Proyectar ocupación para ajustar tarifas.

```bash
claude "Proyecta demanda para [próximo mes]:

Datos históricos:
- Mismo mes año pasado: 72% ocupación, ADR ₲220.000
- Mes anterior: 65% ocupación, ADR ₲195.000
- Tendencia últimos 3 meses: [creciendo/estable/bajando]

Factores del mes:
- Eventos en la ciudad: [lista]
- Feriados: [fechas]
- Temporada: [alta/media/baja]
- Reservas on-the-books: 45% (falta 55%)

Genera:
1. Ocupación proyectada por semana
2. ADR recomendado por tipo de día (weekday/weekend)
3. Días de alta demanda (subir tarifa)
4. Días de baja demanda (promocionar)
5. Ingreso proyectado (RevPAR × habitaciones × días)
6. Riesgos a monitorear"
```

**Aprenderás:** Revenue management, forecasting.

## 16. Estrategia de Distribución Multi-canal
**Qué:** Optimizar presencia en OTAs y venta directa.

```bash
claude "Diseña estrategia de distribución:

Mi hotel: 60 habitaciones, 3 estrellas superior, Asunción
Canales actuales:
- Booking.com (comisión 15%): 50% de reservas
- Expedia (comisión 18%): 15% de reservas
- Web propia (sin comisión): 20% de reservas
- WhatsApp/Walk-in: 15% de reservas

Problema: Mucha dependencia de OTAs, márgenes bajos

Diseña estrategia para:
1. Aumentar venta directa (web + WhatsApp) a 40%
2. Qué incentivos ofrecer por reserva directa
3. Cómo usar OTAs como vitrina sin regalar clientes
4. Paridad de tarifas vs. diferenciación
5. Programa de fidelización simple
6. Métricas a monitorear mensualmente"
```

**Aprenderás:** Distribución hotelera, canal mix.

## 19. Plan de Crisis y Contingencia
**Qué:** Prepararse para situaciones de emergencia.

```bash

2. Corte de energía prolongado (más de 4 horas)
3. Review viral negativo en redes
4. Accidente de huésped en el hotel
5. Pandemia/cierre de fronteras

Para cada escenario:
1. Señales de alerta temprana
2. Responsable de activar el plan
3. Pasos inmediatos (primeras 2 horas)
4. Comunicación interna (quién sabe qué)
5. Comunicación externa (qué decir a clientes)
6. Recursos necesarios (contactos, presupuesto emergencia)
7. Cómo documentar y aprender del incidente"
```

**Aprenderás:** Gestión de crisis, preparación.

## Métricas Clave de Referencia - Paraguay

### Hotelería
| Métrica | Economico | Midscale | Upscale |
|---------|-----------|----------|---------|
| Ocupación | 50-60% | 55-70% | 60-75% |
| ADR | ₲150-200K | ₲250-350K | ₲400-600K |
| Puntuación OTA | 7.5+ | 8.0+ | 8.5+ |

### Estacionalidad
- **Alta temporada:** Julio-Agosto (vacaciones de invierno), Diciembre-Enero
- **Temporada baja:** Marzo-Mayo, Septiembre-Noviembre
- **Eventos clave:** Fiestas patronales, festivales regionales

### Canales de reserva
- OTAs (Booking, Expedia): 40-60%
- Directo (web, WhatsApp, teléfono): 25-40%
- Agencias/Corporativo: 15-25%

## Verificación Obligatoria

Antes de enviar cualquier información a un cliente:
- ✅ ¿El precio está actualizado?
- ✅ ¿La disponibilidad fue verificada en el sistema?
- ✅ ¿Los horarios son correctos?
- ✅ ¿Las políticas mencionadas son las vigentes?
- ✅ ¿La información de terceros (tours, restaurantes) fue confirmada?

La IA no tiene acceso a tu sistema de reservas ni a información en tiempo real.
