# Plantilla de Proyecto Básico

## Estructura Mínima para Proyectos Simples

Esta plantilla proporciona la estructura esencial para proyectos pequeños o de aprendizaje.

---

## Estructura

```
basic-project/
├── README.md           # Este archivo
├── src/
│   └── index.js        # Archivo principal
├── tests/
│   └── index.test.js   # Tests básicos
├── package.json        # Dependencias (Node.js)
├── .gitignore         # Git ignore
└── .env.example       # Ejemplo de variables de entorno
```

---

## Uso

```bash
# 1. Copiar plantilla
cp -r basic-project mi-proyecto

# 2. Inicializar
cd mi-proyecto
npm install

# 3. Desarrollar
# Editar src/index.js

# 4. Probar
npm test

# 5. Ejecutar
npm start
```
