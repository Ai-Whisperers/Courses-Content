# Module 01: Next.js 14 Foundations

## Overview

Master Next.js 14 App Router fundamentals: routing, Server/Client Components, data fetching, and layouts. Build your first full-featured Next.js application.

**Duration**: 4 hours  
**Level**: Beginner-Intermediate  
**Prerequisites**: JavaScript basics, HTML/CSS knowledge

---

## Learning Objectives

1. Create Next.js 14 applications with App Router
2. Understand Server vs Client Components
3. Implement file-based routing and navigation
4. Fetch data using modern patterns
5. Create reusable layouts and templates
6. Deploy a Next.js app to Vercel

---
