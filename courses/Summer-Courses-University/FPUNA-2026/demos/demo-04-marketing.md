# Demo Track 04: Marketing con IA

## Generación de Contenido y Estrategia Digital

### Objetivo
Demostrar cómo Claude puede acelerar la creación de contenido de marketing.

---

## Demo 1: Generador de Captions para Instagram

```bash
claude "Escribe 5 captions para Instagram para una cafetería en Asunción llamada 'Café del Centro'.

Estilo: Cercano, paraguayo
Objetivo: Engagement
Producto a promocionar: Nuevo frappé de dulce de leche

Incluye:
- Hook atractivo
- Emoji relevantes
- Call to action
- 5 hashtags paraguayos"
```

**Resultado esperado:** 5 captions listos para publicar.

---

## Demo 2: Calendario de Contenido Semanal

```bash
claude "Crea un calendario de contenido para una semana (lunes a viernes) para una tienda de ropa femenina en Ciudad del Este.

Audiencia: Mujeres 25-40 años
Plataforma: Instagram
Temporada: Verano paraguayo
Frecuencia: 1 post + 1 story por día

Para cada día incluye:
- Tipo de contenido
- Tema/producto
- Hook sugerido
- Formato (foto/reel/carrusel)
- Mejor hora para publicar"
```

**Resultado esperado:** Calendario estructurado de 5 días.

---

## Demo 3: Email de Bienvenida

```bash
claude "Escribe un email de bienvenida para nuevos suscriptores de 'TecnoParaguay', una tienda online de electrónica.

Incluye:
- Asunto atractivo (2 opciones)
- Saludo personalizado con [NOMBRE]
- Propuesta de valor (por qué suscribirse fue buena idea)
- 3 beneficios de ser suscriptor
- Código de descuento: BIENVENIDO15 (15% off)
- Call to action
- Links a redes sociales

Tono: Profesional pero amigable
Idioma: Español de Paraguay"
```

**Resultado esperado:** Email completo listo para Mailchimp.

---

## Demo 4: Respuesta a Review Negativo

```bash
claude "Un cliente dejó este review en Google:

'Pésimo servicio. Esperé 40 minutos para que me atiendan y cuando finalmente lo hicieron, el pedido llegó frío. No vuelvo más. ⭐'

Escribe una respuesta profesional para un restaurante de comida paraguaya que:
- Muestre empatía
- Se disculpe sinceramente
- Ofrezca solución concreta
- Invite a darle otra oportunidad
- No sea defensiva ni excesiva

Máximo 100 palabras."
```

**Resultado esperado:** Respuesta profesional y empática.

---

## Demo 5: Copy para Meta Ads

```bash
claude "Escribe copy para una campaña de Meta Ads para una academia de inglés en Asunción.

Objetivo: Generar leads (inscripciones a clase gratis)
Audiencia: Profesionales 25-45 años que quieren mejorar su inglés
Oferta: Primera clase gratis + 20% descuento en inscripción

Genera:
1. Headline principal (max 40 caracteres)
2. Texto primario (2 versiones: corta y larga)
3. Descripción del enlace
4. Call to action sugerido

Incluye elementos de urgencia y beneficios claros."
```

**Resultado esperado:** Copy listo para configurar en Ads Manager.

---

## Demo 6: Análisis de Competencia

```bash
claude "Analiza esta información sobre mi competencia y dame insights:

Mi negocio: Restaurante de sushi delivery en Asunción
Mis precios: Combinado 12 piezas ₲89,000

Competidor 1: SushiYa
- Precio similar: ₲85,000
- Delivery gratis desde ₲150,000
- Reviews: 4.5 estrellas
- Fuerte presencia en Instagram

Competidor 2: NikkeiPy
- Precio premium: ₲120,000
- Delivery ₲15,000
- Reviews: 4.8 estrellas
- Apunta a segmento alto

¿Cómo me diferencio? ¿Qué estrategias sugierés?"
```

**Resultado esperado:** Análisis competitivo con recomendaciones.

## Demo 7: SEO - Keywords y Meta Description

```bash
claude "Necesito optimizar la página de 'Alquiler de Departamentos en Asunción' para SEO.

Genera:
1. Lista de 15 keywords objetivo (principal + long-tail)
2. Title tag optimizado (max 60 caracteres)
3. Meta description (max 155 caracteres)
4. Estructura H1, H2, H3 sugerida
5. 5 ideas de contenido relacionado para blog

Enfocado en búsquedas locales de Paraguay."
```

**Resultado esperado:** Estrategia SEO completa para la página.

## Ejercicio Interactivo

Los estudiantes eligen un negocio (real o ficticio) de Paraguay y generan:

1. 3 posts para Instagram con captions
2. 1 email promocional
3. Bio de Instagram optimizada
4. Respuesta a un review positivo y uno negativo
