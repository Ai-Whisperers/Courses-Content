## Running the Examples

### Prerequisites

```bash
# Node.js 18+
node --version

# Install dependencies
npm install
```

### Running JavaScript Examples

```bash
# Run directly
node code-samples/copilot-basics.js

# Run tests
npm test
```

### TypeScript Examples

```bash
# Compile and run
npx ts-node code-samples/architecture-patterns.ts
```
