## Estructura de la Clase (2 horas)

### Módulo 1: Bienvenida + Onboarding (15 min)
- Presentación del instructor
- Expectativas del taller
- **Mentimeter**: Encuesta de onboarding
- Verificar que todos tienen setup completo

### Módulo 2: ¿Qué es la IA Generativa? (15 min)
- Concepto de LLM (Large Language Models)
- Diferencia entre ChatGPT, Claude, etc.
- Por qué usamos Claude/OpenCode
- Demo rápida: pregunta simple a Claude

### Módulo 3: Instalación y Verificación (15 min)
- Verificar instalaciones del grupo
- Resolver problemas comunes
- Primera conversación con Claude
- Ejercicio: "Hola Claude"

### Break (10 min)

### Módulo 4: Anatomía de un Prompt (15 min)
- ¿Qué es un prompt?
- Estructura básica: Contexto + Instrucción + Formato
- Ejemplos buenos vs malos
- Demo: mismo prompt, diferentes resultados

### Módulo 5: Tu Primer Proyecto con IA (15 min)
- Ejercicio guiado: crear un README.md con ayuda de IA
- Práctica de conversación iterativa
- Cómo corregir y refinar respuestas

### Módulo 6: Configuración de Contexto (15 min)
- ¿Qué es CLAUDE.md?
- Crear tu primer archivo de contexto
- Por qué el contexto mejora las respuestas
- Ejercicio: personalizar tu configuración

### Módulo 7: Cierre + Q&A (10 min)
- Resumen de lo aprendido
- Tarea para casa (opcional)
- Preguntas y respuestas
- Adelanto del Día 2
