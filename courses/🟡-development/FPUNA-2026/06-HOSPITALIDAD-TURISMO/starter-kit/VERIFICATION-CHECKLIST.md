## Paso 2: Verificación de Privacidad

### Checklist de Datos Personales

```markdown
## Verificación de Privacidad

### Revisar que el contenido NO contiene:

IDENTIFICACIÓN
- [ ] Nombres completos de clientes/huéspedes
- [ ] Números de documento/pasaporte
- [ ] Fechas de nacimiento
- [ ] Direcciones personales
- [ ] Teléfonos personales
- [ ] Emails personales

FINANCIERO
- [ ] Números de tarjeta de crédito
- [ ] Datos bancarios
- [ ] Historiales de pago

ITINERARIO ESPECÍFICO
- [ ] Fechas exactas con nombres
- [ ] Números de habitación con nombres
- [ ] Números de vuelo con pasajeros

INFORMACIÓN SENSIBLE
- [ ] Alergias vinculadas a personas
- [ ] Condiciones médicas
- [ ] Información de menores

### Si encontraste alguno de estos datos:
1. NO usar el contenido generado
2. Regenerar sin datos personales
3. Reportar si fue error del sistema
```

## Paso 3: Verificación de Tono y Autenticidad

### Checklist de Comunicación

```markdown
## Verificación de Tono

### Apropiado para el canal
- [ ] Formalidad correcta (email vs WhatsApp)
- [ ] Longitud apropiada para el medio
- [ ] Formato adecuado (párrafos, bullets, etc.)

### Apropiado para el cliente
- [ ] Nivel de formalidad correcto
- [ ] Idioma/expresiones apropiadas
- [ ] Consideración cultural (si es extranjero)

### Autenticidad
- [ ] No suena robótico/genérico
- [ ] Refleja la personalidad de nuestra marca
- [ ] Tiene toque humano
- [ ] Evita frases típicas de IA:
      - "Es importante mencionar que..."
      - "En conclusión..."
      - "Como se mencionó anteriormente..."

### Hospitalidad
- [ ] Transmite calidez
- [ ] Ofrece ayuda adicional
- [ ] Invita a contactar si hay dudas
- [ ] Agradece apropiadamente
```

### Señales de Contenido Genérico

```
⚠️ ALERTAS DE CONTENIDO IMPERSONAL:

• Descripciones que podrían ser de cualquier lugar
• Falta de detalles específicos locales
• Vocabulario muy formal sin calidez
• Sin menciones a cultura o tradiciones locales
• Información que suena a enciclopedia

→ AGREGAR toque personal y detalles específicos
→ INCLUIR recomendaciones desde experiencia real
→ MENCIONAR aspectos únicos del lugar
```

## Paso 4: Verificación Cultural

### Checklist de Contexto Local

```markdown
## Verificación Cultural - Paraguay

### Precisión cultural
- [ ] Referencias a tradiciones son correctas
- [ ] Nombres de lugares/comidas bien escritos
- [ ] Historia/datos son precisos
- [ ] No hay estereotipos ofensivos

### Idioma
- [ ] Uso apropiado de guaraní (si aplica)
- [ ] Palabras locales bien escritas
- [ ] Jopara natural (no forzado)

### Recomendaciones
- [ ] Lugares recomendados existen y operan
- [ ] Precios son realistas para Paraguay
- [ ] Distancias y tiempos son precisos
- [ ] Consideraciones de seguridad actualizadas

### Sensibilidad
- [ ] Respeto a comunidades indígenas
- [ ] No exotización de la cultura
- [ ] Precisión histórica (Guerra, dictadura, etc.)
- [ ] Evita clichés turísticos superficiales
```

### Errores Culturales Comunes de IA

```
❌ ERRORES FRECUENTES:

• Confundir Paraguay con Uruguay
• Asumir que todos hablan solo español
• Describir tereré como "mate frío" sin más
• Generalizar cultura guaraní incorrectamente
• Dar precios en USD sin contextualizar
• Asumir infraestructura de país desarrollado
• Ignorar estacionalidad (inundaciones, calor extremo)
• Mezclar tradiciones con otros países latinoamericanos

→ SIEMPRE revisar precisión local
→ CONSULTAR con conocedores locales si hay duda
```

## Paso 5: Verificación de Utilidad

### Checklist de Valor para el Viajero

```markdown
## Verificación de Utilidad

### Información completa
- [ ] Responde la pregunta del cliente completamente
- [ ] Incluye información práctica necesaria
- [ ] Anticipa preguntas de seguimiento
- [ ] Ofrece alternativas si aplica

### Accionable
- [ ] El cliente sabe qué hacer después
- [ ] Hay instrucciones claras
- [ ] Contactos/links son funcionales
- [ ] Call to action claro

### Realista
- [ ] Expectativas son precisas
- [ ] No sobre-promete
- [ ] Advierte de posibles inconvenientes
- [ ] Tiempos y costos son realistas

### Seguridad
- [ ] Información de seguridad incluida si necesario
- [ ] Precauciones apropiadas mencionadas
- [ ] Contactos de emergencia disponibles
- [ ] No pone al viajero en riesgo
```
