# Introduction to AI for Business Professionals - Prompt Library

> Essential prompts for business applications

---

## Getting Started Prompts

### First Interaction

```
I'm new to using AI assistants. I work in [your role] at a [company type].
My main responsibilities include:
- [Responsibility 1]
- [Responsibility 2]
- [Responsibility 3]

What are the top 5 ways you could help me in my daily work?
```

### Understanding Capabilities

```
I need help with [task].

Before you start, can you:
1. Confirm you understand what I'm asking
2. Tell me what information you need from me
3. Explain any limitations I should be aware of
4. Suggest the best way to approach this

Then proceed with the task.
```

## Business Communication Prompts

### Email Drafting

```
Write a professional email to [recipient role] about [topic].

Context:
- My relationship with them: [formal/informal/new contact]
- Purpose: [inform/request/follow-up/apologize]
- Tone: [professional/friendly/urgent]
- Key points to include:
  1. [Point 1]
  2. [Point 2]

Keep it concise (under 150 words) and include a clear call to action.
```

### Meeting Preparation

```
Help me prepare for a meeting about [topic].

Meeting details:
- Attendees: [who will be there]
- My role: [what I'm presenting/contributing]
- Duration: [length]
- Goal: [desired outcome]

Please provide:
1. Key talking points
2. Potential questions I might face
3. Data or examples I should prepare
4. Suggested agenda
```

### Presentation Outline

```
Create an outline for a [duration] presentation on [topic].

Audience: [who they are and their background]
Goal: [what I want them to think/do after]
Key message: [one sentence summary]

Include:
- Opening hook
- 3-4 main sections
- Supporting points for each section
- Transition suggestions
- Strong closing/call to action
```

## Analysis Prompts

### Data Interpretation

```
Help me understand this data:

[Paste data or describe it]

I need to:
1. Summarize the key findings
2. Identify trends or patterns
3. Spot any anomalies or concerns
4. Suggest what actions to take

Present your analysis in bullet points suitable for a business report.
```

### Decision Support

```
I need to make a decision about [decision topic].

Options:
1. [Option A]: [brief description]
2. [Option B]: [brief description]
3. [Option C]: [brief description]

Key factors to consider:
- [Factor 1]
- [Factor 2]
- [Factor 3]

Please analyze each option against these factors and provide a recommendation with reasoning.
```

### Market Research

```
Research [topic/company/industry] and provide:

1. Overview: What is it and why does it matter?
2. Key players: Who are the main companies/people?
3. Trends: What's happening in this space?
4. Opportunities: Where are the growth areas?
5. Risks: What challenges exist?

Keep the analysis at a level suitable for [executive briefing/detailed report].
```

## Documentation Prompts

### Meeting Summary

```
Summarize this meeting for those who couldn't attend:

[Paste notes or transcript]

Include:
- Key decisions made
- Action items with owners and deadlines
- Open questions requiring follow-up
- Next steps

Format as a brief email that takes 2 minutes to read.
```

### Process Documentation

```
Document this process step by step:

Process name: [name]
Purpose: [why we do this]
Who does it: [role responsible]

Steps as I understand them:
1. [Step 1]
2. [Step 2]
3. [Step 3]

Please expand each step with:
- Detailed instructions
- Common pitfalls to avoid
- Tips for efficiency
```

### Report Writing

```
Help me write a [type] report on [topic].

Key findings to include:
1. [Finding 1]
2. [Finding 2]
3. [Finding 3]

Audience: [who will read this]
Length: [target length]
Tone: [formal/informal]

Structure it with:
- Executive summary
- Background/context
- Main findings
- Recommendations
- Next steps
```

## Problem-Solving Prompts

### Brainstorming

```
I'm facing this challenge: [describe problem]

Please help me brainstorm solutions by:
1. Generating 10 possible approaches (include some unconventional ideas)
2. Grouping them by category (quick wins, long-term solutions, etc.)
3. Highlighting the top 3 most promising options
4. Suggesting first steps for the best option
```

### Root Cause Analysis

```
We're experiencing [problem description].

Symptoms:
- [Symptom 1]
- [Symptom 2]

When it started: [timing]
What changed: [recent changes]

Help me identify:
1. Possible root causes
2. Questions to ask to narrow down the cause
3. Data to collect
4. Quick tests to validate hypotheses
```

## Learning Prompts

### Concept Explanation

```
Explain [concept] to me as if I'm a [business professional with no technical background].

Include:
- Simple definition
- Real-world example I can relate to
- Why it matters for business
- Common misconceptions
- How to explain it to others
```

### Industry Update

```
Give me a 5-minute briefing on the latest developments in [industry/topic].

Focus on:
- What's new in the last [timeframe]
- Why it matters for [my role/industry]
- What I should do about it
- Questions I should be asking

Make it actionable, not just informational.
```

## Tips for Better Results

### The CLEAR Framework

**C**ontext: Provide background information
**L**ength: Specify desired output length
**E**xamples: Show what good output looks like
**A**udience: Who will read/use this
**R**esult: What format do you want

### Common Mistakes to Avoid

| Don't | Do Instead |
|-------|------------|
| "Write something about sales" | "Write a 200-word summary of Q3 sales trends for the executive team" |
| "Make it good" | "Make it concise, professional, and action-oriented" |
| "Help me with this" | "Help me draft an email that asks for budget approval" |

### When Results Aren't Right

```
The response you gave me isn't quite what I need.

What I got: [describe the issue]
What I actually need: [be more specific]
Please adjust by: [specific change]
```
