# Checklist de Verificación - Contenido de Marketing Generado por IA

## Verificación específica para copy, campañas y contenido digital

---

## Por Qué Verificar es Crítico

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│   El contenido de IA puede:                                 │
│   • Parecer profesional pero ser genérico                   │
│   • Contener afirmaciones no verificables                   │
│   • Usar tono inconsistente con tu marca                    │
│   • Ignorar contexto cultural paraguayo                     │
│   • Plagiar inadvertidamente contenido existente            │
│   • Inventar datos, estadísticas o testimonios              │
│                                                              │
│   Tu reputación de marca depende de contenido auténtico.    │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

---

## Proceso de Verificación en 5 Pasos

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│   1. VERIFICAR HECHOS     ¿Los datos son reales?            │
│          ↓                                                   │
│   2. VERIFICAR VOZ        ¿Suena como mi marca?             │
│          ↓                                                   │
│   3. VERIFICAR AUDIENCIA  ¿Le habla a mi cliente?           │
│          ↓                                                   │
│   4. VERIFICAR LEGALIDAD  ¿Cumple regulaciones?             │
│          ↓                                                   │
│   5. VERIFICAR IMPACTO    ¿Logrará el objetivo?             │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## Paso 1: Verificación de Hechos

### Checklist de Datos

```markdown
## Verificación de Información Factual

### Estadísticas mencionadas
| Dato citado | Fuente verificada | ¿Es actual? | ¿Aplica a Paraguay? |
|-------------|-------------------|-------------|---------------------|
| [dato 1]    | [ ] Sí [ ] No    | [ ]         | [ ]                 |
| [dato 2]    | [ ] Sí [ ] No    | [ ]         | [ ]                 |

### Afirmaciones sobre productos/servicios
- [ ] Todos los beneficios mencionados son reales
- [ ] Los precios/ofertas son actuales
- [ ] Las características son precisas
- [ ] No hay promesas exageradas

### Testimonios y referencias
- [ ] Los testimonios son de clientes reales
- [ ] Las citas están autorizadas
- [ ] Los casos de éxito son verificables

### ALERTA ROJA - La IA puede inventar:
⚠️ Estadísticas que suenan plausibles
⚠️ Citas de "expertos" inexistentes
⚠️ Estudios científicos falsos
⚠️ Testimonios de clientes ficticios
```

### Verificación de Plagio

```
Herramientas para verificar originalidad:
• Copyscape (copyscape.com)
• Grammarly plagiarism checker
• Duplichecker (gratis)
• Búsqueda directa en Google de frases

SIEMPRE verificar:
- [ ] Titulares principales
- [ ] Slogans o frases clave
- [ ] Descripciones de producto
- [ ] Párrafos extensos
```

## Paso 2: Verificación de Voz de Marca

### Checklist de Tono

```markdown
## ¿El contenido suena como MI marca?

### Comparación con tono definido
Mi tono es: [formal/informal/técnico/amigable/...]

- [ ] El vocabulario es consistente con mi marca
- [ ] El nivel de formalidad es apropiado
- [ ] El humor (si aplica) refleja mi estilo
- [ ] Las expresiones son naturales, no forzadas

### Palabras y frases de marca
Mis términos característicos:
- [ ] ¿Aparecen en el contenido?
- [ ] ¿Se usan correctamente?

Palabras que NUNCA uso:
- [ ] ¿Alguna aparece en el contenido?

### Test de reconocimiento
"Si un cliente fiel lee esto, ¿reconocería que es mío?"
[ ] Definitivamente sí
[ ] Probablemente sí
[ ] No estoy seguro
[ ] No suena como yo → REVISAR
```

### Corrección de Voz Genérica

```
Señales de contenido genérico de IA:

❌ "En el mundo actual..."
❌ "Es importante mencionar que..."
❌ "Como todos sabemos..."
❌ "Sin lugar a dudas..."
❌ "Es fundamental destacar..."

→ Reescribir con VOZ PROPIA
→ Agregar ejemplos específicos de TU marca
→ Usar expresiones que TÚ usarías
```

## Paso 3: Verificación de Audiencia

### Checklist de Relevancia

```markdown
## ¿Esto conecta con MI cliente?

### Demografía
Mi audiencia es: [definir]
- [ ] El lenguaje es apropiado para su edad
- [ ] El nivel de complejidad es correcto
- [ ] Las referencias culturales son relevantes

### Problemas/Dolores
Mi cliente sufre de: [listar]
- [ ] El contenido aborda estos dolores
- [ ] La solución presentada es creíble
- [ ] El beneficio es tangible y deseable

### Contexto paraguayo
- [ ] Expresiones locales apropiadas (si aplica)
- [ ] Referencias culturales correctas
- [ ] Sensibilidad a realidades económicas
- [ ] Consideración de bilingüismo (español/guaraní)
```

### Test de Empatía

```
Pregunta clave:
"Si yo fuera mi cliente, ¿esto me haría parar y leer?"

[ ] Sí, captura atención inmediata
[ ] Tal vez, pero necesita más gancho
[ ] No, es muy genérico
[ ] No, no me habla a mí

Acción: Si no es "Sí", revisar hook y propuesta.
```

---

## Paso 4: Verificación Legal y Ética

### Checklist de Cumplimiento

```markdown
## Verificación Legal

### Publicidad
- [ ] No hay publicidad engañosa
- [ ] Términos y condiciones están claros
- [ ] Promociones tienen fechas de vigencia
- [ ] Descuentos son verificables
- [ ] "Publicidad" está declarada si es pagada

### Derechos de autor
- [ ] Imágenes tienen licencia de uso
- [ ] Música/audio está autorizada
- [ ] Citas tienen atribución correcta
- [ ] No hay logos de terceros sin permiso

### Datos personales
- [ ] No se usan datos sin consentimiento
- [ ] Testimonios tienen autorización
- [ ] Fotografías de personas están autorizadas
- [ ] Cumple con leyes de protección de datos

### Competencia
- [ ] No se difama a competidores
- [ ] Comparaciones son objetivas y verificables
- [ ] No se imitan marcas registradas
```

### Alertas Éticas

```
⚠️ EVITAR:

• Urgencia falsa ("¡Solo HOY!" cuando no es cierto)
• Escasez artificial ("¡Últimas 5 unidades!" inventado)
• Testimonios fabricados
• Before/after manipulados
• Promesas de resultados garantizados
• Explotación de miedos o inseguridades

El costo de una mala práctica ético:
→ Pérdida de confianza
→ Daño reputacional
→ Posibles sanciones legales
```

## Paso 5: Verificación de Impacto

### Checklist de Efectividad

```markdown
## ¿Este contenido logrará su objetivo?

### Objetivo definido
Este contenido busca: [awareness/engagement/conversión/...]

### Estructura
- [ ] Hook claro en primeras palabras
- [ ] Mensaje único y enfocado
- [ ] Beneficio claro para el lector
- [ ] Call to Action específico y alcanzable

### Optimización por plataforma
Plataforma destino: ________

- [ ] Longitud apropiada para la plataforma
- [ ] Formato correcto (imagen, video, carrusel)
- [ ] Hashtags relevantes (si aplica)
- [ ] Horario de publicación considerado

### Test A/B potencial
¿Qué variaciones podría probar?
- Variación de hook: _____
- Variación de CTA: _____
- Variación de imagen: _____
```

### Predicción de Resultados

```
Antes de publicar, estimar:

Alcance esperado: _____
Engagement rate esperado: _____%
Clics esperados: _____
Conversiones esperadas: _____

Después de publicar, medir y comparar.
Esto mejora tu criterio para evaluar IA.
```

## Checklists por Tipo de Contenido

### Post de Redes Sociales

```markdown
## Verificación: Post Social

### Básicos
- [ ] Longitud apropiada (no cortado)
- [ ] Emojis usados con moderación (si corresponde)
- [ ] Sin errores ortográficos
- [ ] Hashtags relevantes (máx 5-10)

### Contenido
- [ ] Hook en primera línea
- [ ] Valor claro para el lector
- [ ] Call to action
- [ ] Conexión con imagen/video

### Marca
- [ ] Tono consistente
- [ ] Alineado con estrategia actual
- [ ] No contradice posts anteriores

### Legal
- [ ] Promoción clara si es publicidad
- [ ] Derechos de imagen verificados
```

### Email Marketing

```markdown
## Verificación: Email

### Técnico
- [ ] Asunto tiene menos de 50 caracteres
- [ ] Preheader está optimizado
- [ ] Links funcionan
- [ ] Se ve bien en móvil
- [ ] Unsubscribe visible

### Contenido
- [ ] Asunto genera curiosidad (sin clickbait)
- [ ] Saludo personalizado
- [ ] Un mensaje principal
- [ ] CTA claro y único
- [ ] Firma profesional

### Legal
- [ ] Cumple con leyes anti-spam
- [ ] Tiene dirección física
- [ ] Explica por qué reciben el email
```

### Artículo de Blog/SEO

```markdown
## Verificación: Artículo

### SEO
- [ ] Keyword principal en título
- [ ] Keyword en primeros 100 palabras
- [ ] Meta description atractiva (150-160 chars)
- [ ] Headers H2, H3 organizados
- [ ] Alt text en imágenes

### Contenido
- [ ] Introducción engancha
- [ ] Contenido original (no plagio)
- [ ] Datos verificados con fuentes
- [ ] Conclusión con valor
- [ ] CTA al final

### Calidad
- [ ] Párrafos cortos (3-4 líneas máx)
- [ ] Uso de listas y bullets
- [ ] Sin relleno innecesario
- [ ] Lenguaje claro y directo
```

### Copy de Anuncio

```markdown
## Verificación: Ad Copy

### Facebook/Instagram Ads
- [ ] Texto primario: gancho + beneficio + CTA
- [ ] Título: claro y atractivo
- [ ] Descripción: refuerza mensaje
- [ ] No más de 20% texto en imagen (revisar)

### Google Ads
- [ ] Títulos dentro de límite de caracteres
- [ ] Keywords incluidas
- [ ] CTA claro
- [ ] URL destino correcta

### General
- [ ] Propuesta de valor clara
- [ ] Urgencia real (no falsa)
- [ ] Beneficio > características
- [ ] Coherente con landing page
```
