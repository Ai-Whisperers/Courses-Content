## SLIDE 4: ¿QUÉ ES UN LLM?
**Tiempo: 2 minutos**

```
+----------------------------------------------------------+
|                                                          |
|         LARGE LANGUAGE MODEL (LLM)                       |
|                                                          |
|  +----------------------------------------------------+  |
|  |                                                    |  |
|  |  Un modelo entrenado con BILLONES de textos:      |  |
|  |                                                    |  |
|  |  • Libros, artículos, código, documentación       |  |
|  |  • Aprende patrones del lenguaje                  |  |
|  |  • Predice qué palabra viene después              |  |
|  |  • Puede GENERAR texto coherente                  |  |
|  |                                                    |  |
|  +----------------------------------------------------+  |
|                                                          |
|  +----------------------------------------------------+  |
|  |  EJEMPLOS:                                        |  |
|  |                                                    |  |
```

## SLIDE 7: EL NUEVO WORKFLOW
**Tiempo: 2 minutos**

```
+----------------------------------------------------------+
|                                                          |
|         ANTES vs AHORA                                   |
|                                                          |
|  ANTES (sin IA):                                         |
|  +----------------------------------------------------+  |
|  |  Idea -> Investigar -> Planear -> Desarrollar ->  |  |
|  |  Probar -> Corregir -> Documentar                 |  |
|  |                                                    |  |
|  |  SEMANAS o MESES por iteración                    |  |
|  +----------------------------------------------------+  |
|                                                          |
|  AHORA (con IA):                                         |
|  +----------------------------------------------------+  |
|  |  Idea -> Prompt -> Generar -> Iterar -> Refinar   |  |
|  |                                                    |  |
|  |  HORAS o DÍAS por iteración                       |  |
|  |                                                    |  |
|  |  Las iteraciones que antes tomaban semanas        |  |
|  |  ahora suceden en el mismo día                    |  |
|  +----------------------------------------------------+  |
|                                                          |
|  El costo de EXPERIMENTAR bajó drásticamente             |
|                                                          |
+----------------------------------------------------------+
```
