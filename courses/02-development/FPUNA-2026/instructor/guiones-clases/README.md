## Como Usar Estos Guiones

### Antes de la Clase

1. Leer el guion completo (30 min)
2. Verificar checklist de materiales
3. Probar demos y ejercicios
4. Preparar respuestas a preguntas comunes

### Durante la Clase

- Seguir la estructura de tiempo
- Adaptar segun nivel del grupo
- Tomar notas de preguntas/problemas
- Ser flexible con el break si es necesario

### Despues de la Clase

- Completar seccion de NOTAS POST-CLASE
- Registrar asistencia
- Documentar problemas tecnicos
- Preparar mejoras para proxima edicion

---

## Estructura de Cada Guion

```
1. Informacion de la Clase
   - Duracion, audiencia, objetivo

2. PRE-CLASE
   - Checklist del instructor
   - Materiales necesarios

3. MODULOS (7 por clase)
   - Tiempo especifico (ej: 0:15 - 0:25)
   - DECIR: texto sugerido para el instructor
   - MOSTRAR: contenido para pantalla
   - HACER: acciones/demos
   - EJERCICIO: actividades para estudiantes

4. BREAK
   - 10 min (5 min en dia 6)

5. CIERRE
   - Resumen
   - Recursos
   - Adelanto siguiente dia

6. NOTAS POST-CLASE
   - Que registrar
   - Seguimiento necesario
```

---

## Tips Generales

### Manejo del Tiempo

- Cada modulo tiene tiempo asignado
- Si un tema se extiende, acortar el siguiente
- El break es flexible
- Nunca saltar el cierre/resumen

### Problemas Tecnicos

- Tener backup offline de demos
- Conocer solucion a errores comunes
- Tener plan B si internet falla

### Engagement

- Hacer preguntas frecuentes
- Caminar por el aula durante ejercicios
- Mostrar resultados de estudiantes (con permiso)
- Celebrar logros pequenos
