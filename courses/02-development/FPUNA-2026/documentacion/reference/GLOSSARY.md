# Glosario Técnico / Technical Glossary
## FPUNA Summer 2026 - AI-Augmented Development Program

**Idiomas / Languages**: Español (ES) + English (EN)

---

## Cómo Usar Este Glosario / How to Use This Glossary

- **Español primero**: Término en español, definición, traducción al inglés
- **Organizado alfabéticamente** por término en español
- **Categorías**: [SW] Software, [HW] Hardware, [AI] Inteligencia Artificial, [AERO] Aeronáutica, [ENG] Ingeniería

---

## A

### API - Application Programming Interface [SW]
**ES**: Interfaz que permite la comunicación entre diferentes sistemas de software.  
**EN**: Interface that allows communication between different software systems.  
**Ejemplo/Example**: REST API, GraphQL API

### Arquitectura de Microservicios [SW]
**ES**: Estilo arquitectónico donde una aplicación se divide en servicios pequeños e independientes.  
**EN**: Microservices Architecture - Architectural style where an application is divided into small, independent services.  
**Relacionado**: Monolith, Distributed Systems

### AWS - Amazon Web Services [SW]
**ES**: Plataforma de servicios en la nube de Amazon.  
**EN**: Amazon's cloud computing platform.

---

## B

### Backend [SW]
**ES**: Parte del sistema que maneja la lógica de negocio y datos, no visible para el usuario.  
**EN**: Server-side part of the system that handles business logic and data.  
**Opuesto**: Frontend

### Breakpoint [SW]
**ES**: Punto de interrupción en el código para depuración.  
**EN**: Stopping point in code for debugging.

---

## C

### CAD - Computer-Aided Design [AERO]
**ES**: Diseño asistido por computadora, software para modelado 3D.  
**EN**: Software for 3D modeling and design.  
**Herramientas**: Fusion 360, CATIA, SolidWorks

### CFD - Computational Fluid Dynamics [AERO]
**ES**: Dinámica de fluidos computacional, simulación de flujos.  
**EN**: Computer simulation of fluid flows.  
**Herramientas**: ANSYS Fluent, OpenFOAM

### CI/CD - Continuous Integration/Continuous Deployment [SW]
**ES**: Integración continua y despliegue continuo, automatización del pipeline de desarrollo.  
**EN**: Automated development pipeline for testing and deployment.  
**Herramientas**: GitHub Actions, Jenkins, GitLab CI

### Clean Code [SW]
**ES**: Código limpio, legible y mantenible.  
**EN**: Readable and maintainable code.  
**Principios**: SOLID, DRY, KISS

### Compilador / Compiler [SW]
**ES**: Programa que traduce código fuente a código máquina.  
**EN**: Program that translates source code to machine code.

---

## D

### DAO - Data Access Object [SW]
**ES**: Patrón que abstrae el acceso a datos.  
**EN**: Pattern that abstracts data access.

### Debugging [SW]
**ES**: Depuración, proceso de encontrar y corregir errores.  
**EN**: Process of finding and fixing bugs.

### Dependency Injection [SW]
**ES**: Inyección de dependencias, patrón de diseño para gestionar dependencias.  
**EN**: Design pattern for managing dependencies.

### DevOps [SW]
**ES**: Combinación de desarrollo (Dev) y operaciones (Ops), cultura de colaboración.  
**EN**: Combination of development and operations, collaboration culture.

### Docker [SW]
**ES**: Plataforma para contenedores, empaqueta aplicaciones con sus dependencias.  
**EN**: Platform for containers, packages applications with dependencies.

### DRY - Don't Repeat Yourself [SW]
**ES**: No repetir código, principio de reutilización.  
**EN**: Principle of code reusability.

---

## E

### Encapsulación / Encapsulation [SW]
**ES**: Ocultar detalles internos de implementación.  
**EN**: Hiding internal implementation details.

### Endpoint [SW]
**ES**: Punto de acceso a una API (URL específica).  
**EN**: Access point to an API (specific URL).  
**Ejemplo**: `GET /api/users/:id`

### ESP32 [HW]
**ES**: Microcontrolador con WiFi y Bluetooth integrado.  
**EN**: Microcontroller with integrated WiFi and Bluetooth.  
**Uso**: IoT, embedded systems

---

## F

### FEA - Finite Element Analysis [AERO]
**ES**: Análisis de elementos finitos, método numérico para análisis estructural.  
**EN**: Numerical method for structural analysis.  
**Uso**: Simulación de esfuerzos, deformaciones

### Firmware [HW]
**ES**: Software que controla hardware directamente.  
**EN**: Software that directly controls hardware.  
**Ejemplo**: Arduino sketches, ESP32 code

### Framework [SW]
**ES**: Marco de trabajo, estructura base para desarrollo.  
**EN**: Base structure for development.  
**Ejemplos**: NestJS, Next.js, React

### Frontend [SW]
**ES**: Parte del sistema visible para el usuario (interfaz).  
**EN**: User-facing part of the system (interface).  
**Tecnologías**: React, Vue, Angular

---

## G

### Git [SW]
**ES**: Sistema de control de versiones distribuido.  
**EN**: Distributed version control system.  
**Comandos**: commit, push, pull, merge

### GitHub [SW]
**ES**: Plataforma de hospedaje de código con Git.  
**EN**: Code hosting platform using Git.

### GraphQL [SW]
**ES**: Lenguaje de consulta para APIs, alternativa a REST.  
**EN**: Query language for APIs, alternative to REST.

---

## H

### Hash [SW]
**ES**: Función que convierte datos en cadena de longitud fija.  
**EN**: Function that converts data into fixed-length string.  
**Uso**: Passwords, integridad de datos

### HTTP - HyperText Transfer Protocol [SW]
**ES**: Protocolo de comunicación web.  
**EN**: Web communication protocol.  
**Métodos**: GET, POST, PUT, DELETE

---

## I

### IDE - Integrated Development Environment [SW]
**ES**: Entorno de desarrollo integrado.  
**EN**: Development environment with code editor, debugger, etc.  
**Ejemplos**: VS Code, IntelliJ, PyCharm

### IoT - Internet of Things [HW]
**ES**: Internet de las Cosas, dispositivos conectados.  
**EN**: Connected devices and sensors.  
**Protocolos**: MQTT, HTTP, CoAP

---

## J

### JSON - JavaScript Object Notation [SW]
**ES**: Formato de intercambio de datos basado en texto.  
**EN**: Text-based data interchange format.  
**Ejemplo**: `{"name": "Juan", "age": 25}`

### JWT - JSON Web Token [SW]
**ES**: Token para autenticación y autorización.  
**EN**: Token for authentication and authorization.

---

## K

### KiCad [HW]
**ES**: Software open-source para diseño de PCB.  
**EN**: Open-source PCB design software.

### Kubernetes [SW]
**ES**: Plataforma para orquestación de contenedores.  
**EN**: Container orchestration platform.  
**Abreviación**: K8s

---

## L

### Layered Architecture [SW]
**ES**: Arquitectura en capas, separación de responsabilidades.  
**EN**: Architecture with separated layers of responsibility.  
**Capas típicas**: Presentation, Business, Data, Database

### Linux [SW]
**ES**: Sistema operativo open-source basado en Unix.  
**EN**: Open-source Unix-based operating system.

---

## M

### MCP - Model Context Protocol [AI]
**ES**: Protocolo para conectar herramientas con LLMs.  
**EN**: Protocol for connecting tools with LLMs.

### Microcontrolador / Microcontroller [HW]
**ES**: Pequeña computadora en un chip.  
**EN**: Small computer on a chip.  
**Ejemplos**: Arduino, ESP32, Raspberry Pi Pico

### Middleware [SW]
**ES**: Software que actúa entre dos aplicaciones.  
**EN**: Software that acts between two applications.

### Monolito / Monolith [SW]
**ES**: Aplicación con todos los componentes en un solo código base.  
**EN**: Application with all components in one codebase.  
**Opuesto**: Microservices

### MQTT - Message Queuing Telemetry Transport [HW]
**ES**: Protocolo de mensajería ligero para IoT.  
**EN**: Lightweight messaging protocol for IoT.

---

## N

### NestJS [SW]
**ES**: Framework Node.js para backend, basado en TypeScript.  
**EN**: Node.js framework for backend, based on TypeScript.

### Next.js [SW]
**ES**: Framework React para aplicaciones web full-stack.  
**EN**: React framework for full-stack web applications.

### Node.js [SW]
**ES**: Runtime de JavaScript del lado del servidor.  
**EN**: JavaScript runtime for server-side.

---

## O

### OOP - Object-Oriented Programming [SW]
**ES**: Programación orientada a objetos.  
**EN**: Programming paradigm based on objects.  
**Conceptos**: Encapsulation, Inheritance, Polymorphism

### OpenCode [AI]
**ES**: IDE potenciado por IA para desarrollo acelerado.  
**EN**: AI-powered IDE for accelerated development.

### ORM - Object-Relational Mapping [SW]
**ES**: Mapeo objeto-relacional, abstracción de base de datos.  
**EN**: Abstraction layer for database.  
**Herramientas**: Prisma, TypeORM, Sequelize

---

## P

### PCB - Printed Circuit Board [HW]
**ES**: Placa de circuito impreso.  
**EN**: Board with electronic circuits printed on it.

### PLC - Programmable Logic Controller [HW]
**ES**: Controlador lógico programable para automatización industrial.  
**EN**: Industrial automation controller.

### PostgreSQL [SW]
**ES**: Base de datos relacional open-source.  
**EN**: Open-source relational database.

### Prompt Engineering [AI]
**ES**: Arte de escribir instrucciones efectivas para IA.  
**EN**: Art of writing effective instructions for AI.

---

## Q

### QA - Quality Assurance [SW]
**ES**: Aseguramiento de calidad, testing de software.  
**EN**: Software testing and quality control.

---

## R

### Redis [SW]
**ES**: Base de datos en memoria para caching.  
**EN**: In-memory database for caching.

### Refactoring [SW]
**ES**: Reestructurar código sin cambiar funcionalidad.  
**EN**: Restructuring code without changing functionality.

### REST - Representational State Transfer [SW]
**ES**: Estilo arquitectónico para APIs web.  
**EN**: Architectural style for web APIs.  
**Métodos**: GET, POST, PUT, DELETE

---

## S

### SCADA - Supervisory Control and Data Acquisition [HW]
**ES**: Sistema de control y adquisición de datos industrial.  
**EN**: Industrial control and data acquisition system.

### SDK - Software Development Kit [SW]
**ES**: Kit de desarrollo de software, conjunto de herramientas.  
**EN**: Set of software development tools.

### SOLID [SW]
**ES**: Cinco principios de diseño OOP (Single responsibility, Open-closed, Liskov substitution, Interface segregation, Dependency inversion).  
**EN**: Five OOP design principles.

---

## T

### TDD - Test-Driven Development [SW]
**ES**: Desarrollo guiado por pruebas, escribir tests primero.  
**EN**: Writing tests before code.

### TypeScript [SW]
**ES**: Superconjunto de JavaScript con tipado estático.  
**EN**: JavaScript superset with static typing.

---

## U

### UAV - Unmanned Aerial Vehicle [AERO]
**ES**: Vehículo aéreo no tripulado (drone).  
**EN**: Unmanned Aerial Vehicle (drone).

### UI - User Interface [SW]
**ES**: Interfaz de usuario, parte visual de la aplicación.  
**EN**: Visual part of the application.

### UX - User Experience [SW]
**ES**: Experiencia de usuario, cómo se siente usar la aplicación.  
**EN**: How it feels to use the application.

## V

### Virtual Environment [SW]
**ES**: Entorno aislado para dependencias de Python.  
**EN**: Isolated environment for Python dependencies.  
**Herramientas**: venv, conda

## W

### WebSocket [SW]
**ES**: Protocolo para comunicación bidireccional en tiempo real.  
**EN**: Protocol for real-time bidirectional communication.

## Y

### YAML [SW]
