# Quiz Template

Use this format for module quizzes. Keep to 3 questions per module.

## Format

```markdown
## Quiz: [Nombre del Modulo]

**Pregunta 1**: [Pregunta clara y directa]
- A) [Opcion]
- B) [Opcion]
- C) [Opcion]
- D) [Opcion]

**Pregunta 2**: [Pregunta]
- A) [Opcion]
- B) [Opcion]
- C) [Opcion]
- D) [Opcion]

**Pregunta 3**: [Pregunta]
- A) [Opcion]
- B) [Opcion]
- C) [Opcion]
- D) [Opcion]

<details>
<summary>Ver Respuestas</summary>

1. [Letra] - [Breve explicacion]
2. [Letra] - [Breve explicacion]
3. [Letra] - [Breve explicacion]

</details>
```

## Guidelines

- 3 questions per module (not 5)
- No elaborate formatting
- Include answer key in collapsible section
- Questions should test understanding, not memorization
