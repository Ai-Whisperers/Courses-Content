# Prerrequisitos - Antes de Usar IA para Marketing y Comunicación

## Conocimientos FUNDAMENTALES antes de crear contenido con IA

---

## Por Qué Estos Prerrequisitos Son Importantes

```
La IA puede generar contenido rápidamente, pero:
• Sin conocer tu marca, genera contenido genérico
• Sin entender tu audiencia, el mensaje no conecta
• Sin saber marketing, no puedes evaluar la calidad
• Sin ética, puedes dañar tu reputación

El contenido generado por IA sin criterio humano
es fácilmente reconocible y poco efectivo.
```

## Conocimientos de Marketing Fundamentales

### 1. Los 4Ps (Marketing Mix)

| Elemento | Preguntas que debes responder |
|----------|------------------------------|
| Producto | ¿Qué vendes? ¿Qué problema resuelve? |
| Precio | ¿Cuánto cuesta? ¿Cómo se compara con competencia? |
| Plaza | ¿Dónde lo vendes? ¿Cómo llega al cliente? |
| Promoción | ¿Cómo comunicas? ¿Qué canales usas? |

### 2. Segmentación y Target

```
Antes de pedir a la IA que cree contenido, define:

DEMOGRAFÍA:
• Edad: _____
• Género: _____
• Ubicación: _____ (¡Paraguay tiene contextos únicos!)
• Nivel socioeconómico: _____
• Ocupación: _____

PSICOGRAFÍA:
• Valores: _____
• Intereses: _____
• Estilo de vida: _____
• Problemas/dolores: _____
• Aspiraciones: _____

COMPORTAMIENTO:
• ¿Dónde busca información?
• ¿Qué redes usa?
• ¿Cómo toma decisiones de compra?
• ¿Qué objeciones tiene?
```

### 3. Propuesta de Valor

```
Fórmula básica:
"Ayudamos a [AUDIENCIA] a [RESOLVER PROBLEMA]
mediante [SOLUCIÓN] para que puedan [BENEFICIO]."

Ejemplo Paraguay:
"Ayudamos a PyMEs paraguayas a digitalizarse
mediante consultoría práctica para que puedan
competir en el mercado moderno."
```

## Conocimientos de Comunicación

### 1. Principios de Copywriting

| Principio | Descripción |
|-----------|-------------|
| AIDA | Atención → Interés → Deseo → Acción |
| Beneficios vs Features | Vende el "para qué", no el "qué" |
| Una idea por pieza | Claridad sobre cantidad |
| Call to Action claro | Decir exactamente qué hacer |
| Prueba social | Testimonios, números, credenciales |

### 2. Tono de Voz

```
Define el tono de tu marca ANTES de usar IA:

FORMAL ◄──────────────────► INFORMAL
TÉCNICO ◄─────────────────► SIMPLE
SERIO ◄────────────────────► HUMOR
CORPORATIVO ◄─────────────► PERSONAL

Tu marca es: _____________________
```

### 3. Estructura de Contenido

**Para redes sociales:**
```
HOOK (primeras palabras que capturan)
     ↓
CUERPO (desarrolla la idea)
     ↓
CTA (call to action)
```

**Para emails:**
```
ASUNTO (determina si abren)
     ↓
SALUDO (personalización)
     ↓
PROBLEMA (que reconocen)
     ↓
SOLUCIÓN (tu oferta)
     ↓
CTA (qué hacer ahora)
```

## Conocimientos Digitales

### 1. Plataformas y Sus Características

| Plataforma | Audiencia en Paraguay | Formato ideal |
|------------|----------------------|---------------|
| Facebook | Masivo, todas edades | Texto + imagen, videos |
| Instagram | 18-45 años | Imágenes, Reels, Stories |
| TikTok | 16-35 años | Videos cortos, trends |
| LinkedIn | Profesionales, B2B | Artículos, posts formales |
| WhatsApp | Todos | Mensajes directos, listas |

### 2. Métricas Básicas

```
Debes entender antes de crear contenido:

ALCANCE: Cuántas personas ven
ENGAGEMENT: Cuántas interactúan
CTR: % que hace clic
CONVERSIÓN: % que compra/actúa
ROI: Retorno sobre inversión

Si no mides, no puedes mejorar.
```

### 3. SEO Básico

```
Para contenido web:
• Keywords: palabras que busca tu audiencia
• Títulos: incluir keyword, ser atractivos
• Meta descripción: resumen para Google
• Headers: estructura H1, H2, H3
• Links: internos y externos relevantes
```

## Contexto Local (Paraguay)

### 1. Particularidades del Mercado

```
Paraguay tiene características únicas:

IDIOMA:
• Español + Guaraní (bilingüismo real)
• Jopara en comunicación informal
• Considerar uso según audiencia

CULTURA:
• Relaciones personales importan
• Confianza se construye gradualmente
• Comunidad y familia son centrales
• Humor local específico

ECONOMÍA:
• Predominan PyMEs y emprendimientos
• Economía informal significativa
• Sensibilidad al precio
• Estacionalidad (clima, fiestas)

DIGITAL:
• Alto uso de móviles
• WhatsApp es dominante
• Facebook más que Instagram en zonas rurales
• Desconfianza en compras online (mejorando)
```

### 2. Referencias Culturales

```
Fechas importantes para marketing:
• Febrero-Marzo: Carnaval, inicio de clases
• Mayo: Día de la Madre (¡grande!)
• Junio: Fiestas de San Juan
• Agosto: Día del Niño
• Octubre: Caacupé comienza
• Diciembre: Navidad, fin de año

Evitar:
• Marzo: Semana Santa (reflexión)
• Días post-feriados largos
```

## Ética en Marketing

### 1. Lo que NO debes hacer

```
❌ Publicidad engañosa
   - Prometer lo que no puedes cumplir
   - Ocultar términos y condiciones
   - Testimonios falsos

❌ Manipulación
   - Crear urgencia falsa
   - Explotar miedos sin base
   - Presión indebida

❌ Spam
   - Mensajes no solicitados
   - Frecuencia excesiva
   - Comprar listas de contactos

❌ Plagio
   - Copiar contenido de otros
   - Usar imágenes sin licencia
   - Imitar marcas competidoras
```

### 2. Buenas Prácticas

```
✓ Transparencia en publicidad pagada
✓ Respeto a datos personales (GDPR, leyes locales)
✓ Claridad en promociones y descuentos
✓ Autenticidad en testimonios
✓ Accesibilidad en contenido
```

### IA como Complemento

```
La IA (ChatGPT, Claude, etc.) es útil para:
✓ Generar borradores iniciales
✓ Variaciones de copy
✓ Lluvia de ideas
✓ Investigación inicial

La IA NO reemplaza:
✗ Tu conocimiento del cliente
✗ Tu voz de marca única
✗ El juicio de calidad
✗ La estrategia
```

## Auto-Evaluación

### Test de Conocimientos (sin IA)

1. ¿Cuál es tu propuesta de valor en una oración?

2. Describe tu cliente ideal en 3 oraciones.

3. ¿Cuáles son tus 3 principales competidores y cómo te diferencias?

4. ¿Qué métricas son más importantes para tu negocio?

5. Escribe un post para Instagram sobre tu producto (máx. 150 palabras).

### Checklist de Preparación

- [ ] Tengo mi propuesta de valor definida
- [ ] Conozco a mi audiencia target
- [ ] Definí el tono de voz de mi marca
- [ ] Sé qué plataformas usa mi audiencia
- [ ] Entiendo métricas básicas
- [ ] Tengo ejemplos de contenido que me gusta
- [ ] Conozco las particularidades del mercado paraguayo
