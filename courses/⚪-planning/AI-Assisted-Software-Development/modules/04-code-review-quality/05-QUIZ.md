### Question 3
**Which is the BEST JSDoc example?**

Option A:
```javascript
// Adds two numbers
function add(a, b) { return a + b; }
```

Option B:
```javascript
/**
 * Calculates the sum of two numbers.
 * @param {number} a - First operand
 * @param {number} b - Second operand

```

- B) Option B
