# Assessments

## AI-Assisted Software Development Course

This directory contains all assessment materials for evaluating student progress throughout the course.

---

## Assessment Structure

| Type | Weight | Location |
|------|--------|----------|
| Module Quizzes (8) | 40% | `module-quizzes/` |
| Hands-on Labs (7) | 30% | In-module labs |
| Capstone Project | 30% | Module 8 |

---

## Passing Requirements

- **Overall Score:** 70% minimum
- **Capstone Project:** Must complete and present
- **Attendance:** 80% minimum

---

## Hands-on Lab Evaluation

Labs are evaluated on:

1. **Functionality (40%)** - Does it work?
2. **AI Usage (30%)** - Effective tool utilization
3. **Quality (20%)** - Code quality and testing
4. **Documentation (10%)** - Clear documentation
