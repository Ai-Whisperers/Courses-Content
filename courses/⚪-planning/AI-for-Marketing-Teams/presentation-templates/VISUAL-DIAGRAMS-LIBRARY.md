## Usage Guidelines

### For Google Slides/PowerPoint
1. Recreate diagrams using shape tools
2. Use brand colors consistently
3. Add animation for step-by-step reveal

### For Markdown/Reveal.js
1. Use code blocks for ASCII diagrams
2. Add syntax highlighting
3. Consider Mermaid.js for dynamic diagrams

### For Canva/Design Tools
1. Use these as wireframes
2. Apply professional styling
3. Export as high-res images

### Accessibility
- Include alt text for all diagrams
- Ensure sufficient color contrast
- Provide text descriptions where needed
