## Design Guidelines

### Brand Colors
- Primary: #2563EB (Marketing Blue)
- Secondary: #7C3AED (AI Purple)
- Accent: #10B981 (Success Green)
- Dark: #1F2937
- Light: #F9FAFB

### Typography
- Headings: Bold, large (36-48pt for titles)
- Body: Clean sans-serif (18-24pt)
- Code/Prompts: Monospace font

### Slide Layouts
1. **Title Slide**: Module name, session number
2. **Content Slide**: Header + 3-5 bullet points max
3. **Visual Slide**: Diagram or screenshot with minimal text
4. **Exercise Slide**: Clear instructions + time allocation
5. **Discussion Slide**: Question for audience engagement
6. **Recap Slide**: Key takeaways summary

## Best Practices

### Keep Slides Simple
- One idea per slide
- Maximum 6 lines of text
- Use visuals over text where possible

### Engagement Techniques
- Start with a relatable problem
- Include live demos where possible
- Allow time for questions every 15-20 minutes
- Use polls for audience participation

### Pacing Guidelines
| Module Length | Slides | Demos | Exercises |
|---------------|--------|-------|-----------|
| 1 hour | 15-20 | 1-2 | 1 |
| 2 hours | 25-35 | 2-3 | 2-3 |
| Half-day | 40-50 | 3-4 | 3-5 |

## Converting to Presentation Software

These markdown templates can be converted to:

### Google Slides
1. Use markdown structure as outline
2. Apply consistent theme
3. Add visuals from VISUAL-DIAGRAMS-LIBRARY

### PowerPoint
1. Import outline structure
2. Apply corporate template
3. Embed demonstration videos

### Keynote
1. Use markdown for content planning
2. Apply presentation theme
3. Add builds and transitions

### reveal.js (Web-based)
1. Convert markdown directly
2. Host on company site
3. Enable remote presentation
