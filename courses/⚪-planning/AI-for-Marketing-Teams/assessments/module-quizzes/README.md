## Quiz Configuration

### Standard Module Quiz
- **Questions:** 10 (randomly drawn from module bank)
- **Time Limit:** 15 minutes
- **Passing Score:** 70% (7/10 correct)
- **Attempts:** 2 per module

### Final Assessment
- **Questions:** 25 (drawn across all modules)
- **Time Limit:** 45 minutes
- **Passing Score:** 70% (18/25 correct)
- **Attempts:** 2

---

## Question Types

All questions are **multiple choice** with:
- 4 answer options (A, B, C, D)
- One correct answer
- Explanation for correct answer (shown post-submission)

---

## Using These Question Banks

### For LMS Implementation

1. **Import Format**
   - Questions formatted for easy import to most LMS platforms
   - Each question includes: question text, options, correct answer, explanation

2. **Randomization**
   - Configure LMS to randomly select 10 questions per attempt
   - Randomize answer order for each student

3. **Review Settings**
   - Show correct answers after submission
   - Display explanations for learning reinforcement

### For Manual Assessment

1. **Select Questions**
   - Choose 10 questions per module
   - Balance across sections (noted in each file)
   - Vary difficulty

2. **Administer**
   - Paper or digital format
   - Time as needed
   - Open or closed book (specify)

3. **Grade**
   - Use answer key at end of each file
   - Provide explanations for missed questions

## Updating Questions

### When to Update
- AI tools change significantly
- Best practices evolve
- New features become standard
- Feedback indicates confusion

### Update Process
1. Review question against current practices
2. Update options/answers as needed
3. Revise explanation
4. Note version in file footer
5. Test updated questions

## Assessment Integration

### Pre-Module (Optional)
- 3-5 questions to gauge baseline knowledge
- No impact on final grade
- Identifies focus areas

### Post-Module (Required)
- 10 questions from module bank
- Immediate feedback with explanations
- Counts toward final grade (20% total)

### Final Assessment
- 25 questions across all modules
- Comprehensive understanding test
- Counts toward final grade (30%)
