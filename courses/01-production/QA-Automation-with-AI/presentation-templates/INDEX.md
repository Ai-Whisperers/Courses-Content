## Creating Your First Presentation

### Step-by-Step Process

#### For Course Kickoff (30 minutes prep)
1. Open [MASTER-SLIDE-DECK-TEMPLATE.md](./MASTER-SLIDE-DECK-TEMPLATE.md)
2. Replace `[Your Name]` placeholders
3. Update dates and times
4. Add your organization logo
5. Test any included links
6. Print quick reference from [SPEAKER-NOTES-TEMPLATE.md](./SPEAKER-NOTES-TEMPLATE.md)
7. Practice once through
8. Ready to present!

#### For Module 2-12 (60-90 minutes prep)
1. Open [MODULE-SLIDE-TEMPLATE.md](./MODULE-SLIDE-TEMPLATE.md)
2. Reference [EXAMPLE-MODULE-1-SLIDES.md](./EXAMPLE-MODULE-1-SLIDES.md) as guide
3. Replace all `[placeholders]` with module content
4. Add code examples from module README
5. Prepare live demos
6. Add diagrams from [VISUAL-DIAGRAMS-LIBRARY.md](./VISUAL-DIAGRAMS-LIBRARY.md)
7. Complete customization checklist
8. Review speaker notes
9. Practice timing
10. Ready to present!

#### For Workshop (2-3 hours prep)
1. Open [WORKSHOP-SLIDE-TEMPLATE.md](./WORKSHOP-SLIDE-TEMPLATE.md)
2. Define what students will build
3. Create starter code repository
4. Test entire coding workflow
5. Create solution branches
6. Prepare for common issues
7. Add setup verification steps
8. Review instructor prep checklist
9. Practice live coding
10. Ready to workshop!
