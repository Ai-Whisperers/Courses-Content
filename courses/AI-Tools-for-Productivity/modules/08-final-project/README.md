In this capstone module, you will bring together all the skills learned throughout the course to create a comprehensive AI productivity transformation plan for your actual work. This isn't just an academic exercise—it's a practical blueprint you'll implement immediately.

### Task 1.2: AI Opportunity Assessment

For each task, assess using the framework from Module 6:

**Assessment Criteria:**
- Text-based? (1-5)
- Repetitive with variations? (1-5)
- Clear inputs/outputs? (1-5)
- Low risk if imperfect? (1-5)
- Time-consuming? (1-5)

**Total Score** = Sum of all criteria (max 25)

**AI Potential:**
- 20-25: High - Prioritize for automation
- 15-19: Medium - Good candidates
- 10-14: Low - Limited benefit
- Below 10: Not suitable

## Success Tips

1. **Be realistic** - Choose workflows you'll actually use
2. **Start measuring now** - Track baseline before AI implementation
3. **Document as you go** - Don't wait until the end
4. **Use real work** - Apply to actual tasks, not hypotheticals
5. **Iterate quickly** - Refine based on results
6. **Plan to share** - Helping others reinforces your learning
