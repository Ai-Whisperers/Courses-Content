# AI for Marketing Teams - Prompt Library

A curated collection of prompts for marketing tasks using AI tools.

---

## How to Use This Library

1. **Choose** the appropriate category for your task
2. **Copy** the prompt template
3. **Customize** the `[BRACKETED]` placeholders
4. **Iterate** based on AI responses

### Best Practice: Always Include Context

```
ROLE: [Specific marketing role]
CONTEXT: [Brand info + situation]
TASK: [What you need]
FORMAT: [How to structure output]
CONSTRAINTS: [Limitations or requirements]
```

## Category 1: Content Creation

### 1.1 Blog Post Outline

```
ROLE: Content strategist for [industry].

CONTEXT:
[Paste brand context]

TOPIC: [Your topic]
TARGET KEYWORD: [Primary keyword]
AUDIENCE: [Who will read this]
WORD COUNT: [Target length]

TASK: Create a detailed blog post outline with:
- 3 title options (include keyword, use power words)
- 6-8 H2 sections with logical flow
- 3 bullet points per section
- Questions to answer in each section
- Where to add examples/data (mark with [EXAMPLE])
- Suggested word count per section

CONSTRAINTS:
- Focus on actionable advice
- Avoid generic marketing speak
- Include contrarian takes where appropriate
```

### 1.2 Blog Post Section Writer

```
Write the [SECTION NAME] section of my blog post.

SECTION GOAL: [What this section accomplishes]
KEY POINTS: [From your outline]
WORD COUNT: [Target for this section]

REQUIREMENTS:
- Start with a hook or transition
- Include one specific example
- Use clear, active language
- End with transition to next section

BRAND VOICE: [Key characteristics]
```

### 1.3 Social Media Batch

```
ROLE: Social media strategist for B2B companies.

CONTEXT: [Brand context]

TASK: Create 5 [PLATFORM] posts about [TOPIC].

FORMAT for each:
1. STORY POST - Personal anecdote with business lesson
2. TIP POST - Single actionable tip
3. CONTRARIAN POST - Challenge common assumption
4. DATA POST - Insight based on statistics
5. ENGAGEMENT POST - Question to spark discussion

For each post include:
- Hook (under 10 words)
- Body (3-4 paragraphs)
- Takeaway
- CTA
- Hashtags (3)

CONSTRAINTS:
- [WORD COUNT] words each
- No: "I'm excited," "In today's world"
- Include specific numbers/examples
```

### 1.4 Email Campaign

```
ROLE: Email copywriter for [industry].

CONTEXT: [Brand context]

EMAIL DETAILS:
- Type: [Welcome / Promotional / Newsletter]
- Goal: [Primary action]
- Audience: [Segment]
- Key message: [One sentence]

TASK: Write the email with:
1. Subject line (3 options, under 50 chars)
2. Preview text (under 90 chars)
3. Opening hook
4. Value statement
5. Body content
6. Social proof
7. CTA
8. P.S. line

CONSTRAINTS:
- Under [X] words
- One main CTA
- Mobile-friendly
- Avoid: "Hope this finds you well"
```

### 1.5 Ad Copy Generator

```
Generate [FRAMEWORK] ad copy for [product/service].

FRAMEWORK: [PAS / AIDA / Before-After-Bridge]
PRODUCT: [What you're selling]
TARGET: [Who it's for]

Create 3 variations with:
- Primary text: Under [X] characters

Include emotional and logical appeals.
```

## Category 2: SEO & Optimization

### 2.1 Keyword Research Expansion

```
ROLE: SEO strategist for [industry].

SEED KEYWORDS:
- [Keyword 1]
- [Keyword 2]
- [Keyword 3]

TASK: Expand into 50 keyword opportunities.

FOR EACH provide:
| Keyword | Intent | Difficulty | Content Type |

ORGANIZE by:
1. Questions (how, what, why)
2. Comparisons (vs, alternative, best)
3. Problem-focused
4. Solution-focused
5. Long-tail specific
```

### 2.2 Content Brief Generator

```
ROLE: SEO content strategist.

TARGET KEYWORD: [Primary keyword]
SECONDARY KEYWORDS: [List 3-5]
SEARCH INTENT: [Informational / Commercial / Transactional]

CREATE A CONTENT BRIEF:

1. TITLE OPTIONS (5)
2. META DESCRIPTION (3)
3. ARTICLE STRUCTURE
   - H1, H2s, H3s
   - FAQ section (5 questions)
4. KEYWORD USAGE GUIDE
5. COMPETITIVE GAPS (what to include that competitors miss)
6. INTERNAL LINKING OPPORTUNITIES
```

### 2.3 Content Optimization Audit

```
Analyze this content for SEO:

[PASTE CONTENT]

TARGET KEYWORD: [keyword]

CHECK:
1. Keyword optimization (placement, density)
2. Readability (sentence length, paragraphs)
3. Structure (headings, lists, tables)
4. Content depth (topics covered, questions answered)
5. Engagement elements (links, visuals, CTAs)

Provide specific recommendations for improvement.
```

2. Patterns and trends

```
Create a [weekly/monthly] marketing report.

[Current period data]

COMPARISON:
[Previous period data]

FORMAT:
1. Executive summary (3-4 sentences)
2. Key metrics with comparisons
3. Channel performance
4. Top insights
5. Priority actions for next period

Keep executive-level, focus on outcomes.
```

### 3.3 Insight Extraction

```
Look at this data for non-obvious insights:

[Paste data]

Find 5 insights that are:
- Surprising or non-intuitive
- Actionable
- Significant business impact

For each: What, Why it matters, What to do about it.
```

- Profile (demographics, behaviors, psychographics)

- Content preferences

- Conversion path

TARGET SEGMENT: [Profile summary]

1. Language and tone
2. Examples and references
3. Benefits emphasis
4. Proof points

## Prompt Chaining Examples

### Chain 1: Blog Post Creation
```
1. Generate outline → Select best structure
2. Write intro → Edit for hook
3. Write each section → Add examples
4. Write conclusion → Optimize CTA
5. Generate meta → SEO check
```

### Chain 2: Campaign Development
```
1. Define segments → Validate profiles
2. Create core message → Segment variations
3. Generate emails → Review sequence
4. Create ads → Test variations
5. Design visuals → Brand consistency check
```

## Tips for Better Prompts

1. **Be Specific** - "Marketing manager at B2B SaaS" > "marketer"
2. **Provide Context** - Always include brand info
3. **Request Format** - Specify structure you want
4. **Set Constraints** - Word limits, what to avoid
5. **Iterate** - First output is starting point
