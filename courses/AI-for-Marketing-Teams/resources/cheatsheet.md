# AI for Marketing - Quick Reference Cheatsheet

## The REVIEW Framework

Before publishing AI content, always REVIEW:

- **R**ead completely - Don't skim
- **E**valuate accuracy - Fact-check
- **V**erify brand voice - Does it sound like you?
- **I**mprove clarity - Edit for audience
- **E**nhance with expertise - Add your insights
- **W**atch for AI tells - Remove generic phrases

---

## AI Content Red Flags

Remove these phrases:
- "In today's fast-paced world..."
- "It's important to note that..."
- "Let's dive in..."
- "In conclusion..."
- "Look no further!"
- "Unlock the power of..."
- "Take your X to the next level"

## Content Type Quick Prompts

### Blog Post
```
Create outline for [topic] targeting [keyword].
Include 6-8 H2s, key points, example placeholders.
```

### Social Media (Batch)
```
Create 5 [platform] posts: story, tip, contrarian, data, engagement.
150-250 words each, hooks under 10 words.
```

### Email
```
Write [type] email. Goal: [action]. Under [X] words.
Include: subject, preview, hook, value, CTA, P.S.
```

### Ad Copy
```
[PAS/AIDA] ad for [product]. Target: [audience].
Primary text + headline + CTA. 3 variations.
```

## SEO Quick Reference

### Title Tags
- Include keyword (front-load if possible)
- Under 60 characters
- Use power words
- Match search intent

### Meta Descriptions
- Under 155 characters
- Include keyword
- Clear value proposition
- Call to action

### Content Structure
- H1: One per page, includes keyword
- H2s: 6-8, keyword variations
- H3s: For subsections
- Lists and tables for scannability

## Quality Metrics

### Good AI Content Has:
- Specific examples (not generic)
- Original data or insights
- Clear brand voice
- Verified facts
- Edited for clarity
- Human perspective

### Bad AI Content Has:
- Generic phrases
- Unverified claims
- Robotic tone
- No examples
- Published without review
- No unique value

## Analytics Report Structure

1. **Executive Summary** (3-4 sentences)
2. **Key Metrics** (with comparisons)
3. **Channel Performance**
4. **Insights** (non-obvious findings)
5. **Actions** (specific next steps)

## Legal Quick Reference

### Disclose AI Use When:
- Realistic AI images in ads (Meta requirement)
- Creating content representing real people
- Platform policy requires it
- Transparency matters to audience

### Always:
- Fact-check AI claims
- Verify AI-generated links
- Review before publishing
- Document AI usage
