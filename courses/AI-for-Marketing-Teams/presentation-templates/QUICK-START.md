## Module-Specific Prep

### Module 1: AI Fundamentals
**Must Have:**
- ChatGPT and Claude accounts
- Sample marketing task to demo
- REVIEW framework printout

**Demo Flow:**
1. Show same prompt in both tools
2. Compare outputs
3. Demonstrate iteration

---

### Module 2: Content Creation
**Must Have:**
- Brand context document example
- Real content to repurpose
- Social media templates

**Demo Flow:**
1. Create blog outline
2. Generate social posts
3. Write email sequence
4. Show editing process

---

### Module 3: SEO Optimization
**Must Have:**
- Keyword research tool access (Semrush/Ahrefs demo)
- Sample content brief
- Competitor content to analyze

**Demo Flow:**
1. Keyword clustering demo
2. Content brief generation
3. Meta description optimization

---

### Module 4: Visual Content
**Must Have:**
- Canva Pro account (for AI features)
- DALL-E or Midjourney access

**Demo Flow:**

2. Request analysis
3. Generate report narrative

---

### Module 6: Personalization

### Module 7: Automation
