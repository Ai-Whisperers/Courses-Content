## Quick Reference

### Effective Prompting Formula

```
[Role] + [Context] + [Task] + [Format] + [Constraints]
```

**Example:**
```
"You are a marketing expert [Role].
We're launching a new product next month [Context].
Write 5 social media posts [Task]
as short tweets under 280 characters [Format]
that highlight the product's ease of use [Constraints]."
```

### Common Prompt Starters

| Need | Start With |
|------|------------|
| Summarize | "Summarize this in 3 bullet points..." |
| Explain | "Explain this concept for a non-technical audience..." |
| Compare | "Compare these options in a table format..." |
| Draft | "Draft a professional [document type] that..." |
| Analyze | "Analyze this data and highlight key trends..." |
| Improve | "Improve this text for clarity and professionalism..." |

## AI Limitations to Know

### What AI Does Well

- Drafting and editing text
- Summarizing long content
- Brainstorming ideas
- Explaining concepts
- Formatting information
- Basic analysis

### What AI Struggles With

| Limitation | Example | Workaround |
|------------|---------|------------|
| Recent events | News after training | Use web-enabled tools |
| Specialized knowledge | Your company's data | Provide context |
| Math calculations | Complex arithmetic | Use calculators |
| Factual accuracy | Statistics, dates | Verify important facts |
| Personal opinions | "What should I do?" | Ask for options instead |

### Red Flags to Watch For

- [ ] Made-up statistics or citations
- [ ] Overly confident wrong answers
- [ ] Outdated information presented as current
- [ ] Generic advice that doesn't fit your context
- [ ] Hallucinated names, companies, or events

## Security Guidelines

### DO's

- Use AI for drafting, not final decisions
- Review all output before sharing
- Keep sensitive data out of prompts
- Follow your company's AI policy
- Verify facts and figures

### DON'Ts

- Share confidential business information
- Include customer personal data
- Trust AI output without verification
- Use for legal, medical, or financial advice
- Assume AI knows your specific context

### Data Sensitivity Guide

| Data Type | Safe to Share? |
|-----------|----------------|
| Public information | Yes |
| General business questions | Yes |
| Industry best practices | Yes |
| Customer names/data | No |
| Financial details | No |
| Proprietary strategies | No |
| HR/employee information | No |

## Glossary

### Key Terms

| Term | Simple Definition |
|------|-------------------|
| **AI (Artificial Intelligence)** | Software that can perform tasks typically requiring human intelligence |
| **LLM (Large Language Model)** | AI trained on vast text to understand and generate language |
| **Prompt** | The instruction or question you give to AI |
| **Hallucination** | When AI generates plausible but false information |
| **Context Window** | Amount of text AI can process at once |
| **Token** | Basic unit of text (roughly 4 characters) |
| **Fine-tuning** | Customizing AI for specific tasks |
| **Generative AI** | AI that creates new content (text, images, etc.) |
