# Presentation Templates - QA Automation with AI

## Overview

This directory contains comprehensive slide presentation templates for delivering the "QA to QA Automation Engineer with AI" course. All templates are designed for flexibility and can be adapted to various presentation formats (PowerPoint, Google Slides, Keynote, Markdown-based presentations).

   - Test all links and demos

2. **Prepare Module 1**

   - See example in completed slides

## Presentation Formats

### PowerPoint / Keynote / Google Slides

**Converting Markdown to Slides**:
1. Copy markdown content
2. Paste into slides (one section per slide)
3. Apply your theme/branding
4. Format code blocks
5. Add images/screenshots
6. Test transitions

**Layout Recommendations**:
- Title slide: Full screen, centered
- Content slides: Left-aligned text, right-side visuals
- Code slides: Large, monospace font (16pt+)
- Diagram slides: Centered, full-width
- Speaker notes: Use notes pane

### Markdown-Based Presentations

**Compatible Tools**:
- Marp
- reveal.js
- Slides.com
- Slidev
- GitPitch

**Usage**:
1. Use templates as-is or convert YAML frontmatter
2. ASCII diagrams render perfectly
3. Code blocks with syntax highlighting
4. Version control friendly
5. Easy collaboration

### Terminal/CLI Presentations

**Tools**:
- present (Go tool)
- mdp (Markdown presentation tool)
- slides (terminal slideshow)

**Advantages**:
- Perfect for technical audiences
- No external dependencies
- Fast and lightweight
- SSH-friendly

## Troubleshooting

### Common Issues

**"Diagrams don't display correctly"**
- Use monospace font
- Check line wrapping
- Increase font size
- Test on actual presentation screen

**"Running out of time"**
- Skip optional examples
- Shorten Q&A
- Assign some content as reading
- Focus on core objectives

**"Students confused"**
- Slow down pace
- Add more examples
- Check prerequisites
- Use different analogies
- Provide supplementary materials

**"Demo fails"**
- Use backup screenshots
- Explain what should happen
- Offer to show later
- Don't dwell on it
