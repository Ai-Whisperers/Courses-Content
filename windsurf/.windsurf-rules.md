# Windsurf IDE Rules
# Configuraciones optimizadas para desarrollo FPUNA con IA

## AI Context Rules
- Mantener contexto académico FPUNA (PyNN, IVA, MERCOSUR)
- Enfocar respuestas en estándares académicos (75% test coverage)
- Preservar inteligencia cultural Paraguaya
- Usar estándares Git Flow apropiados para academía

## Code Quality Rules
- Black formatting (line length 120)
- Type hints obligatorios
- Imports organizados con isort
- Flake8 linting con reglas académicas

## Git Integration
- Conventional commits con prefijos académicos
- Branch naming: feature/nombre-modulo, fix/bug-id
- Pre-commit hooks para calidad de código

## Performance Rules
- Mantener context size < 80% para mejor reasoning
- Usar periodic compaction (cada 10 mensajes)
- Implementar efficient tool calling patterns
