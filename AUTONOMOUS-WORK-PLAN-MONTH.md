## Daily Work Pattern

### Autonomous Execution Flow
1. **Start of day:** Review todo list from mcp_todoread
2. **Pick highest priority pending task**
3. **Mark task in_progress** before starting
4. **Execute work** following requirements
5. **Verify quality** with lsp_diagnostics if applicable
6. **Mark completed** immediately after finishing
7. **Move to next task** - repeat until stopping condition

### Stopping Conditions
- All tasks for the day completed
- Blocking issue encountered (document and move to next)
- Token budget approaching limit (save state, document progress)
- Complex decision needed (document options, await input)

### Progress Tracking
- Update todos in real-time
- Create daily progress summary
- Document decisions made
- Note any issues or questions
- Update STATUS files regularly
