## Adding New Examples

When adding a new example project:

1. **Create project folder** with descriptive name
2. **Include README.md** with:
   - Project overview
   - Prerequisites
   - Setup instructions
   - Key learning points
3. **Add working code** - All examples must be tested and executable
4. **Link from courses** - Reference in relevant course modules

## Standards

- All examples must follow `../_shared/configs/CLAUDE-base.md` guidelines
- Include comprehensive README
- Provide setup/installation instructions
- Keep dependencies minimal and documented
- Include tests where applicable
