# Certificate Codes Reference

## Certificate ID Format

```
CERT-[COURSE-CODE]-[YEAR]-[SEQUENCE]
```

**Example:** `CERT-QAAI-2025-00142`

## Sequence Numbers

- **Format:** 5-digit zero-padded number
- **Range:** 00001 - 99999
- **Reset:** Annually on January 1st

---

## Verification Code Format

Secondary verification code for additional security:

```
[FIRST-3-LETTERS-NAME]-[COURSE-CODE]-[RANDOM-6-CHAR]
```

**Example:** `JOH-QAAI-A7B3X9`

---

## Batch Certificate Codes

For corporate/group training:

```
BATCH-[COMPANY-CODE]-[COURSE-CODE]-[YEAR]-[BATCH-NUM]
```

**Example:** `BATCH-ACME-QAAI-2025-003`

---

## Digital Badge IDs

For integration with digital credential platforms:

```
BADGE-[COURSE-CODE]-[LEVEL]-[HASH]
```

**Levels:**
- F = Foundation
- P = Professional
- E = Expert
- S = Specialist

**Example:** `BADGE-QAAI-E-8f4a2c`

---

## Code Registration

All certificate codes are registered in the central database:
- Unique constraint enforced
- Timestamp recorded
- Linked to participant record
- Verification status tracked

---

## Legacy Codes

For certificates issued before this system:

```
LEGACY-[OLD-FORMAT]-[MIGRATION-DATE]
```

All legacy certificates remain valid and verifiable.
