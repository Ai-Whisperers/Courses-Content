## Document Creation Prompts

### Email Writing

```
Write a professional email to [recipient] about [topic].

Context:
- My role: [your role]
- Their role: [their role]
- Relationship: [formal/informal]
- Goal: [what you want to achieve]
- Key points: [main points to cover]

Tone: [professional/friendly/urgent]
Length: [brief/medium/detailed]
```

### Meeting Summary

```
Summarize this meeting transcript into:
1. Key decisions made
2. Action items with owners
3. Open questions
4. Next steps

Meeting: [paste transcript or notes]
```

### Requirements Document

```
Create a requirements document for [feature/project]:

Background: [context]
Users: [who will use this]
Problem: [what problem it solves]

Include:
- User stories
- Acceptance criteria
- Out of scope items
- Dependencies
```

## Data Analysis Prompts

### Spreadsheet Analysis

```
Analyze this data and provide:
1. Summary statistics
2. Key trends
3. Anomalies or outliers
4. Actionable insights
5. Recommended next steps

Data: [paste data or describe dataset]
Focus area: [what you're trying to learn]
```

### Report Generation

```
Create a [weekly/monthly/quarterly] report on [topic]:

Data points:
- [metric 1]: [value]
- [metric 2]: [value]
- [metric 3]: [value]

Include:
- Executive summary
- Trend analysis
- Comparison to previous period
- Recommendations
```

## Presentation Prompts

### Slide Outline

```
Create a slide deck outline for a [duration] presentation on [topic]:

Audience: [who will watch]
Goal: [what you want them to do/understand]
Key message: [main takeaway]

Include:
- Title slide
- Agenda
- Main content sections
- Summary
- Call to action
```

### Talking Points

```
Generate talking points for each slide:

Slide 1: [title]
Slide 2: [title]
...

For each slide, provide:
- 3-4 key points to mention
- Transition to next slide
- Potential questions to address
```

## Best Practices

### Effective Prompting Tips

1. **Be specific**: Include context, constraints, and desired format
2. **Give examples**: Show what good output looks like
3. **Iterate**: Refine prompts based on results
4. **Save what works**: Build a personal prompt library
5. **Version control**: Note improvements to prompts over time

### Quality Control Checklist

Before using AI output:
- [ ] Verify facts and figures
- [ ] Check for brand voice consistency
- [ ] Review for sensitive information
- [ ] Ensure accuracy of technical details
- [ ] Add personal insights where appropriate
