### ROI Calculator

```
Annual Savings = (Hours Saved/Week) × (Hourly Rate) × 52

Example:
- 5 hours saved/week
- $50/hour rate
- Annual savings: 5 × $50 × 52 = $13,000
```

## Security Guidelines

### Safe Practices

**DO:**
- Use AI for drafting, not final decisions
- Verify facts before publishing
- Remove sensitive data before pasting
- Review output for accuracy
- Follow company AI policies

**DON'T:**
- Share confidential business data
- Include personal identifiable information
- Trust AI output without verification
- Use AI for security-sensitive decisions
- Ignore your company's AI policy

### Data Classification

| Data Type | Can Use with AI? |
|-----------|------------------|
| Public information | Yes |
| Internal documents | Check policy |
| Confidential data | No |
| Customer data | No |
| Financial data | Check policy |

## Troubleshooting

### Common Issues

**AI gives wrong answers:**
- Add more context
- Break down complex questions
- Ask for step-by-step reasoning
- Verify with other sources

**Output too long/short:**
- Specify desired length
- Ask for bullet points vs. paragraphs
- Request executive summary

**Inconsistent results:**
- Save working prompts
- Use consistent formatting
- Be more specific in instructions

## Cheatsheet

### Quick Prompt Starters

| Need | Start With |
|------|------------|
| Summarize | "Summarize this in 3 bullet points..." |
| Explain | "Explain like I'm presenting to executives..." |
| Compare | "Compare X and Y in a table format..." |
| Draft | "Draft a [document type] that..." |
| Analyze | "Analyze this data and identify..." |
| Improve | "Improve this text for clarity..." |

### Output Formats

| Request | Syntax |
|---------|--------|
| Bullet list | "Present as bullet points" |
| Table | "Format as a markdown table" |
| Steps | "Provide step-by-step instructions" |
| Summary | "Summarize in one paragraph" |
| Code | "Provide as code block" |
