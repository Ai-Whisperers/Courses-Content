## ðŸŽ¨ Universal Design System

### Color Palette

**Primary Colors**:
```
Background: #1a2332 (Navy blue)
Accent: #4a9eff (Bright blue)
Success: #10b981 (Green)
Warning: #f59e0b (Orange)
Danger: #ef4444 (Red)
```

**Text Colors**:
```
Primary Text: #ffffff (White)
Secondary Text: #a0aec0 (Light gray)
Muted Text: #6b7280 (Gray)
```

### Typography

**Fonts** (choose one set):
- **Modern**: Montserrat (headings) + Open Sans (body)
- **Professional**: Roboto (headings) + Lato (body)
- **Classic**: Arial Bold (headings) + Arial (body)

**Sizes**:
```
Title: 48-54pt, Bold
Subtitle: 36-40pt, Semi-bold
Body: 24-28pt, Regular
Caption: 18-20pt, Regular
```

### Layout Rules

**Consistency**:
- Slide size: 16:9 (widescreen)
- Margins: 1 inch all sides
- Logo placement: Bottom right (small)
- Page numbers: Bottom left

**Content**:
- Maximum 5 bullets per slide
- Maximum 7 words per bullet (ideal)
- One main concept per slide
- Generous whitespace

## âš¡ Quick Creation Tips

### Speed Hacks

1. **Use Master Slides** (saves 50% time)
2. **Copy-paste formatting** (Ctrl+Shift+C, Ctrl+Shift+V)
3. **Duplicate slides** for similar content
4. **Use slide sorter view** for big picture
5. **Keyboard shortcuts**:
   - Ctrl+M: New slide
   - Ctrl+D: Duplicate
   - F5: Start presentation

---

### Content Tips

1. **Less text, more visual** always
2. **One message per slide** maximum
3. **Use builds** (appear one by one) sparingly
4. **Test from back of room** (font size)
5. **Print handouts** to check readability
