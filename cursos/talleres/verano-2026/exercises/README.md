# Ejercicios - Talleres de Verano FP-UNA 2026

Este directorio contiene ejercicios prácticos para cada clase del taller.

## Estructura

| Clase | Ejercicio | Duración | Dificultad |
|-------|-----------|----------|------------|
| 1 | Primer Prompt Efectivo | 15 min | Principiante |
| 2 | Creación de Contenido | 15 min | Principiante |
| 3 | Generación de Código | 15 min | Intermedio |
| 4 | Automatización Básica | 15 min | Intermedio |
| 5 | Análisis de Datos | 15 min | Intermedio |
| 6 | Proyecto Final | 30 min | Variable |

## Instrucciones Generales

1. Lee el ejercicio completo antes de comenzar
2. Usa OpenCode para completar cada tarea
3. Compara tu resultado con la solución proporcionada
4. Pregunta al instructor si tienes dudas
