# Live Demo: Calculadora Pro (FPUNA 2026)

Este proyecto es el ejemplo práctico del **Módulo 05: Proyecto en Vivo**. Sirve para demostrar cómo un flujo de trabajo basado en **Ingeniería de Contexto** y **Delegación Estratégica** produce software de alta calidad con esfuerzo manual mínimo.

## Cómo usar este demo

1.  **Abre `index.html`** en cualquier navegador para ver la interfaz premium.
2.  **Ejecuta los tests** con `node tests/calculator.test.js` para verificar la robustez del motor matemático.
3.  **Analiza los archivos de contexto**: Revisa `.opencode` y `CLAUDE.md` para entender cómo la IA "entiende" este repositorio.

## Conceptos Clave Demostrados

- **Arquitectura Modular**: El "cerebro" matemático está separado de la "piel" visual.
- **Ingeniería de Precisión**: Manejo seguro de decimales en JavaScript.
- **Estética de Alta Gama**: Glassmorphism, Dark Mode y tipografía moderna.
- **QA Autónomo**: Suite de validación técnica integrada.
