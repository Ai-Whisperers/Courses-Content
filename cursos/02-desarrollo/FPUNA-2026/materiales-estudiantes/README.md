## Por Donde Empezar

### Paso 1: Setup Inicial (OBLIGATORIO)

Antes de la primera clase, instala las herramientas:

**[CONFIGURACION-HERRAMIENTAS-IA.md](./CONFIGURACION-HERRAMIENTAS-IA.md)** - Guia de instalacion

Tiempo estimado: **30 minutos**

### Paso 2: Fundamentos (Dia 1)

Todos los estudiantes completan juntos los fundamentos de IA.

### Paso 3: Elige tu Track (Dias 2-6)

Segun tu carrera, elige un track especializado:

| Track | Para Quien | Carpeta |
|-------|-----------|---------|
| Desarrollo de Software | Ing. Informatica | [tracks/desarrollo-software/](./tracks/desarrollo-software/) |
| Electronica | Ing. Electronica, Mecatronica | [tracks/electronica/](./tracks/electronica/) |
| Aeronautica | Ing. Aeronautica | [tracks/aeronautica/](./tracks/aeronautica/) |
| Marketing | Marketing, Comunicacion, Turismo | [tracks/marketing/](./tracks/marketing/) |
| Investigacion | Academia, Posgrado | [tracks/investigacion/](./tracks/investigacion/) |

## Estructura del Curso

```
6 dias, 2 horas por dia (12 horas total)

Dia 1: Fundamentos (todos juntos)
└── Setup + Primeros pasos con IA

Dias 2-6: Especializacion (por track)
├── Dia 2: Electronica
├── Dia 3: Aeronautica
├── Dia 4: Desarrollo de Software
├── Dia 5: Marketing
└── Dia 6: Investigacion + Cierre
```

## Verificacion Rapida

Antes de empezar, verifica que tienes:

- [ ] Laptop con Windows/Mac/Linux
- [ ] Conexion a internet estable
- [ ] Node.js 18+ instalado (`node --version`)
- [ ] Git instalado (`git --version`)
- [ ] VS Code instalado
- [ ] Cuenta en GitHub

---

## Tips para el Exito

1. **Completa el setup ANTES de clase** - No pierdas tiempo en clase instalando
2. **Practica cada ejercicio** - Leer no es suficiente
3. **Pregunta cuando no entiendas** - No hay preguntas tontas
4. **Experimenta** - Rompe cosas, aprende de errores
5. **Colabora** - Ayuda a companeros, aprende de ellos
