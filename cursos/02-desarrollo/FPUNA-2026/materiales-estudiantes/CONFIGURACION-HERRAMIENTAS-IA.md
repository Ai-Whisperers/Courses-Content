# Modulo 01: Instalacion del Motor de IA

**FPUNA Verano 2026** | Tiempo estimado: 30 minutos

---

## 1. Instala Node.js (El Motor)

Descarga la version **LTS** desde: [nodejs.org](https://nodejs.org/)

Verifica en tu terminal:
```bash
node --version
```
Debe mostrar `v18` o superior.

---

## 2. Instala OpenCode (La Interfaz)

Copia y pega este comando en tu terminal:
```bash
npm i -g opencode-ai
```

Verifica:
```bash
opencode --version
```

---

## 3. Conecta el "Cerebro" (Gemini)

1. Consigue tu **API Key gratis** en: [Google AI Studio](https://aistudio.google.com/app/apikey)

2. En tu terminal escribe:
```bash
opencode
```

3. Usa el comando `/connect` para pegar tu API Key.

---

## 4. Fin de lo manual: Tu primer comando IA

Copia y pega este **Prompt Maestro** en OpenCode:

```
Instala globalmente oh-my-opencode, ejecutalo e inicializa el entorno con omo init. Avisame cuando termines.
```

---

## Listo!

Si todo funciono, ya tienes tu entorno de IA configurado.

**Problemas?** Pregunta en el canal `#soporte-tecnico` de Slack.
