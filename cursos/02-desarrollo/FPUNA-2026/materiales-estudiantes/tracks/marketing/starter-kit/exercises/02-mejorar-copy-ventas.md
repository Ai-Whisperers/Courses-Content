# Ejercicio 2: Mejorar Copy de Ventas con IA

## Objetivo
Usar IA para analizar y mejorar textos de ventas existentes.

## Duración
45-55 minutos

## Prerrequisitos
- Ejemplos de copy de ventas (propios o de referencia)
- Conocimiento básico de copywriting
- Acceso a Claude

## Advertencia Importante

**Al mejorar copy existente:**
- Mantén la voz de marca original
- No exageres beneficios (publicidad engañosa)
- Verifica que las ofertas/precios sean reales
- Respeta la Ley de Defensa del Consumidor

---

## Parte 1: Analizar Copy Débil (10 minutos)

### El problema

Este es el copy actual de una tienda de ropa en Asunción:

```

Remeras, pantalones, vestidos.

### ¿Qué está mal?

Antes de pedir ayuda, identifica los problemas:

- [ ] ¿Hay beneficio claro para el cliente?
- [ ] ¿Hay urgencia real o falsa?
- [ ] ¿Hay diferenciador vs competencia?
- [ ] ¿Hay call to action específico?
- [ ] ¿Conecta emocionalmente?
- [ ] ¿Tiene precio o rango de precio?
- [ ] ¿Tiene dirección completa?

## Parte 3: Generar Versión Mejorada (15 minutos)

### Pide mejora con especificaciones

```
Ahora reescribe ese copy siguiendo estas pautas:

**Estructura AIDA:**
- Atención: Gancho que detenga el scroll
- Interés: Beneficio principal para la clienta
- Deseo: Razón emocional para comprar ahora
- Acción: Call to action claro y específico

**Incluir:**
- Precio o rango de precio en guaraníes
- Dirección exacta con referencia
- Horario de atención
- Urgencia real (si hay stock limitado, decirlo)
- WhatsApp para consultas

**Tono:**
- Profesional pero cercano
- Como amiga que te da un buen dato
- Sin gritar (no usar muchas mayúsculas o signos)

**Formatos:**
1. Para cartel en vidriera (50 palabras máximo)
2. Para post de Instagram (150 palabras)
3. Para mensaje de WhatsApp (3 líneas)
4. Para flyer impreso (con estructura visual sugerida)

**NO usar:**
- "Oferta imperdible" (muy gastado)
- "Los mejores precios" (todos dicen eso)
- "Aprovecha ya" (sin razón real)
```

## Parte 4: Fórmulas de Copywriting (10 minutos)

### Aprende fórmulas útiles

```

   Ejemplo para: [gym en Fernando de la Mora]

2. **AIDA (Atención - Interés - Deseo - Acción)**
   Ejemplo para: [restaurante de comida típica]

3. **4U (Útil - Urgente - Único - Ultra-específico)**
   Ejemplo para: [academia de inglés]

4. **Before-After-Bridge**
   Ejemplo para: [servicio de limpieza]

5. **FAB (Features - Advantages - Benefits)**

## Checklist de Copy Efectivo

### Antes de publicar, verificar:

**Claridad:**
- [ ] ¿Se entiende qué vendes?
- [ ] ¿Se entiende el beneficio para el cliente?
- [ ] ¿Hay un precio o rango claro?

**Persuasión:**
- [ ] ¿Hay una razón para actuar ahora?
- [ ] ¿El diferenciador está claro?
- [ ] ¿Conecta con una emoción o necesidad?

**Acción:**
- [ ] ¿Hay un call to action específico?
- [ ] ¿Es fácil contactar/comprar?
- [ ] ¿Hay múltiples formas de contacto?

**Legal/Ético:**
- [ ] ¿Los beneficios son verificables?
- [ ] ¿Los precios son reales?
- [ ] ¿No hay publicidad engañosa?

**Local:**
- [ ] ¿Precios en guaraníes?
- [ ] ¿Dirección con referencias locales?
- [ ] ¿Tono apropiado para Paraguay?
