```
┌─────────────────────────────────────────────────────────────┐
│                                                              │

│   2. VERIFICAR DATOS         ¿Son precisos?                 │
│          ↓                                                   │
│   3. VERIFICAR CONTENIDO     ¿Dice lo que la IA afirma?     │
│          ↓                                                   │
│   4. VERIFICAR FORMATO       ¿Estilo correcto?              │
│          ↓                                                   │
│   5. VERIFICAR PLAGIO        ¿Es original?                  │
│          ↓                                                   │
│   6. VERIFICAR COHERENCIA    ¿Tiene sentido académico?      │
│                                                              │

```

## Paso 1: Verificación de Referencias

### Checklist para CADA Referencia

```markdown
## Verificación de Referencia #___

### Datos de la cita (como la dio la IA)
- Autor(es): _____
- Año: _____
- Título: _____
- Revista/Editorial: _____
- Volumen/Páginas: _____
- DOI/URL: _____

### Verificación paso a paso

1. BÚSQUEDA EN GOOGLE SCHOLAR
   - [ ] Busqué el título exacto entre comillas
   - [ ] Resultado: [ ] ENCONTRADO / [ ] NO ENCONTRADO
   - Si no encontrado → ALTA PROBABILIDAD DE SER FALSA

2. VERIFICACIÓN DE DOI
   - [ ] Ingresé a doi.org/[número]
   - [ ] Resultado: [ ] FUNCIONA / [ ] ERROR
   - Si error → ALTA PROBABILIDAD DE SER FALSA

3. VERIFICACIÓN DE AUTOR
   - [ ] Busqué autor en Google Scholar
   - [ ] El autor tiene otras publicaciones: [ ] SÍ / [ ] NO
   - [ ] El autor trabaja en el área: [ ] SÍ / [ ] NO
   - Si no existe → LA CITA ES FALSA

4. VERIFICACIÓN DE REVISTA
   - [ ] Busqué la revista en Scielo/Scopus/otro índice
   - [ ] La revista existe: [ ] SÍ / [ ] NO
   - [ ] El volumen/número existe: [ ] SÍ / [ ] NO
   - Si no existe → LA CITA ES FALSA

5. ACCESO AL TEXTO COMPLETO
   - [ ] Accedí al artículo completo (o abstract)
   - [ ] El contenido coincide con lo que la IA afirma: [ ] SÍ / [ ] NO
   - Si no coincide → NO USAR ESTA CITA

### Resultado final
[ ] VERIFICADA - puedo usarla
[ ] FALSA - debo eliminarla
[ ] PARCIALMENTE CORRECTA - debo corregir datos
```

## Paso 3: Verificación de Contenido

### Checklist de Precisión de Argumentos

```markdown
## Verificación de Afirmaciones

### La IA afirma que el autor X dice Y
Para cada afirmación citada:

1. Localizar la cita en el texto original
   - [ ] Encontré el pasaje exacto
   - Página/sección: _____

2. Comparar interpretación
   - [ ] La IA interpreta correctamente
   - [ ] La IA simplifica pero no distorsiona
   - [ ] La IA malinterpreta → CORREGIR
   - [ ] La IA inventa → ELIMINAR

3. Contexto
   - [ ] La cita está en contexto apropiado
   - [ ] No se sacó de contexto para cambiar significado
```

### Señales de Contenido Inventado

```
⚠️ Sospecha si:

• La cita es "demasiado perfecta" para tu argumento
• El autor supuestamente escribió exactamente lo que necesitas
• Múltiples citas dicen versiones muy similares
• No puedes encontrar el pasaje específico en la fuente
• El tono o vocabulario no coincide con la época/autor
```

## Paso 4: Verificación de Formato

### Checklist de Estilo de Citación

```markdown
## Verificación de Formato (APA 7)

### Referencias en lista final
Para cada referencia verificar:

- [ ] Orden alfabético por apellido
- [ ] Formato de autor: Apellido, N. N.
- [ ] Año entre paréntesis
- [ ] Título en minúsculas (excepto primera palabra)
- [ ] Nombre de revista en cursiva
- [ ] Volumen en cursiva, número entre paréntesis
- [ ] DOI en formato https://doi.org/xxx
- [ ] Sangría francesa (segunda línea indentada)

### Citas en texto
- [ ] Formato (Autor, año) para parafraseo
- [ ] Formato (Autor, año, p. X) para cita directa
- [ ] Citas de +40 palabras en bloque
- [ ] "y" vs "&" según contexto
- [ ] et al. correcto (3+ autores)
```

### Errores Comunes de IA en Formato

```
❌ Mezclar estilos (APA con MLA)
❌ DOIs inventados o mal formateados
❌ Números de página inexistentes
❌ Volúmenes/números que no existen
❌ Formato inconsistente entre citas
❌ Citas en texto que no están en referencias
❌ Referencias que no se citan en texto
```

## Paso 5: Verificación de Plagio

### Checklist Anti-Plagio

```markdown
## Verificación de Originalidad

### Herramientas de detección
- [ ] Pasé el texto por Turnitin/similar
- [ ] Porcentaje de similitud: _____%
- [ ] Áreas marcadas revisadas

### Verificación manual
- [ ] Cada cita directa tiene comillas y fuente
- [ ] Paráfrasis son genuinamente reformuladas
- [ ] Ideas propias están claramente diferenciadas
- [ ] No hay bloques copiados sin atribuir

### Específico para IA
- [ ] Texto generado por IA fue editado significativamente
- [ ] No hay "estilo IA" evidente (frases robóticas)
- [ ] El trabajo refleja mi voz y análisis
- [ ] Declaré uso de IA según política institucional
```

### Señales de "Texto de IA" No Editado

```
⚠️ Frases típicas de IA sin editar:

• "Es importante mencionar que..."
• "En conclusión, podemos afirmar que..."
• "Como se mencionó anteriormente..."
• "En el contexto actual..."
• "Es fundamental destacar..."
• "Sin lugar a dudas..."

→ Reescribir con voz propia
→ Ser más específico y directo
→ Usar ejemplos concretos
```

## Paso 6: Verificación de Coherencia Académica

### Checklist de Calidad Académica

```markdown
## Verificación de Calidad del Trabajo

### Estructura
- [ ] Introducción plantea problema claramente
- [ ] Marco teórico cubre conceptos necesarios
- [ ] Metodología es apropiada para objetivos
- [ ] Resultados responden a objetivos
- [ ] Discusión interpreta, no repite
- [ ] Conclusiones se derivan de resultados

### Argumentación
- [ ] Cada afirmación importante tiene respaldo
- [ ] Los argumentos son lógicos y coherentes
- [ ] Las fuentes son apropiadas para el nivel
- [ ] Hay balance de perspectivas (si aplica)

### Calidad de fuentes
- [ ] Mayoría son fuentes primarias/peer-reviewed
- [ ] Fuentes son actuales (< 5-10 años, según campo)
- [ ] Hay diversidad de autores/perspectivas
- [ ] Fuentes latinoamericanas/paraguayas incluidas

### Coherencia
- [ ] El título refleja el contenido
- [ ] Objetivos se cumplen en el desarrollo
- [ ] No hay contradicciones internas
- [ ] Conclusiones no agregan información nueva
```

## Errores Comunes a Evitar

### En Verificación

```
❌ "Ya verifiqué unas, las demás deben estar bien"
   → Verificar TODAS sin excepción

❌ "El DOI tiene formato correcto, debe funcionar"
   → Probar cada DOI manualmente

❌ "El autor suena conocido"
   → Verificar existencia real

❌ "No tengo acceso al artículo, asumo que está bien"
   → Al menos verificar abstract/existencia

❌ "La IA es muy buena, no inventaría"
   → La IA SIEMPRE puede alucinar
```

### En Uso de IA

```
❌ "Dame referencias sobre X"
   → NUNCA pedir referencias a IA

❌ Copiar bibliografía de IA directamente
   → CADA ítem debe verificarse

❌ Asumir que las estadísticas son correctas
   → VERIFICAR en fuente primaria

❌ No declarar uso de IA
   → CONSULTAR política institucional
```
