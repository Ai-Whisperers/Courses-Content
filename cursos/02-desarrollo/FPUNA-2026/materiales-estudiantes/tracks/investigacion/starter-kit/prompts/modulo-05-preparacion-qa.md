# Prompt: Preparación de Preguntas y Respuestas

Este prompt anticipa preguntas difíciles y ayuda a formular respuestas sólidas.

## Prompt

```bash
opencode "Prepara respuestas para preguntas en conferencia:

PRESENTACIÓN COMPLETA:
@attach conference_presentation.md
@attach full_paper.md

PREGUNTAS ANTICIPADAS:
1. **Sobre Metodología**
   - ¿Por qué este diseño experimental?
   - ¿Cómo controlaron variables?
   - ¿Por qué este tamaño muestral?

2. **Sobre Resultados**
   - ¿Cómo interpretan este tamaño del efecto?
   - ¿Por qué no significativo en X?
   - ¿Generalizable a otros contextos?

3. **Sobre Implicaciones**
   - ¿Aplicable a otras universidades?
   - ¿Limitaciones principales?
   - ¿Próximos pasos recomendados?

4. **Sobre IA y Ética**
   - ¿Preocupaciones sobre uso de IA?
   - ¿Sesgos en la intervención?
   - ¿Accesibilidad y equidad?

ESTRUCTURA DE RESPUESTAS:
- Respuesta concisa (30 segundos)
- Evidencia del estudio
- Conexión con literatura
- Reconocimiento de limitaciones

OUTPUT:
- Banco de 15+ preguntas y respuestas
- Estrategia de manejo
- Consejos para nervios
- Materiales de respaldo"
```
