# Prompt: Estrategia de Colaboración

Este prompt diseña una estrategia para buscar y mantener colaboraciones académicas.

## Prompt

```bash
opencode "Diseña estrategia de colaboración académica:

PERFIL ACTUAL:
@attach academic_profile.md

OBJETIVOS DE COLABORACIÓN:
- Publicaciones conjuntas
- Proyectos de investigación
- Presentaciones en conferencias
- Propuestas de financiamiento

ESTRATEGIA DE COLABORACIÓN:
1. **Identificación de Colaboradores**
   - Autores frecuentes en tu área
   - Investigadores con intereses similares
   - Perfiles institucionales relevantes
   - Redes de contacto existentes

2. **Acercamiento Estratégico**
   - Correos personalizados
   - Propuestas específicas
   - Contribución clara
   - Beneficios mutuos

3. **Mantenimiento de Colaboraciones**
   - Comunicación regular
   - Entregas oportunas
   - Calidad consistente
   - Referencias cruzadas

4. **Expansión de Red**
   - Asistencia a conferencias
   - Participación en talleres
   - Revisión de papers
   - Mentoría activa

MATERIALES A GENERAR:
- Plantilla de correo de colaboración
- Propuesta de proyecto conjunta
- Plan de comunicación
- Cronograma de contactos

OUTPUT:
- Estrategia documentada completa
- Materiales listos para usar
- Métricas de seguimiento"
```
