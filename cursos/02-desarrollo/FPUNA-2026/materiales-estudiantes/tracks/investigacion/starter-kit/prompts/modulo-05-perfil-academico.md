# Prompt: Perfil Académico

Este prompt crea un perfil académico profesional para plataformas digitales.

## Prompt

```bash
opencode "Crea perfil académico profesional completo:

INFORMACIÓN PERSONAL:
- Nombre: [Tu nombre completo]
- Afiliación: FPUNA
- Email: [email institucional]
- Intereses: [Tus áreas de investigación]

ELEMENTOS DEL PERFIL:
1. **Biografía Académica** (150 palabras)
   - Antecedentes educativos
   - Experiencia investigativa
   - Intereses principales
   - Objetivos futuros

2. **Lista de Publicaciones**
   - Artículos completos
   - Conferencias presentadas
   - Capítulos de libro
   - Documentos de trabajo (Working papers)

3. **Intereses de Investigación**
   - Palabras clave específicas
   - Proyectos actuales
   - Colaboraciones buscadas
   - Métodos preferidos

4. **Información de Contacto**
   - Correo profesional
   - Perfiles académicos
   - Redes sociales académicas
   - Sitio web personal

PLATAFORMAS A CONFIGURAR:
- ID ORCID
- Perfil de Google Scholar
- ResearchGate
- LinkedIn académico
- GitHub (si aplica)

OUTPUT:
- Biografía completa en español e inglés
- Lista de publicaciones formateada
- Palabras clave para indexación
- Plantillas para diferentes plataformas"
```
