# Capstone Rubric - Research Project
## Track 05

---

## 1. Literature Review (15 points)

### Excelente (13-15)
- Systematic search (3+ databases)
- 30+ papers reviewed
- Clear gaps identified (5+)
- Citation network analysis
- Well organized

### Bueno (10-12)
- Good search strategy
- 20-29 papers
- Gaps identified (3-4)
- Organized

### Suficiente (7-9)
- Basic search
- 15-19 papers
- Some gaps (1-2)

### Insuficiente (<7)
- Poor search
- <15 papers
- No clear gaps

---

## 2. Methodology (15 points)

### Excelente (13-15)
- Rigorous design
- Appropriate methods
- Well justified
- Ethical considerations
- Feasible

### Bueno (10-12)
- Good design
- Appropriate methods
- Justified
- Ethical

### Suficiente (7-9)
- Basic design
- Methods acceptable
- Some justification

### Insuficiente (<7)
- Poor design
- Inappropriate methods

---

## 3. Data Analysis (15 points)

### Excelente (13-15)
- Correct statistics
- Beautiful visualizations (5+)
- Reproducible code
- Well documented

### Bueno (10-12)
- Correct stats
- Good visualizations (3-4)
- Code provided

### Suficiente (7-9)
- Basic stats
- Simple visualizations (1-2)

### Insuficiente (<7)
- Incorrect stats
- Poor/no visualizations

---

## 4. Paper Writing (25 points)

### Excelente (22-25)
- 6000-7000 words
- IMRaD structure perfect
- Clear academic writing
- 20+ citations
- Professional LaTeX
- All sections complete

### Bueno (18-21)
- 5000-6000 words
- Good structure
- Clear writing
- 15-19 citations
- LaTeX formatted

### Suficiente (14-17)
- 4000-5000 words
- Basic structure
- Understandable writing
- 10-14 citations

### Insuficiente (<14)
- <4000 words
- Poor structure
- Unclear writing
- <10 citations

---

- Good Q&A prep

### Bueno (10-12)

### Suficiente (7-9)
