# Ejercicio Resuelto: Crear CLAUDE.md

## Objetivo
Crear un archivo CLAUDE.md personalizado para mejorar las respuestas de la IA.

---

## Instrucciones

Crear archivo CLAUDE.md en la raiz del proyecto con informacion personal.

---

## Ejemplo Resuelto: Estudiante de Electronica

```markdown
# Mi Contexto

## Quien Soy
Estudiante de 3er ano de Ingenieria Electronica en FPUNA (Facultad Politecnica,
Universidad Nacional de Asuncion).

## Nivel de Experiencia
- Arduino: Intermedio (2 anos de practica)
- Programacion C/C++: Basico-Intermedio
- Diseno PCB: Principiante (KiCAD)
- Python: Basico

## Mis Proyectos Actuales
1. Sistema de riego automatico (Arduino + sensores)
2. Robot seguidor de linea (competencia universitaria)
3. Medidor de consumo electrico (ESP32 + IoT)

## Como Quiero Ayuda
- Codigo siempre comentado en espanol
- Explicaciones paso a paso, no asumir conocimiento avanzado
- Preferencia por soluciones simples antes que elegantes
- Si hay multiples formas, explicar pros y contras
- Incluir warnings de seguridad cuando sea relevante

## Herramientas que Uso
- Arduino IDE 2.x
- KiCAD 7.0
- VS Code
- Proteus (simulacion)
- Multimetro digital

## Restricciones
- Presupuesto limitado (preferir componentes baratos/locales)
- Disponibilidad en Paraguay (no todos los componentes se consiguen)

## Idioma
Siempre responder en espanol.
```

## Verificacion

Despues de crear CLAUDE.md, probar con:

```
Recomiendame un sensor para mi proximo proyecto.
```

La respuesta deberia:
- Considerar tu nivel de experiencia
- Usar tu idioma preferido
- Tener en cuenta tus restricciones
- Estar relacionada con tus proyectos
