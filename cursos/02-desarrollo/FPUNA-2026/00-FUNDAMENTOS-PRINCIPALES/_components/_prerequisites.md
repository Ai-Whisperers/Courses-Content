# Prerequisites

## Hardware
- Laptop: Windows 10+, macOS 10.15+, or Linux
- RAM: 4GB+ available
- Storage: 2GB free
- Internet: Stable connection

## Software (install before class)
- Git: `git --version` should show 2.x+
- Node.js 18+: `node --version` should show v18+
- VS Code (recommended editor)

## Accounts
- GitHub account (https://github.com)
- Anthropic account (optional, for Claude API)

## Verification Commands
```bash
git --version    # Expected: git version 2.x.x
node --version   # Expected: v18.x.x or higher
npm --version    # Expected: 9.x.x or higher
```

If any command fails, arrive 1 hour early on Day 1 for tech support.
