## Cómo Elegir tu Proyecto

1. **Evalúa tu experiencia**:
   - ¿Has programado microcontroladores antes?
   - ¿Tienes experiencia con sensores/actuadores?
   - ¿Conoces protocolos como I2C, SPI, MQTT?

2. **Considera tus recursos**:
   - Presupuesto disponible
   - Tiempo extra-clase
   - Acceso a herramientas (multímetro, soldador)

3. **Revisa el catálogo** en [PROYECTOS-ESCALONADOS.md](./PROYECTOS-ESCALONADOS.md)

4. **Confirma con el instructor** antes de comenzar
