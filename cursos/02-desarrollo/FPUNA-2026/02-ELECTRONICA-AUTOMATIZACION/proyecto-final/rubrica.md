# Capstone Rubric - Smart Home System
## Track 02

---

## 1. Hardware Design (20 points)

### Excelente (18-20)
- Complete circuit schematic (professional)
- 2-layer PCB layout (manufacturing-ready)
- Clean Gerber files
- Detailed BOM with local prices
- Protection circuits included
- Modular design

### Bueno (14-17)
- Good schematic
- PCB layout complete
- Gerbers generated
- BOM present
- Basic protection

### Suficiente (10-13)
- Basic schematic
- PCB layout attempted
- Some documentation
- BOM incomplete

### Insuficiente (<10)
- Incomplete schematic
- No PCB or poor quality
- Missing documentation

---

## 2. Firmware (20 points)

### Excelente (18-20)
- All features working
- Clean code structure
- Error handling robust
- OTA updates implemented
- Web interface functional
- Well commented

### Bueno (14-17)
- Most features work
- Code organized
- Basic error handling
- Web interface present

### Suficiente (10-13)
- Core features work
- Code functional but messy
- Limited error handling

### Insuficiente (<10)
- Many features broken
- Poor code quality
- No error handling

---

## 3. IoT Integration (15 points)

### Excelente (13-15)
- MQTT working perfectly
- Cloud logging active
- Grafana dashboard complete
- Mobile app configured
- Real-time updates

### Bueno (10-12)
- MQTT functional
- Cloud logging works
- Dashboard present
- Some real-time features

### Suficiente (7-9)
- Basic MQTT
- Limited cloud integration
- Simple dashboard

### Insuficiente (<7)

- MQTT not working

| Puntos | Equivalente |
|--------|-------------|
