## SLIDE 4: CONTENIDO EN MINUTOS, NO HORAS
**Tiempo: 2 minutos**

```
+----------------------------------------------------------+
|                                                          |
|         ANTES vs AHORA                                   |
|                                                          |
|  ANTES:                                                  |
|  +----------------------------------------------------+  |
|  |  Pensar idea -> Escribir -> Editar -> Diseñar     |  |
|  |                                                    |  |
|  |  Un post de Instagram = 30-60 minutos             |  |
|  |  Una secuencia de emails = DÍAS                   |  |

|  +----------------------------------------------------+  |
|  |  "Crea 5 posts para Instagram sobre [tema]        |  |

|  |  (luego editas y ajustas a tu voz de marca)       |  |
|  +----------------------------------------------------+  |
|                                                          |
+----------------------------------------------------------+
```

## SLIDE 5: ¿QUÉ ES COPYWRITING?
**Tiempo: 2 minutos**

```
+----------------------------------------------------------+
|                                                          |
|         COPYWRITING = TEXTO QUE VENDE                    |
|                                                          |
|  +----------------------------------------------------+  |
|  |  FÓRMULAS QUE LA IA CONOCE:                       |  |
|  |                                                    |  |
|  |  AIDA: Atención -> Interés -> Deseo -> Acción     |  |
|  |  PAS:  Problema -> Agitar -> Solución             |  |
|  |  FAB:  Features -> Advantages -> Benefits         |  |
|  +----------------------------------------------------+  |
|                                                          |
|  Ejemplo de prompt:                                      |
|  +----------------------------------------------------+  |
|  |                                                    |  |
|  |  "Escribe copy usando la fórmula AIDA para        |  |
|  |  promocionar un delivery de comida saludable"     |  |
|  |                                                    |  |
|  +----------------------------------------------------+  |
|                                                          |
|  La IA aplica la fórmula + tu contexto                   |
|                                                          |
+----------------------------------------------------------+
```
