## MODULO 2: Creacion de Contenido (25 min)

### 0:15 - 0:25 | Copy para Redes Sociales

**PROMPT BASICO:**
```
Crea 5 posts de Instagram para una cafeteria artesanal en Asuncion.
Tono: casual, amigable, local
Incluir: hashtags relevantes, llamado a accion
Longitud: 150-200 caracteres cada uno
```

**PROMPT AVANZADO:**
```
Soy community manager de [MARCA]. Necesito posts para:

MARCA: Cafeteria "Guarani Coffee"
AUDIENCIA: Jovenes profesionales 25-35, Asuncion
TONO: Moderno pero con orgullo paraguayo
PRODUCTO A PROMOCIONAR: Nuevo cafe organico de Caaguazu

Crea:
1. Post de lanzamiento (con urgencia)
2. Post educativo (sobre cafe organico)
3. Post de engagement (pregunta a seguidores)
4. Post testimonial (cliente ficticio)
5. Post de behind-the-scenes

Cada post con:
- Texto principal
- 5 hashtags (mezcla populares + nicho)
- Sugerencia de imagen/visual
- Mejor hora para publicar
```

### 0:25 - 0:35 | Ejercicio: Tu Marca

## MODULO 5: Analisis de Metricas (15 min)

### 1:25 - 1:35 | Interpretar Datos

**DECIR:**
> "La IA puede analizar datos de analytics y darte insights accionables."

**PROMPT:**
```
Analiza estas metricas de Instagram del ultimo mes:

Seguidores: 5,200 (+120 vs mes anterior)
Alcance: 45,000 (-15%)
Engagement rate: 3.2% (-0.5%)
Mejor post: Video de receta (8,500 views, 450 likes)
Peor post: Promocion 2x1 (1,200 views, 45 likes)
Horarios con mas interaccion: 12-14h y 19-21h
Audiencia: 70% mujeres, 25-34 anos, Asuncion

Dame:
1. Diagnostico de la situacion
2. Por que bajo el alcance?
3. Que tipo de contenido funciona mejor?
4. 5 recomendaciones concretas para el proximo mes
5. Que metricas priorizar?
```

### 1:35 - 1:40 | KPIs y Objetivos

**PROMPT:**
```
Para una cafeteria que quiere aumentar ventas 20% en 3 meses:

Define:
1. KPIs principales de redes sociales
2. Metas mensuales realistas
3. Como medir el impacto en ventas
4. Herramientas necesarias
```

**BRIEF:**
