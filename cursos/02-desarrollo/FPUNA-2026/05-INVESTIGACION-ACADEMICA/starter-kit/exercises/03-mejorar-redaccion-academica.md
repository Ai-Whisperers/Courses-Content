# Ejercicio 3: Mejorar Redacción Académica con IA

## Objetivo
Usar IA para revisar y mejorar la calidad de textos académicos propios.

## Duración
50-60 minutos

## Prerrequisitos
- Un borrador de texto académico propio (1-2 páginas mínimo)
- Conocimiento de estilo APA
- Acceso a Claude

## ⚠️ ADVERTENCIA ÉTICA

**La IA es una HERRAMIENTA DE REVISIÓN, no un escritor.**

**Permitido:**
- ✅ Pedir que revise gramática y estilo
- ✅ Pedir sugerencias de mejora
- ✅ Pedir que explique errores
- ✅ Usar como "tutor de escritura"

**Prohibido (plagio académico):**
- ❌ Que la IA escriba secciones completas
- ❌ Presentar texto de IA como propio
- ❌ No declarar uso de IA si tu institución lo requiere

**Consulta las políticas de tu universidad sobre uso de IA.**

## Parte 2: Revisión General (15 minutos)

### Prompt para revisión completa

```
Actúa como un tutor de escritura académica.

Revisa este texto de mi [tesis/artículo/trabajo de grado]:

---
[PEGAR TU TEXTO AQUÍ - 300-500 palabras]
---

**Contexto:**
- Nivel: [pregrado/maestría/doctorado]
- Área: [tu disciplina]
- Tipo de documento: [tesis/artículo/ensayo]
- Estilo de citas: APA 7ma edición

**Revisa y comenta:**

1. **Claridad:**
   - ¿Las ideas son claras?
   - ¿Hay oraciones confusas?
   - ¿Los términos están bien definidos?

2. **Estructura:**
   - ¿Los párrafos tienen idea principal?
   - ¿Las transiciones son fluidas?
   - ¿El orden lógico es correcto?

3. **Estilo académico:**
   - ¿El tono es apropiado?
   - ¿Hay expresiones coloquiales?
   - ¿El vocabulario es preciso?

4. **Gramática:**
   - Errores de concordancia
   - Puntuación
   - Uso de tiempos verbales

5. **Formato APA:**
   - ¿Las citas están bien formateadas?
   - ¿Hay errores comunes?

**Para cada problema:**
- Señala la oración exacta
- Explica por qué es un problema
- Sugiere cómo corregirlo
- Califica severidad: menor/moderado/grave

**NO reescribas todo el texto.**
Dame feedback para que YO lo mejore.
```

## Parte 3: Problemas Específicos (15 minutos)

### Prompt para párrafos problemáticos

```
Este párrafo de mi texto me parece débil:

---
[PEGAR PÁRRAFO ESPECÍFICO]
---

**Problemas que percibo:**
- [Lo que crees que está mal]
- [Otra preocupación]

**Preguntas:**
1. ¿Cuál es la idea principal del párrafo?
   (Si no es clara, ayúdame a identificarla)

2. ¿Qué oraciones sobran o confunden?

3. ¿Cómo puedo hacer más fluida la transición
   desde el párrafo anterior?

4. ¿El nivel de detalle es apropiado?

Dame 3 versiones alternativas de la primera oración
para que yo elija cuál me gusta más y la desarrolle.
```

### Prompt para vocabulario académico

```
En mi texto uso estas expresiones que no estoy seguro
si son apropiadas para escritura académica:

1. "[expresión 1]"
2. "[expresión 2]"
3. "[expresión 3]"

Para cada una:
- ¿Es apropiada para un texto académico?
- Si no, ¿cómo la reformulo?
- ¿Hay un término más preciso?

También, ¿hay "muletillas académicas" que debo evitar?
(ej: "cabe destacar que", "es importante mencionar")
```

## Parte 4: Revisar Citas y Referencias (10 minutos)

### Prompt para formato de citas

```
Revisa si estas citas están en formato APA 7ma edición:

**En el texto:**
1. "[Oración con cita 1]"
2. "[Oración con cita 2]"
3. "[Oración con cita 3]"

**Para cada cita:**
- ¿El formato es correcto?
- ¿La puntuación está bien?
- ¿Debería ser cita directa o paráfrasis?
- ¿Falta el número de página?

**Tipos de citas que usé:**
- [ ] Cita narrativa (Apellido (año) dice que...)
- [ ] Cita entre paréntesis (...texto (Apellido, año))
- [ ] Cita directa corta
- [ ] Cita directa larga (bloque)

¿Hay variedad o repito mucho el mismo estilo?
```

### Prompt para lista de referencias

```
¿Estas referencias están correctas en APA 7?

1. [Referencia 1 como la tengo]
2. [Referencia 2 como la tengo]
3. [Referencia 3 como la tengo]

Para cada una:
- ¿El formato es correcto?
- ¿Qué errores hay?
- ¿Cómo debería quedar?

Tipos de fuentes:
- [ ] Artículo de revista
- [ ] Libro
- [ ] Capítulo de libro
- [ ] Tesis
- [ ] Página web
- [ ] Otro: ____
```

## Parte 5: Versión Mejorada (10 minutos)

### Aplica las correcciones

Después de recibir feedback, reescribe TÚ el texto.

```
Reescribí mi texto aplicando tus sugerencias:

**Versión original:**
[Tu texto original]

**Mi versión corregida:**
[Tu reescritura]

**Cambios que hice:**
1. [Cambio 1]
2. [Cambio 2]
3. [Cambio 3]

Por favor:
1. ¿Mejoró el texto?
2. ¿Hay algo que todavía deba ajustar?
3. ¿Mantuve mi voz como autor?
4. Califica de 1-10 la mejora

No reescribas por mí, solo dame feedback final.
```

## Errores Comunes en Escritura Académica

### La IA te ayudará a identificar:

**Claridad:**
- [ ] Oraciones demasiado largas (>40 palabras)
- [ ] Sujetos ambiguos ("esto", "lo cual")
- [ ] Terminología no definida
- [ ] Ideas incompletas

**Estilo:**
- [ ] Uso de primera persona sin justificación
- [ ] Expresiones coloquiales
- [ ] Adjetivos valorativos sin evidencia
- [ ] Generalizaciones sin fuente

**Estructura:**
- [ ] Párrafos sin oración principal
- [ ] Falta de transiciones
- [ ] Orden ilógico de ideas
- [ ] Repetición de conceptos

**APA:**
- [ ] Citas sin año
- [ ] Parafraseo sin cita
- [ ] "et al." mal usado
- [ ] Referencias incompletas

## Guía Rápida APA 7

### Citas en el texto:

| Tipo | Formato |
|------|---------|
| 1 autor | (García, 2020) o García (2020) |
| 2 autores | (García & López, 2020) |
| 3+ autores | (García et al., 2020) |
| Cita directa | (García, 2020, p. 45) |
| Múltiples | (García, 2020; López, 2019) |

### Errores frecuentes:

| Error | Correcto |
|-------|----------|
| Garcia | García (con tilde) |
| (Garcia, 2020, pág. 5) | (García, 2020, p. 5) |
| et. al | et al. |
| Garcia y Lopez (2020) | García y López (2020) |
