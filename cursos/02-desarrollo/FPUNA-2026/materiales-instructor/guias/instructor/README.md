## Resumen Rápido por Track

### Track 00: Fundamentos (TODOS)
- Claude Code / OpenCode instalado
- API key de Anthropic configurada
- Terminal funcionando

### Track 01: Desarrollo de Software
- Node.js 18+
- Python 3.11+
- Git
- VS Code con extensiones

### Track 02: Electrónica
- Arduino IDE 2.x
- KiCAD 7+
- Drivers CH340/CP2102
- Librerías de sensores

### Track 03: Ingeniería Aeronáutica
- Python con NumPy/Matplotlib
- XFLR5 (opcional)
- OpenVSP (opcional)
- Calculadora científica

### Track 04: Marketing Digital
- Navegador con cuentas de redes sociales
- Canva (cuenta gratuita)
- Meta Business Suite (acceso)
- Google Analytics (acceso demo)

### Track 05: Investigación Académica
- Zotero
- Python con pandas/scipy
- JASP o jamovi (estadística)
- Acceso a bases de datos académicas

### Track 06: Hospitalidad y Turismo
- WhatsApp Business (cuenta demo)
- Google Sheets
- Canva
- Acceso a OTAs (Booking, TripAdvisor)
