## Frameworks de Copywriting

### AIDA (Attention, Interest, Desire, Action)
```
Prompt: "Escribe copy siguiendo AIDA para [producto].
Atención: Hook inicial
Interés: Beneficio principal
Deseo: Prueba social/urgencia
Acción: CTA claro"
```

### PAS (Problem, Agitate, Solution)
```
Prompt: "Escribe copy PAS para [producto].
Problema: El dolor del cliente
Agitar: Consecuencias de no resolver
Solución: Cómo tu producto resuelve"
```

### FAB (Features, Advantages, Benefits)
```
Prompt: "Lista FAB para [producto].
Features: Características técnicas
Advantages: Por qué es mejor
Benefits: Cómo mejora la vida del cliente"
```

## Consideraciones Éticas

### Buenas Prácticas
- ✅ Revisar y editar contenido generado
- ✅ Mantener voz de marca consistente
- ✅ Verificar datos y claims
- ✅ Ser transparente cuando aplica

### Evitar
- ❌ Publicar contenido sin revisar
- ❌ Hacer claims falsos o exagerados
- ❌ Usar imágenes de personas sin permiso
- ❌ Spam o contenido engañoso

## Métricas Clave

| Métrica | Qué mide | Bueno si |
|---------|----------|----------|
| CTR | Click-through rate | > 2% |
| Open Rate | Apertura de emails | > 20% |
| Engagement | Interacciones/alcance | > 3% |
| Conversion | Acciones completadas | Depende |
| ROAS | Return on Ad Spend | > 3x |
