## Consideraciones Éticas en Academia

### Uso Apropiado de IA
- ✅ Brainstorming de ideas
- ✅ Mejora de redacción (gramática, claridad)
- ✅ Explicación de conceptos
- ✅ Generación de código para análisis
- ✅ Revisión de literatura exploratoria

### Uso Inapropiado
- ❌ Generar texto y presentarlo como propio
- ❌ Fabricar datos o resultados
- ❌ Ocultar uso de IA cuando se requiere declarar
- ❌ Parafrasear sin entender el contenido
- ❌ Evitar el aprendizaje real

### Declaración de Uso
Muchas revistas ahora requieren declarar el uso de IA. Ejemplo:
```
"Se utilizó Claude AI para asistir en la mejora de la
redacción y claridad del texto. Todo el contenido intelectual,
análisis de datos y conclusiones son trabajo original de los autores."
```

## Estilos de Citación

### APA 7 (más común en ciencias sociales)
```
Apellido, N. N. (Año). Título del artículo. Nombre de la Revista, volumen(número), páginas. https://doi.org/xxx
```

### IEEE (ingeniería)
```
[1] N. N. Apellido, "Título del artículo," Nombre de la Revista, vol. X, no. Y, pp. ZZ-ZZ, Mes Año.
```

### Chicago
```
Apellido, Nombre. "Título del artículo." Nombre de la Revista vol, no. X (Año): páginas.
```

## Estructura de Tesis/Paper

### Típica
1. **Resumen/Abstract**
2. **Introducción**
3. **Marco Teórico/Revisión de Literatura**
4. **Metodología**
5. **Resultados**
6. **Discusión**
7. **Conclusiones**
8. **Referencias**

### Prompts para Cada Sección
```
Para Introducción:
"Ayúdame a estructurar una introducción sobre [tema].
Debe incluir: contexto, problema, objetivo, y estructura del trabajo."

Para Metodología:
"Describe la metodología para un estudio [cuantitativo/cualitativo]
sobre [tema] con [N] participantes usando [instrumento]."

Para Discusión:
"Mis resultados muestran [hallazgos]. Ayúdame a discutir
cómo se relacionan con [teoría/estudios previos]."
```
