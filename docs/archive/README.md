## Archive Policy

### What Gets Archived
- Documents superseded by newer versions
- Completed one-time reports/analyses
- Historical planning documents (post-execution)
- Old guides replaced by better alternatives

### What Stays Active
- Current status reports (`HONEST-STATUS-REPORT.md`)
- Recent change summaries (`RESTRUCTURING-SUMMARY.md`, `IMPROVEMENTS-SUMMARY.md`)
- Active guides (`docs/guides/`)
- Current templates (`docs/templates/`)
- Active planning (`docs/planning/`)

### Archive Schedule
- **After major updates:** Archive superseded documents immediately
- **Quarterly review:** Check for documents to archive
- **Annual cleanup:** Review all archives, remove if no longer relevant
