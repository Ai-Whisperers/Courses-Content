## Step 4: Learning Outcomes

Write 6-10 measurable outcomes using Bloom's Taxonomy:

### Bloom's Verbs by Level

| Level | Verbs | Example |
|-------|-------|---------|
| Remember | Define, list, identify | "List the 5 principles of..." |
| Understand | Explain, describe | "Explain how X affects Y" |
| Apply | Implement, execute | "Implement X using Y" |
| Analyze | Compare, examine | "Analyze trade-offs between..." |
| Evaluate | Assess, justify | "Evaluate the best approach for..." |
| Create | Design, develop | "Design a solution for..." |

### Learning Outcomes Template

By completing this course, students will be able to:

1. [Remember level] Define/List/Identify...
2. [Understand level] Explain/Describe...
3. [Apply level] Implement/Execute...
4. [Apply level] Use/Apply...
5. [Analyze level] Compare/Analyze...
6. [Evaluate level] Assess/Evaluate...
7. [Create level] Design/Develop...

## Step 5: Business Case

### Development Estimate

| Activity | Hours | Notes |
|----------|-------|-------|
| Planning | 8-16 | This phase |
| Structure | 4-8 | Directory, syllabus |
| Content (with AI) | 20-40 | Per module: 2-4 hours |
| Review | 8-16 | Quality check |
| **Total** | **40-80** | For 8-12 module course |

### Pricing Strategy

| Delivery | Price Range | Notes |
|----------|-------------|-------|
| Corporate Training | $8,000-20,000 | Per session/cohort |
| Public Course | $500-1,500 | Per person |
| Self-Paced | $200-500 | Lifetime access |
| Enterprise License | $10,000-50,000 | Annual, unlimited |

### Break-Even Analysis

```
Development Cost: [Hours] x [Rate] = $[Total]
Price per Student: $[Price]
Break-even: [Cost] / [Price] = [Students needed]
```
