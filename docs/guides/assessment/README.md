## Assessment Philosophy

Our assessments are:

- **Practical** - Focus on hands-on skills over theoretical knowledge
- **Progressive** - Increase in complexity throughout the course
- **Real-world** - Mirror actual job responsibilities
- **Transparent** - Clear rubrics and expectations

---

## Standard Assessment Weights

### Self-Paced Courses (12 modules)

| Component | Weight |
|-----------|--------|
| Module Exercises | 30% |
| Module Quizzes | 20% |
| Milestone Projects | 20% |
| Final Capstone | 30% |

### Instructor-Led Courses (8 modules)

| Component | Weight |
|-----------|--------|
| Module Quizzes | 15% |
| Labs & Exercises | 25% |
| Participation | 10% |
| Final Capstone | 40% |
| Presentation | 10% |

## Grading Scale

| Grade | Score | Description |
|-------|-------|-------------|
| A | 90-100% | Excellent |
| B | 80-89% | Good |
| C | 70-79% | Satisfactory (passing) |
| F | <70% | Needs improvement |

**Passing threshold:** 70% overall
