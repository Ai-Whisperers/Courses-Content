# Assessment Quick Reference

> Fast lookup for assessment weights, grading, and time estimates

---

## Assessment Weights by Format

### Self-Paced Courses (12 modules)

| Component | Weight | Per Item |
|-----------|--------|----------|
| Module Exercises | 30% | 2.5% each |
| Module Quizzes | 20% | 1.67% each |
| Milestone Projects | 20% | ~7% each (3) |
| Final Capstone | 30% | - |
| **Total** | **100%** | |

### Instructor-Led Courses (8 modules)

| Component | Weight | Per Item |
|-----------|--------|----------|
| Module Quizzes | 15% | 1.875% each |
| Labs & Exercises | 25% | 3.125% each |
| Participation | 10% | - |
| Final Capstone | 40% | - |
| Presentation | 10% | - |
| **Total** | **100%** | |

## Grading Scale

| Grade | Score | Description |
|-------|-------|-------------|
| A | 90-100% | Excellent |
| B | 80-89% | Good |
| C | 70-79% | Satisfactory |
| F | <70% | Needs Improvement |

**Pass threshold:** 70%

## Common Rubric Criteria

### Code Quality

| Level | Score | Criteria |
|-------|-------|----------|
| Excellent | 90-100% | Clean, documented, follows all conventions |
| Good | 80-89% | Works well, minor issues |
| Satisfactory | 70-79% | Functional, needs improvement |
| Needs Work | <70% | Missing or broken |

### Test Coverage

| Level | Coverage |
|-------|----------|
| Excellent | >90% |
| Good | 70-90% |
| Satisfactory | 50-70% |
| Needs Work | <50% |

## Policies at a Glance

### Retakes

| Type | Allowed | Wait |
|------|---------|------|
| Quiz | 1 | 24 hours |
| Exercise | 1 revision | 1 week |
| Milestone | 1 revision | Per feedback |
| Final | 1 revision | 2 weeks |

### Late Penalties

| Days Late | Penalty |
|-----------|---------|
| 1-2 | -10% |
| 3-7 | -25% |
| >7 | Contact instructor |

## Quick Checklists

### Code Review

- [ ] Runs without errors
- [ ] Tests pass
- [ ] Naming conventions
- [ ] No hardcoded values
- [ ] Error handling
- [ ] Comments where needed
- [ ] No sensitive data

### Feedback Must Include

- [ ] Score with breakdown
- [ ] What worked well
- [ ] Specific improvements
- [ ] Action items
