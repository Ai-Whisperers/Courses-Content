```markdown
Each exercise should:
- Reinforce key learning objectives
- Include realistic business scenario
- Have step-by-step instructions
- Produce concrete deliverable
- Include discussion questions

Generate complete exercises now.
```

```markdown
Create a 10-15 question quiz for [MODULE NAME].

- Mix of multiple choice, true/false, short answer
- Answers with explanations
- Learning objective mapping

Generate complete quiz now.
```

## Common Challenges & Solutions

### "Claude's output is too generic"
**Solution:**
```markdown
This is too generic. Please revise with:
- Specific company names and real examples
- Actual metrics and dollar amounts
- Current 2024-2025 references
- Industry-specific details
Make it feel like you've actually worked with these companies.
```

### "The tone doesn't match Module 1"
**Solution:**
```markdown
The tone is too [formal/technical/casual]. Please revise to match Module 1:
- Conversational but professional
- Uses "you" language
- Includes rhetorical questions
- Empowering, not intimidating
- Light analogies when helpful
```

### "Content is too short"
**Solution:**
```markdown
This is only [X] lines. Please expand to 400-450 lines by:
- Adding 3 more industry examples
- Expanding best practices section
- Adding more detailed explanations
- Including additional frameworks
```

### "Examples are outdated"
**Solution:**
```markdown
Please update all examples to 2024-2025:
- Current AI tools (ChatGPT-4, Claude 3.5, Gemini)
- Recent company examples
- Updated pricing
- Latest statistics
No examples older than 2023.
```
